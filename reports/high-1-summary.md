### 2.2 다항식의 연산

#### 곱셈 공식 (고등 수준)

| 공식 | 설명 |
|------|------|
| (a+b+c)² = a²+b²+c²+2(ab+bc+ca) | 세 항의 완전제곱식 |
| (a+b)³ = a³+3a²b+3ab²+b³ | 세제곱 전개 (계수 1,3,3,1) |
| (a-b)³ = a³-3a²b+3ab²-b³ | 세제곱 전개 |

> **기하학적 이해**: (a+b+c)²는 한 변의 길이가 a+b+c인 정사각형을 9개 조각으로 분할하는 것과 같다.

#### 다항식의 나눗셈

다항식의 나눗셈은 항등식 형태로 귀결된다:

```
A = B × Q + R   (단, deg(R) < deg(B))

A: 피제수, B: 제수, Q: 몫, R: 나머지
```

**조립제법 (Synthetic Division)**: 일차식 (x-a)로 나눌 때 계수만으로 몫과 나머지를 구하는 알고리즘. 컴퓨터 과학의 호너의 방법(Horner's Method)과 동일한 원리.

### 2.3 나머지정리와 인수정리

#### 나머지정리

> P(x)를 (x-α)로 나눈 나머지 R = **P(α)**

증명: P(x) = (x-α)Q(x) + R에서 x=α를 대입하면 R = P(α)

#### 인수정리

> P(α) = 0 ⟹ P(x)는 (x-α)를 인수로 갖는다

**고차방정식 해결 알고리즘**:
1. 상수항의 약수를 P(x)에 대입
2. P(α) = 0인 α를 발견
3. 조립제법으로 차수 낮추기 (Deflation)
4. 반복

### 2.4 인수분해 (고급)

| 유형 | 공식 | 비고 |
|------|------|------|
| 세제곱 합 | a³+b³ = (a+b)(a²-ab+b²) | 부호: +,-,+ |
| 세제곱 차 | a³-b³ = (a-b)(a²+ab+b²) | 부호: -,+,+ |
| 변형 공식 | a³+b³+c³-3abc = (a+b+c)(a²+b²+c²-ab-bc-ca) | a+b+c=0이면 a³+b³+c³=3abc |

**인수분해 전략**:
1. **공통인수 묶기**: 항상 첫 번째로 시도
2. **치환**: 반복 부분을 t로 치환하여 2차식으로 변환
3. **복이차식**: x⁴+ax²+b → x²=t 치환 또는 A²-B² 변형
4. **대칭식/교대식**: 내림차순 정리 알고리즘 적용

### 3.2 복소수

#### 허수단위와 복소수

```
i = √(-1),  i² = -1

복소수 z = a + bi  (a: 실수부, b: 허수부)

켤레복소수: z̄ = a - bi
```

#### 허수단위 i의 거듭제곱 (4주기)

| 거듭제곱 | 값 |
|----------|----|
| i¹ | i |
| i² | -1 |
| i³ | -i |
| i⁴ | 1 |
| i⁵ | i (반복) |

#### 복소수의 연산

| 연산 | 방법 |
|------|------|
| 덧뺄셈 | (a+bi) ± (c+di) = (a±c) + (b±d)i |
| 곱셈 | 다항식 곱셈과 동일, i²=-1 적용 |
| 나눗셈 | 분모·분자에 **켤레복소수** 곱하기 → 분모 실수화 |

> **주의**: 음수의 제곱근에서 √a × √b = √(ab)는 a, b가 모두 음수일 때 성립하지 않는다.

### 3.3 이차방정식

#### 판별식의 기하학적 의미

판별식 D = b² - 4ac은 이차함수 y = ax²+bx+c의 그래프와 x축의 관계를 결정한다.

| D 값 | 근의 종류 | 그래프와 x축 |
|------|-----------|-------------|
| D > 0 | 서로 다른 두 실근 | 두 점에서 교차 |
| D = 0 | 중근 | x축에 접함 |
| D < 0 | 서로 다른 두 허근 | x축과 만나지 않음 |

#### 근과 계수의 관계

ax² + bx + c = 0의 두 근 α, β에 대해:

| 관계식 | 공식 |
|--------|------|
| 두 근의 합 | α + β = -b/a |
| 두 근의 곱 | αβ = c/a |

**활용**: 근을 직접 구하지 않고도 α²+β², (α-β)², 1/α+1/β 등을 계산할 수 있다.

- α² + β² = (α+β)² - 2αβ
- (α-β)² = (α+β)² - 4αβ

### 3.4 고차방정식

#### 삼차/사차 방정식 풀이 절차

```
고차방정식
    │
    ├── 인수정리: 상수항 약수를 대입하여 P(α)=0 찾기
    │       ↓
    ├── 조립제법: (x-α)로 나누어 차수 낮추기
    │       ↓
    └── 반복하여 최종적으로 인수분해 완성
```

#### 상반방정식 (계수가 대칭)

계수가 좌우 대칭인 방정식 (예: x⁴ + 3x³ + 5x² + 3x + 1 = 0)

**풀이 알고리즘**: x²으로 양변을 나누고, t = x + 1/x로 치환

#### 연립이차방정식

| 유형 | 전략 |
|------|------|
| 일차식 + 이차식 | 일차식에서 한 문자 정리 → 이차식에 **대입** |
| 이차식 + 이차식 | 인수분해 가능한 식 유도, 또는 이차항/상수항 **소거** |

### 3.5 부등식

| ID | 개념명 | 설명 | 키워드 | 선수 개념 |
|----|--------|------|--------|----------|
| H1-ALG-11 | 이차부등식 | ax²+bx+c > 0 풀이 (그래프 이용) | 이차부등식, 해의 범위 | M3-ALG-09, M2-ALG-06 |
| H1-ALG-12 | 연립부등식 | 여러 부등식의 공통 해 | 연립부등식, 공통 범위 | H1-ALG-11 |
| H1-ALG-13 | 절대값 방정식 · 부등식 | \|ax+b\| = c, \|ax+b\| < c | 절대값, 경우 나누기 | M2-ALG-06 |

#### 이차부등식 풀이 (그래프 이용)

```
ax² + bx + c > 0 (a > 0)

D > 0: 두 근 α, β (α < β)
  → x < α 또는 x > β  (가장자리 값)
  → α < x < β         (사이 값, ≤0일 때)

D = 0: 중근 α
  → x ≠ α인 모든 실수  (>0일 때)

D < 0: 실근 없음
  → 모든 실수           (항상 양수)
```

#### 절대값 핵심 규칙

| 형태 | 풀이 |
|------|------|
| \|x\| = a (a>0) | x = a 또는 x = -a |
| \|x\| < a (a>0) | -a < x < a |
| \|x\| > a (a>0) | x < -a 또는 x > a |

#### 절대부등식 (모든 실수에서 성립)

ax² + bx + c > 0이 모든 실수에서 성립할 조건:
- a > 0 이고 D < 0

### 4.2 이차함수와 방정식의 통합

2022 개정 교육과정은 방정식과 함수를 **동일한 현상의 두 가지 표현**으로 가르친다.

```
y = f(x) = ax² + bx + c

f(x) = 0  →  x축과의 교점  →  방정식의 근
f(x) > 0  →  그래프가 x축 위  →  부등식의 해
f(x) < 0  →  그래프가 x축 아래  →  부등식의 해
```

### 4.3 정의역 제한 시 최대·최소

꼭짓점이 정의역 구간 안에 있는지에 따라 최대·최소 위치가 달라진다.

```
y = a(x-p)² + q   (a > 0일 때)

[구간 α, β]에서:
  ├── p ∈ [α, β]  →  최솟값 = q (꼭짓점)
  │                   최댓값 = max(f(α), f(β))
  │
  └── p ∉ [α, β]  →  최솟값 = min(f(α), f(β))
                       최댓값 = max(f(α), f(β))
```

### 5.2 집합의 연산

| 연산 | 기호 | 의미 |
|------|------|------|
| 교집합 | A ∩ B | A이면서 B |
| 합집합 | A ∪ B | A이거나 B |
| 여집합 | Aᶜ | A가 아닌 것 (전체집합 U 기준) |
| 차집합 | A - B | A이면서 B가 아닌 것 = A ∩ Bᶜ |

#### 원소의 개수

| 공식 |
|------|
| n(A∪B) = n(A) + n(B) - n(A∩B) |
| n(Aᶜ) = n(U) - n(A) |
| n(A∪B∪C) = n(A)+n(B)+n(C) - n(A∩B)-n(B∩C)-n(C∩A) + n(A∩B∩C) |

#### 드모르간 법칙

| 법칙 |
|------|
| (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ |
| (A ∩ B)ᶜ = Aᶜ ∪ Bᶜ |

### 5.3 명제

#### 명제의 4가지 형태

| 이름 | 형태 | 원래 명제 p→q에 대해 |
|------|------|---------------------|
| 원래 | p → q | - |
| 역 | q → p | 참↔거짓 무관 |
| 이 | ~p → ~q | 참↔거짓 무관 |
| 대우 | ~q → ~p | **원래와 진리값 동일** |

> **핵심**: 명제의 참/거짓 증명이 어려울 때 **대우**를 증명한다.

#### 필요조건·충분조건

| 관계 | 조건 | 기호 |
|------|------|------|
| p → q (참) | p는 q의 **충분조건**, q는 p의 **필요조건** | P ⊂ Q |
| p ↔ q (참) | **필요충분조건** | P = Q |

### 6.2 합의 법칙과 곱의 법칙

| 법칙 | 조건 | 키워드 | 계산 |
|------|------|--------|------|
| 합의 법칙 | A **또는** B (동시 불가) | "또는", 분류 | n(A) + n(B) |
| 곱의 법칙 | A **그리고** B (잇달아) | "그리고", 각각에 대해 | n(A) × n(B) |

#### 포함-배제 원리

두 사건이 겹칠 수 있는 경우:
```
n(A ∪ B) = n(A) + n(B) - n(A ∩ B)
```

#### 지불 방법 vs 지불 금액 구분

| 문제 유형 | 접근 |
|-----------|------|
| 지불 **방법**의 수 | 동전 구성이 다르면 다른 경우 → 곱의 법칙 |
| 지불 **금액**의 수 | 같은 금액이면 중복 → 큰 단위를 작은 단위로 환전 후 중복 제거 |

### 6.3 순열

서로 다른 n개에서 r개를 택하여 **일렬로 나열** (순서 있음)

```
nPr = n × (n-1) × ... × (n-r+1) = n! / (n-r)!

n! = n × (n-1) × ... × 2 × 1
0! = 1 (정의)
```

#### 순열의 활용 전략

| 유형 | 전략 | 예시 |
|------|------|------|
| 이웃하는 순열 | 이웃할 대상을 **한 묶음**으로 → 배열 × 묶음 내부 배열 | A,B가 이웃: (AB를 1묶음 + 나머지) × 2! |
| 이웃하지 않는 순열 | 나머지를 먼저 배열 → **사이사이 공간**에 끼워넣기 | 5명 중 A,B 비이웃: 3명 배열 → 4개 틈에 A,B 배치 |

### 6.4 조합

서로 다른 n개에서 r개를 **택하기만** (순서 없음)

```
nCr = nPr / r! = n! / (r! × (n-r)!)
```

#### 조합의 주요 성질

| 성질 | 공식 | 의미 |
|------|------|------|
| 대칭성 | nCr = nC(n-r) | r개 선택 = (n-r)개 남기기 |
| 파스칼 | nCr = (n-1)C(r-1) + (n-1)Cr | 특정 원소 포함/미포함 분류 |

#### 도형 응용

| 문제 | 공식 | 보정 |
|------|------|------|
| n개 점으로 직선 | nC2 | 일직선 위 k개 점 → -kC2 +1 |
| n개 점으로 삼각형 | nC3 | 일직선 위 k개 점 → -kC3 |
| 가로m, 세로n 선으로 평행사변형 | mC2 × nC2 | 교차 영역 |

### 7.2 행렬의 정의

행렬은 수나 문자를 직사각형으로 배열하여 괄호로 묶은 것이다.

```
     열1  열2  열3
행1 [ a₁₁  a₁₂  a₁₃ ]     ← 2×3 행렬
행2 [ a₂₁  a₂₂  a₂₃ ]

성분(Entry): aᵢⱼ = i행 j열의 값
```

> 프로그래밍의 2차원 배열 인덱싱, 엑셀의 스프레드시트와 개념적으로 동일

### 7.3 행렬의 연산

#### 덧셈·뺄셈·실수배
같은 위치의 성분끼리 연산한다. 교환법칙, 결합법칙 성립.

#### 행렬의 곱셈

A의 **행**과 B의 **열**의 내적(dot product)으로 정의:

```
A(m×n) × B(n×p) = C(m×p)

C의 (i,j) 성분 = A의 i행 · B의 j열
```

### 7.4 실수와 행렬의 연산 성질 비교

| 성질 | 실수 | 행렬 | 주의 |
|------|------|------|------|
| 교환법칙 | ab = ba ✓ | AB ≠ BA ✗ | **순서 엄수 필수** |
| 영인자 | ab=0 ⟹ a=0 or b=0 ✓ | AB=O ⟹/ A=O or B=O ✗ | **0 아닌 것끼리 곱해 0 가능** |
| 곱셈 공식 | (A+B)² = A²+2AB+B² | **(A+B)² = A²+AB+BA+B²** | AB≠BA이므로 2AB 불가 |

> **핵심**: 단위행렬 E가 포함된 식 (A+E, A-2E 등)은 AE=EA이므로 곱셈 공식을 자유롭게 사용할 수 있다.

### 7.5 단위행렬과 역행렬

#### 단위행렬 (Identity Matrix)

```
E = [ 1  0 ]     AE = EA = A
    [ 0  1 ]
```

실수의 '1'에 해당하는 곱셈의 항등원.

#### 역행렬 (Inverse Matrix)

```
A = [ a  b ]  일 때
    [ c  d ]

A⁻¹ = 1/(ad-bc) × [ d  -b ]     (단, ad-bc ≠ 0)
                     [-c   a ]
```

| 용어 | 정의 |
|------|------|
| 행렬식 (Determinant) | det(A) = ad - bc |
| 역행렬 존재 조건 | ad - bc ≠ 0 |
| 연립방정식 표현 | AX = B ⟹ X = A⁻¹B |

### 7.6 케일리-해밀턴 정리

2차 정사각행렬 A = [a,b; c,d]에 대해:

```
A² - (a+d)A + (ad-bc)E = O
```

**활용**:
- 행렬 A의 고차식 (A³, A¹⁰⁰ 등)을 **1차식 pA + qE**로 환원
- 인수분해와 결합: A³-E = (A-E)(A²+A+E)
- A²+A+E=O이면 A³=E임을 유도 가능
