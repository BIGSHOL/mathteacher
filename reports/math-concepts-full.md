## 1. 영역 체계

2022 개정 교육과정의 수학은 5개 영역으로 구분된다.

| 영역 코드 | 초등 명칭 | 중등 명칭 | 고등 명칭 |
|-----------|----------|----------|----------|
| `NUM` | 수와 연산 | 수와 연산 | 수와 연산 |
| `ALG` | 변화와 관계 | 문자와 식 | 대수 |
| `GEO` | 도형과 측정 | 기하 | 기하 |
| `STA` | 자료와 가능성 | 확률과 통계 | 확률과 통계 |
| `ANA` | - | - | 해석 |

```
수학
├── 수와 연산 (NUM)
│   자연수 → 정수 → 유리수 → 실수 → 복소수
│   사칙연산 → 거듭제곱 → 제곱근
│
├── 변화와 관계 / 문자와 식 / 대수 (ALG)
│   문자와 식 → 방정식 → 부등식
│   비와 비율 → 함수
│
├── 도형과 측정 / 기하 (GEO)
│   평면도형 → 입체도형
│   측정 → 넓이 → 부피
│
├── 자료와 가능성 / 확률과 통계 (STA)
│   자료 정리 → 대푯값 → 분포
│   경우의 수 → 확률
│
└── 해석 (ANA, 고등)
    수열 → 급수
    극한 → 미분 → 적분
```
