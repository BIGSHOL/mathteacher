### 1. 수와 연산

#### 1-1. 소인수분해

| 항목 | 내용 |
|------|------|
| **소수** | 1보다 큰 자연수 중 1과 자기 자신만을 약수로 가지는 수 |
| **합성수** | 1보다 큰 자연수 중 소수가 아닌 수 |
| **거듭제곱** | 같은 수를 여러 번 곱한 것. $a^n$ (밑: $a$, 지수: $n$) |
| **소인수분해** | 자연수를 소인수들의 곱으로 나타내는 것 |

**필수 공식**
- 최대공약수(GCD): 공통 소인수의 최소 지수 곱
- 최소공배수(LCM): 모든 소인수의 최대 지수 곱

**문제 유형**
- 개념: 소수/합성수 판별, 소인수분해 결과 표현
- 연산: 최대공약수·최소공배수 구하기, 약수의 개수 구하기
  - 약수의 개수: $N = p_1^{a} \times p_2^{b}$ → 약수의 개수 = $(a+1)(b+1)$

#### 1-2. 정수와 유리수

| 항목 | 내용 |
|------|------|
| **정수** | 양의 정수, 0, 음의 정수를 통틀어 이르는 말 |
| **유리수** | $\frac{a}{b}$ (정수 $a$, 0이 아닌 정수 $b$) 꼴로 나타낼 수 있는 수 |
| **절댓값** | 수직선 위에서 원점으로부터의 거리. $\|a\|$ |

**필수 연산 규칙**
- 부호가 같은 두 수의 덧셈: 절댓값의 합에 공통 부호
- 부호가 다른 두 수의 덧셈: 절댓값의 차에 절댓값이 큰 수의 부호
- 곱셈 부호 규칙: $(+) \times (+) = (+)$, $(-) \times (-) = (+)$, $(+) \times (-) = (-)$
- 나눗셈: 역수를 곱하는 것으로 변환

**문제 유형**
- 개념: 수직선 위 수 배치, 절댓값 의미, 대소 비교
- 연산: 정수·유리수 사칙연산 (특히 음수 포함 혼합 계산)

## 중학교 2학년

### 1. 수와 연산

#### 1-1. 유리수와 순환소수

| 항목 | 내용 |
|------|------|
| **유한소수** | 소수점 아래 숫자가 유한개인 소수 |
| **순환소수** | 소수점 아래 일정한 숫자 배열이 무한 반복되는 소수 |
| **순환마디** | 반복되는 가장 짧은 부분 |

**판별법**
- 기약분수의 분모의 소인수가 2와 5뿐 → **유한소수**
- 기약분수의 분모에 2, 5 이외의 소인수 존재 → **순환소수**

**순환소수 → 분수 변환 알고리즘**
1. $x =$ 순환소수로 놓기
2. 순환마디 길이만큼 $10^n$ 곱하기
3. 빼서 순환 부분 소거
4. $x$에 대해 풀기

**공식**: 분모에 순환마디 자릿수만큼 9, 비순환 자릿수만큼 0
예) $1.2\dot{3}$ → 분모 = 90, 분자 = $123 - 12 = 111$ → $\frac{111}{90} = \frac{37}{30}$

**문제 유형**
- 개념: 유한소수/순환소수 판별, 순환마디 표기
- 연산: 분수 ↔ 순환소수 변환, 유한소수 조건 (분모의 소인수 분석)

### 2. 변화와 관계 (1) — 식의 계산

#### 2-1. 지수법칙

| 법칙 | 공식 | 핵심 |
|------|------|------|
| 곱셈 | $a^m \times a^n = a^{m+n}$ | 밑 같으면 지수 덧셈 |
| 거듭제곱의 거듭제곱 | $(a^m)^n = a^{mn}$ | 지수 곱셈 |
| 나눗셈 ($m > n$) | $a^m \div a^n = a^{m-n}$ | 지수 뺄셈 |
| 나눗셈 ($m = n$) | $a^m \div a^n = 1$ | |
| 나눗셈 ($m < n$) | $a^m \div a^n = \frac{1}{a^{n-m}}$ | |
| 분배 | $(ab)^n = a^n b^n$ | 괄호 안 모두에 적용 |

#### 2-2. 단항식과 다항식의 사칙연산

**필수 연산**
- 단항식 × 단항식: 계수끼리, 문자끼리 (지수법칙 적용)
- 단항식 ÷ 단항식: 역수의 곱셈으로 변환
- 다항식 ± 다항식: 동류항끼리 계산
- 단항식 × 다항식: 분배법칙 적용
- 다항식 ÷ 단항식: 각 항을 단항식으로 나눔

**문제 유형**
- 개념: 지수법칙 적용 판별, 동류항 식별
- 연산: 지수법칙 혼합 계산, 다항식 사칙연산

### 3. 변화와 관계 (2) — 부등식과 연립방정식

#### 3-1. 일차부등식

**핵심 규칙**
- 양변에 같은 양수를 곱/나누기 → 부등호 방향 유지
- 양변에 같은 **음수**를 곱/나누기 → **부등호 방향 반전**

**풀이 절차**: 일차방정식과 동일하되 음수 곱/나눗셈 시 부등호 방향 주의

#### 3-2. 연립일차방정식

| 방법 | 설명 |
|------|------|
| **가감법** | 두 식의 계수를 맞춰 더하거나 빼서 한 미지수 소거 |
| **대입법** | 한 식을 한 변수로 정리 후 다른 식에 대입 |

**특수한 경우**
- 해가 무수히 많음 → 두 직선이 일치
- 해가 없음 → 두 직선이 평행

**문제 유형**
- 개념: 부등식 해의 의미, 연립방정식 해의 기하학적 의미
- 연산: 일차부등식 풀이, 연립방정식 (가감법/대입법), 활용 문제

### 4. 변화와 관계 (3) — 일차함수

#### 4-1. 일차함수의 정의와 그래프

| 항목 | 내용 |
|------|------|
| **일차함수** | $y = ax + b$ ($a \neq 0$) |
| **기울기 ($a$)** | $\frac{y\text{의 증가량}}{x\text{의 증가량}}$ — 변화율 |
| **$y$절편 ($b$)** | $x = 0$일 때의 $y$ 값 (그래프와 $y$축의 교점) |
| **$x$절편** | $y = 0$일 때의 $x$ 값 → $x = -\frac{b}{a}$ |
| **평행이동** | $y = ax$를 $y$축 방향으로 $b$만큼 이동 |

**기울기에 따른 그래프 방향**
- $a > 0$: 우상향 (증가함수)
- $a < 0$: 우하향 (감소함수)
- $|a|$ 클수록 가파름

**일차함수와 일차방정식의 관계**
- $ax + by + c = 0$ → $y = -\frac{a}{b}x - \frac{c}{b}$ (일차함수)
- 연립방정식의 해 = 두 직선의 교점

**문제 유형**
- 개념: 그래프 해석, 기울기·절편의 의미
- 연산: 기울기·절편 구하기, 일차함수 식 구하기, 그래프-방정식 연결

### 5. 도형과 측정 (1) — 삼각형과 사각형

#### 5-1. 삼각형의 성질

| 항목 | 내용 |
|------|------|
| **이등변삼각형** | 두 변의 길이가 같은 삼각형 → 두 밑각의 크기가 같음 |
| **직각삼각형 합동** | RHS (빗변+한 변), RHA (빗변+한 예각) |
| **외심** | 세 변의 수직이등분선의 교점 (외접원의 중심) |
| **내심** | 세 내각의 이등분선의 교점 (내접원의 중심) |

**필수 공식**
- 내심 이용 넓이: $S = \frac{1}{2}r(a + b + c)$ ($r$: 내접원 반지름)

#### 5-2. 사각형의 성질과 포함 관계

```
정사각형 ⊂ 직사각형 ⊂ 평행사변형
정사각형 ⊂ 마름모    ⊂ 평행사변형
```

| 사각형 | 핵심 성질 |
|--------|-----------|
| **평행사변형** | 대변의 길이·대각의 크기 동일, 대각선이 서로를 이등분 |
| **직사각형** | 평행사변형 + 네 각이 직각, 대각선의 길이가 같음 |
| **마름모** | 평행사변형 + 네 변의 길이가 같음, 대각선이 수직 |
| **정사각형** | 직사각형 + 마름모의 모든 성질 |
| **등변사다리꼴** | 평행하지 않은 대변의 길이가 같음, 대각선 길이가 같음 |

**문제 유형**
- 개념: 삼각형 중심 성질 판별, 사각형 포함 관계 (참/거짓)
- 연산: 외심·내심 이용 길이/각도 구하기, 사각형 넓이·대각선 구하기

### 6. 도형과 측정 (2) — 닮음과 피타고라스 정리

#### 6-1. 도형의 닮음

| 조건 | 내용 |
|------|------|
| **AA 닮음** | 두 각이 각각 같음 |
| **SAS 닮음** | 두 변의 비가 같고 그 끼인각이 같음 |
| **SSS 닮음** | 세 변의 비가 각각 같음 |

**필수 개념**
- 닮음비: 대응변의 길이의 비
- 넓이비: 닮음비의 제곱
- 부피비: 닮음비의 세제곱
- 평행선과 선분의 비: 삼각형에서 한 변에 평행한 선분 → 다른 두 변을 같은 비로 나눔
- **무게중심**: 세 중선의 교점, 중선을 $2:1$로 내분

#### 6-2. 피타고라스 정리

$$a^2 + b^2 = c^2$$

($c$: 빗변, $a$, $b$: 나머지 두 변)

**피타고라스의 수** (자주 출제되는 세 쌍)
- $(3, 4, 5)$, $(5, 12, 13)$, $(8, 15, 17)$, $(7, 24, 25)$

**문제 유형**
- 개념: 닮음 조건 판별, 무게중심 성질
- 연산: 닮음비를 이용한 변의 길이 구하기, 피타고라스 정리 적용

#### 7-1. 경우의 수

| 법칙 | 조건 | 계산 |

| **곱의 법칙** | 잇달아 일어나는 두 사건 A 그리고 B | $m \times n$ |

**문제 유형**
- 개념: 합의 법칙 vs 곱의 법칙 구분, 순서 유무 판단
- 연산: 줄 세우기, 대표 뽑기, 경로 문제

#### 7-2. 확률

$$P(A) = \frac{\text{사건 A가 일어나는 경우의 수}}{\text{모든 경우의 수}}$$

**필수 성질**
- $0 \leq P(A) \leq 1$
- $P(\text{전체}) = 1$, $P(\text{공집합}) = 0$
- 여사건 확률: $P(A^c) = 1 - P(A)$

**문제 유형**
- 개념: 확률의 의미, 여사건 활용 전략
- 연산: 경우의 수 기반 확률 계산, "적어도 ~일 확률" 구하기

## 중학교 3학년

### 1. 수와 연산 — 제곱근과 실수

#### 1-1. 제곱근

| 항목 | 내용 |
|------|------|
| **제곱근** | $x^2 = a$를 만족하는 $x$ ($a \geq 0$) |
| **양의 제곱근** | $\sqrt{a}$ |
| **음의 제곱근** | $-\sqrt{a}$ |

**핵심 성질**
- 양수 $a$의 제곱근: $\pm\sqrt{a}$ (2개)
- 0의 제곱근: 0 (1개)
- 음수의 제곱근: 실수 범위에서 없음
- $(\sqrt{a})^2 = a$, $\sqrt{a^2} = |a|$

#### 1-2. 무리수와 실수

| 분류 | 설명 |
|------|------|
| **유리수** | 유한소수 또는 순환소수 |
| **무리수** | 순환하지 않는 무한소수 (예: $\sqrt{2}$, $\pi$) |
| **실수** | 유리수 + 무리수 |

#### 1-3. 근호를 포함한 식의 계산

**필수 공식**
- $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$ ($a > 0, b > 0$)
- $\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}$
- 분모의 유리화: $\frac{a}{\sqrt{b}} = \frac{a\sqrt{b}}{b}$
- 대소 비교: $a > 0, b > 0$일 때 $a < b$ → $\sqrt{a} < \sqrt{b}$

**문제 유형**
- 개념: 제곱근 의미, 실수 분류 (유리수/무리수 판별)
- 연산: 근호 계산, 분모의 유리화, 제곱근의 대소 비교

### 2. 문자와 식 — 다항식의 곱셈과 인수분해

#### 2-1. 곱셈 공식 (전개)

| 번호 | 공식 | 이름 |
|------|------|------|
| 1 | $(a+b)^2 = a^2 + 2ab + b^2$ | 완전제곱식 (합) |
| 2 | $(a-b)^2 = a^2 - 2ab + b^2$ | 완전제곱식 (차) |
| 3 | $(a+b)(a-b) = a^2 - b^2$ | 합차 공식 |
| 4 | $(x+a)(x+b) = x^2 + (a+b)x + ab$ | 이차식 전개 (계수 1) |
| 5 | $(ax+b)(cx+d) = acx^2 + (ad+bc)x + bd$ | 이차식 전개 (일반) |

#### 2-2. 인수분해

| 유형 | 공식 |
|------|------|
| 공통인수 | $ma + mb = m(a+b)$ |
| 완전제곱식 | $a^2 \pm 2ab + b^2 = (a \pm b)^2$ |
| 합차 공식 | $a^2 - b^2 = (a+b)(a-b)$ |
| 이차식 (계수 1) | $x^2 + (a+b)x + ab = (x+a)(x+b)$ |
| 이차식 (일반) | $acx^2 + (ad+bc)x + bd = (ax+b)(cx+d)$ (대각선법) |

**인수분해 순서**
1. 공통인수 확인 → 있으면 먼저 묶기
2. 항의 수와 구조 확인 → 적절한 공식 적용
3. 대각선법(Cross Method): 이차항 계수가 1이 아닌 경우

**문제 유형**
- 개념: 전개와 인수분해의 역관계, 인수분해 전략 선택
- 연산: 곱셈 공식 계산, 인수분해, 식의 값 구하기

### 3. 이차방정식

#### 3-1. 풀이 방법 (위계순)

| 방법 | 적용 조건 | 공식/과정 |
|------|-----------|-----------|
| **인수분해** | 좌변이 인수분해 가능 | $AB = 0$ → $A = 0$ 또는 $B = 0$ |
| **제곱근 이용** | $(x-p)^2 = q$ 꼴 변형 가능 | $x = p \pm \sqrt{q}$ |
| **완전제곱식** | 일반적인 이차방정식 | 좌변을 완전제곱식으로 변형 |
| **근의 공식** | 모든 이차방정식 | $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ |

**판별식** ($D = b^2 - 4ac$)
- $D > 0$: 서로 다른 두 실근
- $D = 0$: 중근 (한 개의 실근)
- $D < 0$: 실근 없음

**문제 유형**
- 개념: 이차방정식의 해의 의미, 판별식 해석
- 연산: 네 가지 방법으로 이차방정식 풀기, 활용 문제

### 4. 이차함수

#### 4-1. 기본형과 표준형

| 형태 | 식 | 꼭짓점 | 대칭축 |
|------|------|--------|--------|
| 기본형 | $y = ax^2$ | $(0, 0)$ | $x = 0$ |
| 표준형 | $y = a(x-p)^2 + q$ | $(p, q)$ | $x = p$ |
| 일반형 | $y = ax^2 + bx + c$ | $\left(-\frac{b}{2a},\ -\frac{b^2-4ac}{4a}\right)$ | $x = -\frac{b}{2a}$ |

**그래프 특성**
- $a > 0$: 아래로 볼록 (최솟값 $q$)
- $a < 0$: 위로 볼록 (최댓값 $q$)
- $|a|$가 클수록 폭이 좁음

**평행이동**
- $y = ax^2$ → $x$축 방향으로 $p$, $y$축 방향으로 $q$ → $y = a(x-p)^2 + q$

**문제 유형**
- 개념: 꼭짓점·대칭축 의미, 그래프 개형 판단
- 연산: 일반형 ↔ 표준형 변환, 이차함수 식 구하기, 최댓값/최솟값

### 5. 삼각비

#### 5-1. 정의

| 삼각비 | 정의 | 기억법 |
|--------|------|--------|
| $\sin A$ | $\frac{\text{높이(대변)}}{\text{빗변}}$ | SOH |
| $\cos A$ | $\frac{\text{밑변(인접변)}}{\text{빗변}}$ | CAH |
| $\tan A$ | $\frac{\text{높이(대변)}}{\text{밑변(인접변)}}$ | TOA |

#### 5-2. 특수각의 삼각비

| 각도 | $\sin$ | $\cos$ | $\tan$ |
|------|--------|--------|--------|
| $30°$ | $\frac{1}{2}$ | $\frac{\sqrt{3}}{2}$ | $\frac{\sqrt{3}}{3}$ |
| $45°$ | $\frac{\sqrt{2}}{2}$ | $\frac{\sqrt{2}}{2}$ | $1$ |
| $60°$ | $\frac{\sqrt{3}}{2}$ | $\frac{1}{2}$ | $\sqrt{3}$ |

#### 5-3. 삼각비의 변화 ($0° \sim 90°$)

| 삼각비 | $0°$ → $90°$ 변화 |
|--------|-------------------|
| $\sin$ | $0$ → $1$ (증가) |
| $\cos$ | $1$ → $0$ (감소) |
| $\tan$ | $0$ → $\infty$ (증가, $90°$에서 정의 불가) |

**문제 유형**
- 개념: 삼각비의 의미, 특수각 값 이해
- 연산: 삼각비 계산, 변의 길이 구하기, 높이·거리 측량 문제

### 6. 원의 성질

#### 6-1. 원과 현·접선

| 성질 | 내용 |
|------|------|
| 현의 수직이등분선 | 원의 중심을 지남 |
| 중심에서 현까지의 수선 | 현을 이등분 |
| 접선의 수직성 | 접점에서의 반지름과 수직 |
| 외부 한 점에서의 접선 | 두 접선의 길이가 같음 |

#### 6-2. 원주각

**핵심 정리**
- 원주각 = $\frac{1}{2}$ × 중심각 (같은 호에 대해)
- 같은 호에 대한 원주각은 모두 같다
- 반원(지름)에 대한 원주각 = $90°$
- **접현각**: 접선과 현이 이루는 각 = 그 각 내부 호에 대한 원주각

**문제 유형**
- 개념: 원주각과 중심각의 관계, 접현각의 의미
- 연산: 원주각·중심각 구하기, 접선 길이, 현의 길이 구하기

### 7. 통계

#### 7-1. 대표값과 산포도

| 항목 | 공식 |
|------|------|
| **평균** | $\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n}$ |
| **분산** | $\sigma^2 = \frac{\sum(x_i - \bar{x})^2}{n}$ |
| **표준편차** | $\sigma = \sqrt{\text{분산}}$ |

#### 7-2. 상관관계와 산점도

| 유형 | 특징 |
|------|------|
| 양의 상관관계 | $x$ 증가 → $y$ 증가 (우상향) |
| 음의 상관관계 | $x$ 증가 → $y$ 감소 (우하향) |
| 상관관계 없음 | 뚜렷한 경향 없음 |

**산점도 해석 팁**
- $y = x$ 기준 위: $y > x$, 아래: $x > y$
- 점이 직선 주위에 밀집될수록 상관관계 강함

**문제 유형**
- 개념: 산포도의 의미, 상관관계 판별
- 연산: 분산·표준편차 계산, 산점도 해석
