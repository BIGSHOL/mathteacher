## 📗 1학기 단원별 정리

### 1단원: 유리수와 순환소수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **유리수 정의**: $\frac{a}{b}$ ($a, b$는 정수, $b \neq 0$) 꼴로 나타낼 수 있는 수<br>• **유한소수 조건**: 기약분수의 분모 소인수가 **2 또는 5뿐**일 때<br>• **순환소수**: 무한소수이지만 순환마디가 있어 분수로 표현 가능 → 유리수<br>• **순환마디 표기**: 양 끝에 점 ($0.\dot{1}4\dot{2}$) |
| **주요 오개념** | • **⭐⭐ $0.999... < 1$ 직관적 저항**: "한없이 가까워질 뿐, 결코 1이 될 수 없다"<br>  → $x = 0.999...$, $10x = 9.999...$, $9x = 9$, $x = 1$ 대수적 증명<br>• **순환마디 결정 오류**: $0.297297...$에서 순환마디를 972나 729로 착각<br>• **$\pi$와 순환소수 혼동**: "무한히 계속되는 소수 = 순환소수" 잘못된 일반화 |
| **출제 포인트** | ✓ 유한소수 판별: **'약분 → 분해 → 판단'** 3단계 알고리즘<br>✓ 순환소수의 분수 표현: 공식 암기 지양, **등식의 성질 이용 과정** 경험<br>✓ 계산기로 $\frac{1}{7}, \frac{2}{7}, \frac{3}{7}$ 등 순환마디 패턴 발견 |

### 2단원: 식의 계산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **지수법칙**:<br>  - $a^m \times a^n = a^{m+n}$ (곱 → 지수의 합)<br>  - $(a^m)^n = a^{mn}$ (거듭제곱 → 지수의 곱)<br>  - $a^m \div a^n$: $m>n$이면 $a^{m-n}$, $m=n$이면 $1$, $m<n$이면 $\frac{1}{a^{n-m}}$<br>• **다항식 계산**: 동류항 개념, 분배법칙으로 전개<br>※ 중학교에서는 **음의 지수, 0지수 미도입** → 경우의 수 분류 필수 |
| **주요 오개념** | • **⭐⭐ 연산 기호와 지수법칙 불일치**:<br>  - $a^3 \times a^2 = a^6$ (곱셈 기호에 이끌려 지수도 곱함)<br>  - $(a^3)^2 = a^5$ (지수끼리 더함)<br>• **나눗셈과 뺄셈 혼동**: $a^2 \div a^5 = a^3$이라고 답함<br>  → 분수 꼴로 풀어 약분: $\frac{a \cdot a}{a \cdot a \cdot a \cdot a \cdot a} = \frac{1}{a^3}$<br>• **분배법칙 누락**: $(2x^2y)^3 = 2x^6y^3$ (계수 2 거듭제곱 안 함) |
| **출제 포인트** | ✓ 지수 정의로 돌아가 시각적 확인: $a^3 \times a^2 = (a \cdot a \cdot a) \times (a \cdot a)$<br>✓ 특정 수($x=1, y=2$) 대입하여 **검산 습관**<br>✓ 대수 막대(Algebra Tiles)로 동류항 기하학적 이해 |

### 3단원: 부등식과 연립방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **일차부등식**: 부등호($<, >, \le, \ge$)로 대소 관계 표현<br>• **핵심 성질**: 음수를 곱/나누면 **부등호 방향 바뀜**<br>• **연립일차방정식**: 미지수 2개, 방정식 2개<br>  - **가감법**: 더하거나 빼서 한 문자 소거<br>  - **대입법**: 한 식을 다른 식에 대입<br>• **해의 특수성**: 해 1개(일반), 해 없음(불능, 평행), 해 무수히 많음(부정, 일치) |
| **주요 오개념** | • **⭐ 부등호 방향 고착화**: $-2x > 4$ → $x > -2$로 품 (방향 안 바꿈)<br>  → 수직선 모델로 대칭 이동 시 대소 역전 시각화<br>• **연립방정식 해 ↔ 그래프 교점 분리**: 대수적 해 $(x, y)$가 두 직선의 교점이라는 기하학적 의미 연결 못함 |
| **출제 포인트** | ✓ 실생활 맥락 모델링: "휴대폰 요금제 비교" 부등식<br>✓ 정형화 유형(농도, 거리-속력-시간) 지양, **삶과 밀접한 소재** 활용<br>✓ 가감법 vs 대입법 **효율성 판단** 메타인지 질문 |

### 4단원: 일차함수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **일차함수 정의**: $y = ax + b$ ($a \neq 0$)<br>• **기울기($a$)**: $\frac{\Delta y}{\Delta x}$ = 변화율, 그래프의 가파른 정도와 방향<br>• **절편**: $x$절편($y=0$일 때), $y$절편($x=0$일 때)<br>• **일차함수 ↔ 일차방정식**: $ax + by + c = 0$의 그래프는 직선<br>  → 방정식의 해 = 직선 위의 점 |
| **주요 오개념** | • **⭐ 기울기 크기 비교 혼동**: $-2$와 $-5$ 중 어느 그래프가 더 가파른지?<br>  → "가파름" = 절댓값 $|a|$, 부호 = 방향(우상향/우하향)<br>• **절편과 좌표 혼동**: $x$절편 3을 좌표평면에 찍을 때 엉뚱한 곳에 표시<br>  → 절편은 '값'이자 그래프에서는 '위치'<br>• **$x = 2$ (세로선)**: 함수 틀에 억지로 맞추려 함<br>  → 함수의 정의(모든 $x$가 하나의 $y$에 대응)와 연결하여 **함수 아님** 설명 |
| **출제 포인트** | ✓ **공학 도구 필수**: 슬라이더로 $a, b$ 변화 → 그래프 회전/평행이동 동적 관찰<br>✓ 실생활 데이터 산점도 → 추세선을 일차함수로 모델링<br>✓ 상수함수 $y = 3$과 $x = 2$의 차이 |

### 4단원: 확률

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **합의 법칙**: A 또는 B가 일어나는 경우 = $m + n$<br>• **곱의 법칙**: A와 B가 동시에 일어나는 경우 = $m \times n$<br>• **수학적 확률**: $\frac{사건의 경우의 수}{모든 경우의 수}$<br>  ※ 전제: 각 근원사건이 **동등하게(Equiprobable)** 일어남 |

| **출제 포인트** | ✓ **인지적 갈등 유발하는 반례** + 토론 수업<br>✓ 근원사건 동등성 확인 문제<br>✓ 합의 법칙 vs 곱의 법칙 구분 |

## 💡 문제 개발 전략

### 1. 지수법칙 오류 교정

| 오류 유형 | 교정 전략 |
|-----------|-----------|
| $a^3 \times a^2 = a^6$ | 지수 정의로 풀어서 확인: $(a \cdot a \cdot a) \times (a \cdot a) = a^5$ |
| $(a^3)^2 = a^5$ | 괄호 의미 강조: $a^3$을 2번 곱함 = $a^3 \times a^3 = a^6$ |
| $a^2 \div a^5 = a^3$ | 분수로 약분: $\frac{a \cdot a}{a \cdot a \cdot a \cdot a \cdot a} = \frac{1}{a^3}$ |

### 2. 정의와 성질 구분 훈련

| 도형 | 정의 | 성질 (증명 대상) |
|------|------|------------------|
| 이등변삼각형 | 두 변의 길이가 같음 | 두 밑각이 같음 |
| 직사각형 | 네 각이 모두 직각 | 대각선의 길이가 같음 |
| 평행사변형 | 두 쌍의 대변이 평행 | 대각선이 서로를 이등분 |

> "정의는 증명 없이 사용하는 전제, 성질은 증명해야 하는 대상"

### 3. 확률 오개념 교정: 반례 활용

| 오개념 상황 | 반례 |
|-------------|------|
| "흰 공 나올 확률 = $\frac{1}{2}$" (종류만 셈) | 흰 공 2개, 검은 공 1개 → 정답 $\frac{2}{3}$ |
| "윷놀이 각 면 확률 = $\frac{1}{2}$" | 윷의 구조상 평평한 면과 둥근 면의 확률 다름 |
| "'동시에' = 시간적 동시" | 주사위를 순서대로 던져도 곱의 법칙 적용 |

### 4. 공학 도구 활용 (2022 개정 필수)

| 단원 | 활용 방법 |
|------|-----------|
| 일차함수 | 슬라이더로 $a, b$ 변화 → 그래프 동적 관찰 |

| 순환소수 | 계산기로 $\frac{1}{7}, \frac{2}{7}$... 순환마디 패턴 발견 |

## ✅ 핵심 원칙

> **중2 수학의 본질**
> 
> **대수**: 유리수 체계 완성, 지수법칙과 다항식의 **구조적 사고**

> **확률**: 결정론적 세계 → **비결정론적 세계**로 사고 전환
