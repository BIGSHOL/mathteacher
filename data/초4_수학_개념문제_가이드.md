# 초등학교 4학년 수학 개념 문제 개발 가이드
> 2022 개정 교육과정 기반

## 📌 4학년 수학의 특징

- **전환점**: 구체적 조작기 → 형식적 조작기로 넘어가는 과도기
- **핵심 변화**: 추상화 수준이 급격히 높아지는 시기
- **교육 방향**: 단순 연산 숙달 → 수학적 문해력과 개념적 이해 강조

## 📗 1학기 단원별 정리

### 1단원: 큰 수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 10배의 원리 (자릿수가 왼쪽으로 한 칸 → 10배)<br>• 4자리씩 끊어 읽기 (만, 억, 조)<br>• 수의 크기 비교 (자릿수 → 최상위 자릿수부터 비교) |
| **주요 오개념** | • 0이 포함된 수에서 빈 자리 누락 (예: "삼억 오천만" → 35000으로 표기)<br>• 자릿값과 자릿수 혼동 (5가 나타내는 값 ≠ 숫자 5)<br>• 큰 수의 양감 부족 |
| **출제 포인트** | ✓ 연속된 0이 있는 수의 받아쓰기<br>✓ 특정 단위가 생략된 수 읽기/쓰기<br>✓ 국가 예산, 인구 등 실생활 데이터 연결 |

---

### 2단원: 각도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 각의 크기 = 두 변이 벌어진 정도 (변의 길이와 무관)<br>• 각의 분류: 예각(0°~90°), 직각(90°), 둔각(90°~180°)<br>• 삼각형 내각의 합 = 180° / 사각형 = 360° |
| **주요 오개념** | • 변이 길면 각도 크다고 착각<br>• 각도기 안쪽/바깥쪽 눈금 혼동 (둔각을 60°로 읽음)<br>• 삼각형 모양이 달라지면 내각의 합도 변한다고 오해 |
| **출제 포인트** | ✓ 변의 길이는 다르나 각이 같은 도형 비교<br>✓ 다양한 삼각형의 내각의 합 확인 문제 |

### 3단원: 곱셈과 나눗셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • (세 자리 수) × (두 자리 수) 알고리즘<br>• (세 자리 수) ÷ (두 자리 수): 가늠 → 곱하기 → 빼기 → 확인<br>• 검산: 나누는 수 × 몫 + 나머지 = 나누어지는 수 |
| **주요 오개념** | • 곱셈 세로셈에서 자릿수 맞추기 오류<br>• 나눗셈 몫 가늠하기 어려움<br>• 나머지의 맥락적 해석 실패 (올림/버림 판단) |
| **출제 포인트** | ✓ 몫의 범위 어림하기 문항<br>✓ 잘못된 계산 과정에서 오류 찾기<br>✓ 실생활 문장제에서 나머지 처리 판단 |

### 4단원: 평면도형의 이동

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **밀기**: 위치만 변화, 모양·크기·방향 불변<br>• **뒤집기**: 대칭축 기준 좌우/상하 반전<br>• **돌리기**: 중심점 기준 90°, 180°, 270°, 360° 회전 |
| **주요 오개념** | • 오른쪽 뒤집기 ↔ 시계방향 180° 돌리기 혼동<br>• 시계/반시계 방향 혼동<br>• 90° 돌리기 ↔ 270° 돌리기 결과 착각 |
| **출제 포인트** | ✓ 비대칭 도형(ㄱ, ㄴ, P 등) 활용 필수<br>✓ 뒤집기+돌리기 복합 적용 문제<br>✓ 기준선(변) 추적하는 분석적 접근 유도 |

| **주요 오개념** | • 눈금 한 칸을 무조건 1로 설정<br>• 눈금선 중간값 읽기 어려움<br>• 자료 성격과 그래프 종류 매칭 오류 |
| **출제 포인트** | ✓ 적절한 눈금 크기 결정하기<br>✓ 중간값 어림하여 읽기 |

### 6단원: 규칙 찾기

## 📘 2학기 단원별 정리

### 1단원: 분수의 덧셈과 뺄셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 분모가 같은 분수: 분모는 그대로, 분자끼리 연산<br>• 대분수 덧셈: 자연수끼리 + 분수끼리 (또는 가분수로 변환)<br>• 대분수 뺄셈 받아내림: 자연수에서 1 빌려와 가분수로 변환 |
| **주요 오개념** | • **⭐ 분모끼리 더하기**: ⅓ + ⅓ = 2/6 (가장 흔한 오류)<br>• 자연수 - 분수 계산 오류 (3 - ⅖ 처리 못함)<br>• 받아내림 시 분자에 10 더함 (분모만큼 더해야 함) |
| **출제 포인트** | ✓ 파이, 막대 등 그림 모델과 식 연결<br>✓ 자연수를 가분수로 바꾸는 재그룹화 과정 시각화 |

### 2단원: 삼각형

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **이등변삼각형**: 두 변 같음, 두 밑각 같음<br>• **정삼각형**: 세 변 같음, 세 각 모두 60°<br>• **각에 따른 분류**: 예각/직각/둔각삼각형<br>• 포함 관계: 정삼각형 ⊂ 이등변삼각형 |
| **주요 오개념** | • 예각이 하나만 있어도 예각삼각형이라 착각<br>• 회전된 이등변삼각형 인식 못함<br>• "정삼각형은 이등변삼각형인가?" → "아니오" 오답 |
| **출제 포인트** | ✓ 둔각삼각형에서 예각 찾고 오답 유도<br>✓ 다양한 방향의 도형 제시<br>✓ 포함 관계 확인 문제 |

### 3단원: 소수의 덧셈과 뺄셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 0.01, 0.001의 의미 (1 = 0.1 × 10 = 0.01 × 100)<br>• 소수점 앞부터 차례로 크기 비교<br>• **소수점 위치 맞추어 세로로 정렬** |
| **주요 오개념** | • **소수가 길면 크다고 착각** (0.125 > 0.5 오류)<br>• 자릿수 정렬 오류 (2.5 + 12에서 5와 2를 더함)<br>• 3.5 = 3.50 이해 못함 |
| **출제 포인트** | ✓ 자릿값 판(Place Value Chart) 활용<br>✓ 소수점 기준 정렬 강조 문항 |

### 4단원: 사각형

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **수직**: 두 직선이 만나 직각 형성<br>• **평행**: 두 직선이 만나지 않음<br>• **분류 체계**:<br>  - 사다리꼴: 평행한 변 1쌍 이상<br>  - 평행사변형: 평행한 변 2쌍<br>  - 마름모: 네 변 같음<br>  - 직사각형: 네 각 직각<br>  - 정사각형: 네 각 직각 + 네 변 같음 |
| **주요 오개념** | • **⭐ 포함 관계 이해 부족**<br>  - "정사각형은 직사각형인가?" → 예<br>  - "정사각형은 마름모인가?" → 예<br>  - "직사각형은 평행사변형인가?" → 예<br>• 평행선 사이 거리를 비스듬히 잼<br>• 마름모를 다이아몬드 형태로만 인식 |
| **출제 포인트** | ✓ 벤다이어그램으로 포함 관계 평가<br>✓ "평행사변형을 모두 고르시오"에 직사각형·정사각형 포함<br>✓ 다양한 방향의 마름모 제시 |

| 구분 | 내용 |

## 💡 문제 개발 전략

### 1. 오개념 격파 문항 설계
- **반례 찾기**: "네 변의 길이가 같으면 모두 정사각형이다" → 참/거짓 + 반례(마름모)
- **비표준적 예시**: 회전되거나 변형된 도형, 복잡한 맥락 속 연산

### 2. 수학적 문해력 강화
- 계산기로 큰 수의 규칙 찾기 등 공학적 도구 활용 상황
- "왜 그렇게 생각했는지 설명하시오" 서술형 문항

### 3. 학년 간 연계
- 4학년 사각형 → 5학년 도형 넓이 공식 유도의 기초
- 도형의 등적 변형 개념 암시적 포함
