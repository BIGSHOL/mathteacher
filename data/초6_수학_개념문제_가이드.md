# 초등학교 6학년 수학 개념 문제 개발 가이드
> 2022 개정 교육과정 기반

## 📌 6학년 수학의 특징

- **위상**: 초등 수학의 완성 + 중등 수학(대수, 기하)의 기초
- **인지 발달**: 구체적 조작기 → 형식적 조작기 과도기
- **교육 방향**: 계산 숙달 → 연산의 의미 파악, 개념의 연결성과 원리 이해 강조
- **핵심 변화**: '계산하기' → '이해하고 설명하기', '추론하기'로 평가 주안점 이동

## 📗 1학기 단원별 정리

### 1단원: 분수의 나눗셈 (자연수÷자연수, 분수÷자연수)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 나눗셈의 몫을 분수로 표현: $a \div b = \frac{a}{b}$<br>• $(분수) \div (자연수)$: $\frac{a}{b} \div c = \frac{a}{b} \times \frac{1}{c}$<br>• 나눗셈 기호(÷)와 분수 막대의 동일한 의미 이해 |
| **주요 오개념** | • "나눗셈 하면 숫자가 작아진다" (자연수 경험칙에 갇힘)<br>• 교환법칙 오류: 작은 수÷큰 수 상황에서 순서 뒤바꿈<br>  (예: 3L를 4명이 나눔 → $4\div3$으로 계산) |
| **출제 포인트** | ✓ 숫자 크기만 보고 식 세우면 틀리는 문항<br>  (예: "2m 끈을 5등분" → 정답 $\frac{2}{5}$m, 오답 $\frac{5}{2}$m)<br>✓ 넓이 모델로 원리 설명하기 |

### 2단원: 각기둥과 각뿔

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 분류 기준: 밑면의 모양에 따라 이름 결정<br>• 구성 요소: 밑면, 옆면, 모서리, 꼭짓점, 높이<br>• **각기둥 높이**: 두 밑면 사이의 거리<br>• **각뿔 높이**: 꼭짓점에서 밑면에 내린 수선의 길이<br>• 전개도 접었을 때 맞닿는 선분/점 찾기 |
| **주요 오개념** | • **⭐ "밑면 = 바닥에 있는 면"** (회전/눕힌 도형에서 오류)<br>• 각뿔의 높이 ↔ 모선(옆 모서리) 길이 혼동<br>• 오일러 정리 공식 단순 암기 |
| **출제 포인트** | ✓ 옆으로 누운 삼각기둥에서 밑면/높이 찾기<br>✓ 각뿔 내부 수직 막대 vs 옆면 타고 내려오는 실 비교 그림<br>✓ 전개도에서 구성 요소 직접 세기 |

### 3단원: 소수의 나눗셈 (소수÷자연수)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 자연수 나눗셈과의 연결: $24.8 \div 4$는 $248 \div 4$의 $\frac{1}{10}$배<br>• 몫의 소수점 위치: 피제수의 소수점 위치를 그대로 올려 찍기<br>• 소수점 이동 = 단위의 변환/비율의 유지 |
| **주요 오개념** | • **⭐ 몫의 중간 0 누락**: $6.12 \div 3 = 2.4$ (정답: 2.04)<br>• 나머지 해석 오류: $13.4 \div 4 = 3$...나머지를 14나 1로 표기<br>  (나머지의 소수점은 원래 피제수 위치 따름) |
| **출제 포인트** | ✓ 몫의 중간에 0이 들어가는 문항 필수 포함<br>  (예: $6.12 \div 3$, $8.32 \div 4$)<br>✓ 자릿값 개념 테스트 |

### 4단원: 비와 비율 ⭐ (최고 난도)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **비의 구조**: $a:b$에서 $a$=비교하는 양, $b$=기준량<br>  (기호 ':'의 오른쪽이 기준)<br>• **비율**: $\frac{비교하는 양}{기준량}$ → 분수/소수로 표현<br>• **백분율**: 기준량을 100으로 환산한 비율 |
| **주요 오개념** | • **⭐ 기준량 찾기 실패**: "~에 대한"이 붙은 대상이 기준량<br>• 농도 문제에서 분모를 '물'로 잡음 (정답: 전체=소금+물)<br>• 문장 어순과 비의 순서 혼동 |
| **출제 포인트** | ✓ 문장 어순과 비 순서가 반대인 문제<br>  (예: "사과 3개, 배 4개. 배에 대한 사과의 비는?")<br>✓ 소금물 농도 함정 문제<br>  (예: "물 100g + 소금 25g → 진하기?" 오답 25%, 정답 20%) |

## 📘 2학기 단원별 정리

### 1단원: 분수의 나눗셈 (분수÷분수)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **통분 이용법**: 분모 같게 만들면 분자끼리 나눗셈<br>  $\frac{a}{m} \div \frac{b}{m} = a \div b$<br>• **알고리즘 형식화**: $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$<br>  (나눗셈→곱셈, 나누는 수의 역수) |
| **주요 오개념** | • **대분수 처리 미숙**: 대분수 상태에서 바로 역수 취함<br>  (반드시 가분수로 변환 후 계산)<br>• **앞의 수 역수 취하기**: 피제수를 뒤집는 오류 |
| **출제 포인트** | ✓ $2\frac{1}{2} \div 1\frac{1}{3}$ 같은 대분수÷대분수 문제<br>✓ 오류가 있는 계산 과정 제시 → 틀린 이유 서술하기 |

### 2단원: 소수의 나눗셈 (소수÷소수)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **소수점 이동의 동치성**: 나누는 수와 나누어지는 수에 같은 수를 곱해도 몫 불변<br>• 나머지 처리: 소수점 이동 후 계산해도 나머지는 이동 전 위치에 표기 |
| **주요 오개념** | • **⭐ 반올림 조건 혼동**<br>  - "소수 둘째 자리까지 구하시오" ≠ "소수 둘째 자리에서 반올림하시오" |
| **출제 포인트** | ✓ 두 가지 유형 교차 출제로 문제 조건 정독 습관 유도 |

### 4단원: 비례식과 비례배분

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **비례식의 성질**: 외항의 곱 = 내항의 곱 ($a \times d = b \times c$)<br>• **비의 성질**: 전항·후항에 0 아닌 같은 수를 곱/나눠도 비의 값 동일<br>• **비례배분**: $전체 \times \frac{해당 비}{비의 합}$ |
| **주요 오개념** | • **⭐ 덧셈적 사고의 잔재**<br>  $2:3 = 4:\square$ → "2가 4가 되려면 +2, 그러니 3도 +2 = 5"<br>  (정답: 배수 관계로 6)<br>• **비례배분 분모 오류**: 2:3으로 나눌 때 분모를 3으로 잡음<br>  (정답: 분모는 $2+3=5$) |
| **출제 포인트** | ✓ 덧셈적 추론이 통하지 않는 수 제시<br>  (예: $4:6 = 10:x$)<br>✓ 전체 막대 시각적 모델로 비례배분 확인 |

### 5단원: 원의 넓이

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **원주율(π)**: 지름에 대한 원주의 비율 (근삿값 3.14 사용)<br>• **넓이 공식 유도**: 원을 잘게 잘라 이어 붙이면 직사각형에 근접<br>  - 가로: 원주의 $\frac{1}{2}$, 세로: 반지름<br>  - $넓이 = \pi r^2$ |
| **주요 오개념** | • **⭐ 공식 혼용**: 원주($2\pi r$) ↔ 원의 넓이($\pi r^2$) 혼동<br>  - "지름 × 지름 × 원주율"로 넓이 계산하는 실수<br>• **반원/부채꼴 둘레 오류**: 곡선 부분만 계산, 직선(지름) 누락 |
| **출제 포인트** | ✓ 관계적 사고 문항: "반지름 2배 → 원주 몇 배? 넓이 몇 배?"<br>✓ 공식 구조 이해 확인 (단순 계산보다 원리) |

### 6단원: 원기둥, 원뿔, 구

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **회전체**: 평면도형을 한 직선(축)으로 1회전시켜 만든 입체<br>• **단면의 종류**<br>  - 회전축에 수직인 단면: 원<br>  - 회전축 포함 단면: 선대칭도형<br>• **원기둥 겉넓이**: 옆면(직사각형)의 가로 = 밑면의 원주 |

### 3. 오개념 활용 선지 구성 (예시)

> **문제**: 남학생 12명, 여학생 13명일 때 여학생 수에 대한 남학생 수의 비는?

| 선지 | 오개념 유형 |
|------|-------------|
| ① 13:12 | 단순 숫자 나열 순서 오류 |
| ② **12:13** | ✅ 정답 |
| ③ 12:25 | 전체에 대한 비로 착각 (Part to Whole) |
| ④ 13:25 | 여학생 기준 전체 비 오류 |
| ⑤ $\frac{12}{13}$ | 비가 아닌 비율 값 제시 |
