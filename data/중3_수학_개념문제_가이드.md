## 📗 1학기 단원별 정리

### 1단원: 실수와 그 연산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **제곱근의 정의**: $x^2 = a$일 때 $x$를 $a$의 제곱근<br>  → 양수의 제곱근은 **양수와 음수 두 개** 존재<br>• **근호 $\sqrt{}$의 의미**: $\sqrt{a}$ = 제곱하여 $a$가 되는 수 중 **양수**<br>• **실수의 분류**: 실수 = 유리수 + 무리수<br>  → 무리수 = **순환하지 않는 무한소수**<br>• **실수의 대소 관계**: $a - b$의 부호로 판단, 또는 근사값 이용<br>• **근호를 포함한 식의 계산**:<br>  - $\sqrt{a}\sqrt{b} = \sqrt{ab}$ (단, $a>0, b>0$)<br>  - **분모의 유리화**: $\frac{1}{\sqrt{a}} = \frac{\sqrt{a}}{a}$<br>  - $\frac{c}{\sqrt{a}+\sqrt{b}} = \frac{c(\sqrt{a}-\sqrt{b})}{a-b}$ |
| **주요 오개념** | • **⭐⭐⭐ "$a$의 제곱근"과 "제곱근 $a$($\sqrt{a}$)" 혼동**<br>  - "$a$의 제곱근(Square root of $a$)" = 방정식 $x^2=a$의 해<br>  - "제곱근 $a$" = 기호 $\sqrt{a}$를 읽은 것<br>  → 예: "25의 제곱근" = $\pm 5$, "$\sqrt{25}$" = $5$<br>• **⭐ $\sqrt{a^2} = a$ 오류**: $a < 0$일 때 성립 안 함<br>  → 정확한 정의: $\sqrt{a^2} = |a|$<br>• **"모든 무한소수는 무리수" 착각**: 순환소수는 유리수!<br>• **⭐⭐ $\sqrt{2} + \sqrt{3} = \sqrt{5}$ 오류 (선형성 오류)**<br>  → $\sqrt{2} \approx 1.414$, $\sqrt{3} \approx 1.732$ → 합 ≈ 3.146<br>  → $\sqrt{5} \approx 2.236$ (전혀 다름!)<br>• **분배법칙 오류**: 근호 안/밖을 무분별하게 곱함 |
| **출제 포인트** | ✓ "25의 제곱근"과 "$\sqrt{25}$"를 구분하는 문항 (단골 함정)<br>✓ $\sqrt{(-3)^2}$의 값 묻기 (답: 3, $|-3|$)<br>✓ "모든 무한소수는 무리수이다" 참/거짓 (반례: 순환소수)<br>✓ 유리화의 목적: 근사값 계산, 통분 용이 |

### 2단원: 식의 계산 (다항식의 곱셈과 인수분해)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **분배법칙의 확장**: $(a+b)(c+d) = ac + ad + bc + bd$<br>  → 직사각형 넓이 모델(Area Model)로 시각화<br>• **곱셈 공식**:<br>  - 완전제곱식: $(a \pm b)^2 = a^2 \pm 2ab + b^2$<br>  - 합차 공식: $(a+b)(a-b) = a^2 - b^2$<br>  - $(x+a)(x+b) = x^2 + (a+b)x + ab$<br>  - $(ax+b)(cx+d) = acx^2 + (ad+bc)x + bd$<br>• **인수분해**: 다항식을 곱의 형태로 (전개의 역연산)<br>  - 공통인수 묶기: $ma + mb = m(a+b)$<br>  - X자형 분리법: 합과 곱 조건 만족하는 두 수 찾기 |
| **주요 오개념** | • **⭐⭐⭐ 중간항 누락 (가장 유명한 오류)**<br>  $(x+3)^2 = x^2 + 9$ ✗<br>  → $2ab$ 항의 기하학적 의미(직사각형 두 개 넓이) 강조<br>• **부호 혼동**: $(x-a)^2 = (a-x)^2$ 성립 인식 못함<br>  → $(-A)^2 = A^2$<br>• **⭐ 불완전한 인수분해**: $2x^2 - 8 = 2(x^2 - 4)$에서 멈춤<br>  → 끝까지: $2(x+2)(x-2)$<br>• **⭐ 방정식과 혼동**: 인수분해 결과에 "$x=2$ 또는 $x=3$" 덧붙임<br>  → 식의 변형(인수분해) ≠ 등식의 풀이(방정식)<br>• **완전제곱식 판별**: 일차항 계수의 반의 제곱 원리 단순 암기<br>  → 이차항 계수가 1이 아닐 때($4x^2 + ...$) 오류 발생 |
| **출제 포인트** | ✓ $(x+3)^2$ 전개에서 중간항 확인<br>✓ 완전제곱식 조건: $x^2 + 10x + \Box$ → $\Box = (\frac{10}{2})^2 = 25$<br>✓ 불완전 인수분해 찾기 문제<br>✓ "인수분해하시오"에서 해 구하지 않기 |

### 3단원: 이차방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **정의**: $ax^2 + bx + c = 0$ ($a \neq 0$)<br>• **풀이 방법**:<br>  ① 인수분해: $AB = 0 \Leftrightarrow A=0$ or $B=0$<br>  ② 제곱근 이용: $(x-p)^2 = q$ → $x = p \pm \sqrt{q}$<br>  ③ 완전제곱식: 제곱 완성(Completing the Square)<br>  ④ **근의 공식**: $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$<br>• **판별식**: $D = b^2 - 4ac$<br>  - $D > 0$: 서로 다른 두 실근<br>  - $D = 0$: 중근 (서로 같은 두 실근)<br>  - $D < 0$: 실근 없음<br>• **근과 계수의 관계(심화)**: 두 근의 합 = $-\frac{b}{a}$, 곱 = $\frac{c}{a}$ |
| **주요 오개념** | • **⭐⭐ 이차항 계수 조건 간과**: "$(a-2)x^2 + ... = 0$이 이차방정식"이면<br>  → $a \neq 2$ 조건 필수 (단골 출제)<br>• **중근의 의미 오해**: "한 개의 근"이 아니라 **"서로 같은 두 실근"**<br>• **판별식 음수**: 중학교에서는 "해가 없다"로 답해야 함 (허수 X)<br>• **⭐ 물리적 타당성 무시**: 길이, 시간 구하는 문제에서 음수 해를 그대로 답함<br>  → 문제 뜻에 맞는지 검산 필수 |
| **출제 포인트** | ✓ "이차방정식이 되기 위한 조건" ($a \neq ?$)<br>✓ 중근을 가질 조건 (판별식 = 0)<br>✓ 활용 문제: 음수 해 제외 여부 판단 |

### 4단원: 이차함수 ⭐ (최대·최소 신설)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **기본형** $y = ax^2$: 원점 꼭짓점, $y$축이 축<br>  - $a > 0$: 아래로 볼록 / $a < 0$: 위로 볼록<br>  - $|a|$ 클수록 폭 좁음 ($y$축에 가까워짐)<br>• **표준형** $y = a(x-p)^2 + q$: 꼭짓점 $(p, q)$, 축 $x = p$<br>• **일반형** $y = ax^2 + bx + c$: 표준형으로 변형하여 꼭짓점 파악<br>  - $c$ = $y$절편<br>• **최대·최소** (2022 개정 신설):<br>  - $a > 0$: 꼭짓점에서 **최솟값 $q$**, 최댓값 없음<br>  - $a < 0$: 꼭짓점에서 **최댓값 $q$**, 최솟값 없음<br>  - 완전제곱식: $(x-p)^2 \geq 0$임을 이용해 정당화 |
| **주요 오개념** | • **⭐⭐ 평행이동 부호 혼동**:<br>  $x$축으로 $p$만큼 이동 → $y = a(x-p)^2$ (부호 반대!)<br>  → 점의 이동 $(x, y) \to (x+p, y)$와 함수 이동은 다름<br>• **⭐ $b$의 부호 결정 오류**: 그래프 보고 $a, b, c$ 부호 맞출 때<br>  → 축 $x = -\frac{b}{2a}$의 위치로 추론<br>  → 축이 $y$축 오른쪽($-\frac{b}{2a} > 0$) → $a$와 $b$ 부호 반대<br>• **증가/감소 구간**: 축을 기준으로 좌/우가 나뉨을 간과<br>• **⭐⭐⭐ 최댓값/최솟값을 $x$좌표로 답함**<br>  "$x=1$일 때 최댓값 5" → 답을 1이라고 씀<br>  → 최댓값/최솟값은 항상 **함숫값($y$)** |
| **출제 포인트** | ✓ 표준형 → 꼭짓점, 축 찾기<br>✓ "최댓값은?" 질문에 $y$값으로 답하는지 확인<br>✓ 완전제곱식 변형 이유 서술 ($(x-p)^2 \geq 0$ 이용)<br>✓ 그래프 보고 $a, b, c$ 부호 판별 |

## 📘 2학기 단원별 정리

### 5단원: 삼각비

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **정의** (직각삼각형, 기준각 $A$):<br>  - $\sin A = \frac{높이(대변)}{빗변}$<br>  - $\cos A = \frac{밑변}{빗변}$<br>  - $\tan A = \frac{높이}{밑변}$<br>• **특수각**: $30°, 45°, 60°$ 값 (정삼각형, 직각이등변삼각형 유도)<br>• **삼각비의 변화**: 각이 커질 때 $\sin$↑, $\cos$↓<br>• **사분원**: 반지름 1인 원에서 $0°, 90°$ 삼각비 정의<br>• **삼각형 넓이**: $S = \frac{1}{2}ab\sin x$ (두 변과 끼인각) |
| **주요 오개념** | • **⭐⭐ 변의 위치 착각**: 삼각형이 회전되면 밑변/높이 못 찾음<br>  → **빗변 = 직각의 대변** (항상 고정)<br>  → **높이 = 기준각의 대변(맞은편)**<br>• **삼각비 크기 관계 혼동**:<br>  $\tan 45° = 1$ → $45° < A < 90°$이면 $\tan A > 1$<br>  $\sin A, \cos A$는 항상 1 이하<br>  → 대소 비교 문제($\sin 50°$ vs $\tan 50°$)에서 오류 |
| **출제 포인트** | ✓ 회전된 삼각형에서 삼각비 구하기<br>✓ $\sin 50°$, $\cos 50°$, $\tan 50°$ 크기 비교<br>✓ 특수각 삼각비 값 유도 과정 |

### 6단원: 원의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **현의 수직이등분선**: 원의 중심에서 현에 내린 수선은 현을 이등분<br>  → 역도 성립: 현의 수직이등분선은 원의 중심 지남<br>• **접선의 성질**: 원 밖의 한 점에서 그은 두 접선의 길이 같음<br>  → RHS/RHA 합동으로 증명<br>• **원주각과 중심각**: 같은 호에 대해 원주각 = $\frac{1}{2}$ × 중심각<br>• **원주각의 성질**:<br>  - 같은 호에 대한 원주각은 모두 같음<br>  - **반원(지름)의 원주각 = 90°**<br>• **원에 내접하는 사각형**: 대각의 합 = 180°<br>• **접현각**: 접선과 현이 이루는 각 = 그 현의 내부 호에 대한 원주각 |
| **주요 오개념** | • **⭐ 접선 문제에서 직각 표시 안 함**: 접점에서 반지름 긋고 직각 표시 필수<br>  → "접선이 나오면 무조건 중심→접점 반지름 + 직각 표시"<br>• **⭐⭐ 중심각 착각**: 원 내부의 아무 점을 중심으로 착각<br>  → 문제에 점 $O$ 표시 반드시 확인<br>• **이등변삼각형 간과**: 반지름 연결하면 이등변삼각형 형성<br>  → 각도 계산에 활용 |
| **출제 포인트** | ✓ "원주각"과 "중심각" 구분 문제<br>✓ 반원의 원주각 = 90° 활용<br>✓ 내접사각형 대각의 합 |

### 7단원: 통계 ⭐⭐ (2022 개정 핵심 변화)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **대푯값**: 평균, 중앙값, 최빈값<br>  → 아웃라이어 있을 때는 **중앙값이 유리**<br>• **산포도**: 분산, 표준편차<br>  - 편차 = (변량) - (평균), 편차의 합 = 항상 0<br>  - 분산 = 편차 제곱의 평균<br>  - 분산: 단위 없음(제곱 상태), 표준편차: 원래 자료 단위<br>• **산점도와 상관관계**:<br>  - 양의 상관관계(우상향), 음의 상관관계(우하향), 상관관계 없음<br>  - **상관관계 ≠ 인과관계** |
| **⭐⭐ 사분위수와 상자그림 (신설/강화)** | • **사분위수**:<br>  - $Q_1$ (제1사분위수): 하위 50%의 중앙값<br>  - $Q_2$ (제2사분위수): 전체 중앙값<br>  - $Q_3$ (제3사분위수): 상위 50%의 중앙값<br>• **사분위범위(IQR)**: $Q_3 - Q_1$ (중간 50%의 퍼진 정도)<br>• **이상치(Outlier) 판별 기준**:<br>  - $Q_1 - 1.5 \times IQR$ 보다 작은 값<br>  - $Q_3 + 1.5 \times IQR$ 보다 큰 값<br>• **상자그림 그리기**:<br>  - 상자: $Q_1$ ~ $Q_3$, 내부에 $Q_2$ 선<br>  - 수염: 이상치 **제외한** 최솟값/최댓값까지<br>  - **중요**: 수염은 경계값이 아닌 **실제 데이터 값**까지 |
| **주요 오개념** | • **⭐ 편차 계산 순서**: (변량) - (평균)인데 거꾸로 계산<br>• **⭐⭐⭐ 수염 길이 오류**: 경계값($Q_3 + 1.5 \times IQR$)까지 긋는 실수<br>  → 수염은 **실제 데이터 값**까지만!<br>• **⭐⭐ 상자 구간 = 데이터 개수 착각**: 구간이 길면 데이터 많다고 생각<br>  → 각 구간에는 **25%씩 동일한 개수**<br>  → 구간 길다 = 데이터가 듬성듬성(밀도 낮음)<br>• **$Q_2$ = 평균 착각**: 상자 내부 선($Q_2$)은 **중앙값**<br>  → 상자그림만으로는 평균을 알 수 없음 |
| **출제 포인트** | ✓ 상자그림 두 개 비교: "A반이 B반보다 고르다" 참/거짓 (IQR 비교)<br>✓ 이상치 판별 기준 적용<br>✓ 수염이 어디까지인지 (경계값 vs 실제 데이터)<br>✓ 상관관계 ≠ 인과관계 구분 |

| 1-1 | 식의 계산 | ★★★ | 불완전한 인수분해 | 끝까지 분해했는지 확인 |
| 1-1 | 이차방정식 | ★★★★ | 이차항 계수 조건 ($a \neq 0$) 간과 | 이차방정식 되기 위한 조건 |
| 1-1 | 이차함수 | ★★★★ | 평행이동 부호 혼동 | 점 이동 vs 함수 이동 |
| 1-1 | 이차함수 | ★★★★★ | **최댓값을 $x$좌표로 답함** | 최댓값 = 항상 $y$값 |
| 1-2 | 삼각비 | ★★★ | 회전된 삼각형에서 변 못 찾음 | 빗변 = 직각의 대변 |
| 1-2 | 원의 성질 | ★★★ | 중심각 착각 (아무 점을 중심으로) | 점 $O$ 표시 확인 |
| 1-2 | 통계 | ★★★★★ | **수염을 경계값까지 긋는 오류** | 실제 데이터 값까지만 |
| 1-2 | 통계 | ★★★★ | 상자 구간 길이 = 데이터 개수 | 각 구간 25%씩 동일 |

## 💡 문제 개발 전략

### 1. 제곱근 오개념 교정

| 질문 | 정답 | 오답 (흔한 실수) |
|------|------|------------------|
| "25의 제곱근은?" | $\pm 5$ | 5 |
| "$\sqrt{25}$는?" | 5 | $\pm 5$ |
| "$\sqrt{(-3)^2}$는?" | 3 (= $|-3|$) | -3 |

> **핵심**: 언어적 차이가 수학적 의미 차이를 만듦

### 2. 이차함수 최대·최소 (2022 신설)

| $a$의 부호 | 그래프 모양 | 꼭짓점에서 | 반대쪽 |
|-----------|-------------|-----------|--------|
| $a > 0$ | 아래로 볼록 (∪) | **최솟값** | 최댓값 없음 |
| $a < 0$ | 위로 볼록 (∩) | **최댓값** | 최솟값 없음 |

> **주의**: 답은 항상 **$y$값(함숫값)**으로!

---

### 3. 상자그림 핵심 (2022 강화)

| 구성 요소 | 의미 | 주의사항 |
|-----------|------|----------|
| 상자 | $Q_1$ ~ $Q_3$ | 중간 50% 데이터 |
| 상자 내 선 | $Q_2$ (중앙값) | **평균 아님!** |
| 수염 | 이상치 제외 최소~최대 | **경계값까지 아님!** |
| 각 구간 | 25%씩 동일 | **길이 ≠ 개수** |

#### 상자그림 오개념 비교

| 오개념 | 올바른 이해 |
|--------|-------------|
| 수염을 $Q_3 + 1.5 \times IQR$까지 | 실제 데이터 값까지만 |
| 구간 길면 데이터 많다 | 25%씩 동일, 길면 밀도 낮음 |
| 상자 내 선 = 평균 | 중앙값 ($Q_2$)임 |
