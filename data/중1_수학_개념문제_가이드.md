# 중학교 1학년 수학 개념 문제 개발 가이드
> 2022 개정 교육과정 기반

## 📌 중1 수학의 특징

- **위상**: 초등 수학(직관적 단계) → 중등 수학(추상적·논리적 단계)의 결정적 전환기
- **인지적 도약**: 산술적 사고(Arithmetic) → 대수적 사고(Algebraic)
- **수 체계 확장**: 자연수 → 정수 → 유리수
- **2022 개정 핵심 변화**:
  - 최대공약수/최소공배수 복잡한 활용 문제 **삭제**
  - 대푯값, 산점도 **중3 → 중1로 이동** (데이터 리터러시 강화)
  - '문자와 식' + '함수' → **'변화와 관계'** 영역으로 통합

> 💡 개념 문제는 단순 연산 숙달이 아닌 **'개념적 이해(Conceptual Understanding)'** 평가로 전환

## 📗 1학기 단원별 정리

### 1단원: 소인수분해

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **소수와 합성수**: 소수는 약수가 2개(1과 자신), 합성수는 약수가 3개 이상<br>• **1은 소수도 합성수도 아님** (산술의 기본 정리 - 유일성 조건)<br>• **거듭제곱**: 밑(base)과 지수(exponent)의 역할 구분<br>• **소인수분해**: 모든 합성수는 소수들의 곱으로 유일하게 표현<br>• **최대공약수(GCD) / 최소공배수(LCM)**: 소인수분해 형태에서 지수 비교 |
| **주요 오개념** | • **1을 소수로 분류**: "약수가 하나뿐이므로 가장 단순한 소수"<br>• **⭐ 서로소 의미 혼동**: "서로소인 두 수는 둘 다 소수여야 한다"<br>  (반례: 4와 9는 서로소이지만 둘 다 합성수)<br>• **지수와 밑 혼동**: $2^3 = 2 \times 3 = 6$으로 계산<br>• **최대/최소 직관 오류**: "최대공약수니까 큰 수", "최소공배수니까 작은 수" |
| **출제 포인트** | ✓ "왜 1은 소수에서 제외되는가?" 심층 이해<br>✓ 서로소는 '두 수 사이의 관계'임을 강조<br>✓ "두 수의 공배수는 최소공배수의 배수이다" 같은 **수의 성질** 문항<br>⚠️ **삭제 유형**: 타일 붙이기, 톱니바퀴, 버스 동시 출발 등 작위적 활용 |

### 2단원: 정수와 유리수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **음수의 도입**: 서로 반대되는 성질 표현 (이익/손해, 영상/영하)<br>• **수직선 모델**: 수의 크기와 순서 시각화<br>• **절대값**: 원점으로부터의 거리 (기하학적 정의)<br>• **유리수의 사칙연산**: 부호 결정 규칙의 논리적 이해<br>• **유리수의 조밀성**: 두 유리수 사이에 무수히 많은 유리수 존재 |
| **주요 오개념** | • **⭐⭐ 가장 치명적: "$-a$는 반드시 음수이다"**<br>  ($a = -3$이면 $-a = +3$임을 이해 못함)<br>  → 마이너스 기호는 '부호를 반대로 바꿈'의 연산자<br>• **분수 형태 = 정수 아님**: "$\frac{6}{2}$는 분수이므로 정수 아니다"<br>  (약분하면 정수 3)<br>• **절대값 대소 관계 오류**: "절대값이 큰 수가 항상 더 크다"<br>  (음수는 절대값이 클수록 작아짐) |
| **출제 포인트** | ✓ 계산 법칙 성립 여부: "정수의 뺄셈에서 교환법칙이 성립하지 않음을 보이는 반례"<br>✓ "1과 2 사이에 존재하는 유리수는 몇 개인가?" (무수히 많다)<br>✓ 수 체계 분류 벤 다이어그램 활동 |

### 3단원: 문자의 사용과 식의 계산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **대수적 언어 습득**: 수량 관계를 문자로 표현 (일반화, 간결성)<br>• **대수적 관습**: 곱셈 기호 생략, 나눗셈의 분수 표현<br>• **동류항**: 문자와 차수가 같은 항끼리 계산<br>• **분배법칙**: 괄호를 풀어 식을 간단히 함<br>• **대입과 식의 값**: 변수(Variable) 개념 정착 |
| **주요 오개념** | • **⭐ 식과 방정식 혼동**: $2x + 4$를 보고 무의식적으로 $= 0$ 붙여서 $x = -2$로 품<br>  → 등식(Equation) vs 식(Expression) 구분 필수<br>• **분배법칙 부호 오류**: $-(2x - 3) = -2x - 3$<br>  (괄호 안 **모든 항**에 $-1$ 곱해야 함)<br>• **역수 취하기 오류**: $\frac{2}{3}x$의 역수를 $\frac{3}{2}x$로 착각<br>  ($\frac{2}{3}x = \frac{2x}{3}$임을 먼저 인식) |
| **출제 포인트** | ✓ "식을 간단히 하시오" vs "방정식을 푸시오" 구분<br>✓ 분배법칙 시각적 화살표로 '분배' 과정 추적<br>✓ 역수 변환 전 분수 형태 정리 |

### 4단원: 일차방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **등식의 성질**: 양변에 같은 수를 ±×÷ 해도 등식 성립<br>• **이항**: 등식의 성질에서 파생된 결과 (단순 규칙 아님)<br>• **방정식의 해**: 방정식을 참이 되게 하는 미지수의 값<br>• **항등식**: 어떤 값을 대입해도 항상 참 ($a=c, b=d$일 때) |
| **주요 오개념** | • **이항 시 부호 유지**: 항을 옮길 때 부호를 안 바꿈<br>• **계수 처리 오류**: $3x = 6$에서 $x = 6 - 3$으로 계산<br>• **소수/분수 계수 처리**: $0.1x = 2$에서 $x = 0.2$로 착각 |
| **출제 포인트** | ✓ 해의 검증 과정: "$x = -2$가 해가 되는 방정식은?"<br>✓ 항등식 조건: "$3(x-1) = ax + b$가 항등식이 되도록 하는 $a, b$는?" |

### 5단원: 좌표평면과 그래프

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **순서쌍과 좌표**: 평면 위의 점을 $(x, y)$로 표현 (데카르트 좌표계)<br>• **사분면의 부호 특성**: 1사분면(+,+), 2사분면(-,+), 3사분면(-,-), 4사분면(+,-)<br>• **그래프 해석**: 변화의 경향(증가, 감소, 일정, 주기적) 읽기 |
| **주요 오개념** | • **⭐ 축 위의 점 사분면 오류**: "$(3, 0)$은 제1사분면 또는 제4사분면"<br>  → 좌표축은 어느 사분면에도 포함 안 됨<br>• **변수 순서 혼동**: $(2, 3)$과 $(3, 2)$를 같은 점으로 인식 |
| **출제 포인트** | ✓ 물통 모양에 따른 수면 높이 변화 그래프 **정성적(Qualitative)** 유추<br>✓ 축 위의 점 분류 문제 |

### 6단원: 정비례와 반비례

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **정비례** ($y = ax$, $a \neq 0$): $x$가 2배 → $y$도 2배, 그래프는 원점 지나는 직선<br>• **반비례** ($y = \frac{a}{x}$, $a \neq 0$): $x$가 2배 → $y$는 $\frac{1}{2}$배, 그래프는 쌍곡선<br>• **$xy = a$ (일정한 곱)**: 반비례의 핵심 성질 |
| **주요 오개념** | • **반비례 그래프 직선화**: 점들을 선분으로 연결하려 함<br>  → $x$값 사이사이 무수히 많은 점을 찍으면 곡선됨<br>• **⭐ 증감 상태 일반화 오류**: "$y = \frac{2}{x}$에서 $x$ 증가하면 $y$ 감소"를 전체 구간에 적용<br>  → $x < 0$과 $x > 0$ 구간 **각각**에서 감소, 0 건너뛰는 불연속성 인지 |
| **출제 포인트** | ✓ 식만 보고 그래프 개형 유추<br>✓ 반비례에서 직사각형 넓이 일정함 연결 (기하학적 접근) |

## 📘 2학기 단원별 정리

### 1단원: 기본 도형과 작도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **기본 요소**: 점, 선, 면의 정의와 위치 관계<br>• **두 직선의 위치 관계**: 만남, 평행, **꼬인 위치(Skew Lines)**<br>  → 꼬인 위치: 만나지도 않고 평행하지도 않은 (공간에서만)<br>• **평행선의 성질**: 동위각, 엇각 → "평행하면 크기가 같다" + 역도 성립<br>• **작도**: 눈금 없는 자 + 컴퍼스로 수직이등분선, 각의 이등분선<br>• **삼각형 합동 조건**: SSS, SAS, ASA |
| **주요 오개념** | • **⭐ 꼬인 위치 찾기 실패**: 평행한 모서리를 꼬인 위치로 착각<br>• **동위각 항상 같다고 착각**: "동위각은 항상 같다"<br>  → **평행할 때만** 같음! 평행 아닌 경우 반례 필수<br>• **⭐ SSA 합동 오류**: 끼인 각이 아닌 다른 한 각이 같을 때도 합동이라 판단<br>  → 삼각형 하나로 결정 안 되는 작도 반례(Ambiguous Case) 제시 |
| **출제 포인트** | ✓ 직육면체 구체물로 꼬인 위치 직접 확인<br>✓ 평행하지 않은 두 직선의 동위각 반례 그림<br>✓ SAS에서 **'끼인 각(Included Angle)'** 중요성 각인 |

### 2단원: 평면도형의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **다각형 내각의 합**: $(n-2) \times 180°$<br>• **다각형 외각의 합**: 항상 $360°$<br>• **삼각형 외각 성질**: 한 외각 = 이웃하지 않는 두 내각의 합<br>• **원과 부채꼴**: 중심각에 정비례(호의 길이, 넓이) vs 비정비례(현의 길이)<br>• **부채꼴 넓이**: $S = \pi r^2 \times \frac{x}{360}$ 또는 $S = \frac{1}{2}rl$ |
| **주요 오개념** | • **원주율 $\pi$ 취급 오류**: 문자로 취급해 단위 생략하거나 순서 혼동<br>  → $\pi$는 3.14...를 나타내는 **특정 상수**, $10\pi$ cm처럼 표기<br>• **중심각 2배 → 현의 길이 2배?**: 거짓! (삼각형 성질에 의해 2배보다 작음) |
| **출제 포인트** | ✓ "중심각이 2배가 되면 현의 길이도 2배가 되는가?" 참/거짓 문항<br>✓ 복잡한 도형(별 모양)에서 외각 성질 활용 |

### 3단원: 입체도형의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **다면체**: 면, 모서리, 꼭짓점 개수 (오일러 공식 탐구)<br>• **회전체 단면**: 회전축에 수직/평행인 단면 모양<br>• **겉넓이와 부피**: 기둥, 뿔, 구 공식<br>• **뿔 부피 = 기둥 부피 × $\frac{1}{3}$**<br>• **구**: 부피 $\frac{4}{3}\pi r^3$, 겉넓이 $4\pi r^2$<br>• **아르키메데스의 묘비**: 원기둥 : 구 : 원뿔 = 3 : 2 : 1 |
| **주요 오개념** | • **⭐ 원뿔 겉넓이 오류**: 전개도 부채꼴 호의 길이 = 밑면 원주 연결 못함<br>• **단면 모양 단정**: 원기둥 비스듬히 자른 단면(타원)을 원으로 착각 |
| **출제 포인트** | ✓ 전개도 접었을 때 맞닿는 부분 시각적 추적<br>✓ 다양한 단면 모양 유추<br>⚠️ **지양**: 지나치게 복잡하게 조합된 입체도형 계산 |

### 4단원: 자료의 정리와 해석

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **줄기와 잎 그림 / 도수분포표**: 계급, 계급값, 도수 용어<br>• **히스토그램과 도수분포다각형**: 분포 시각화, 면적 = 도수 총합에 비례<br>• **상대도수**: $\frac{그 계급의 도수}{도수의 총합}$ (총합이 다른 집단 비교용) |

| **출제 포인트** | ✓ 계급값의 의미와 평균 추정 과정 이해<br>✓ 상대도수를 이용한 두 집단 비교 |

|------|------|
| **핵심 개념** | • **산점도(Scatter Plot)**: 두 변량 $x, y$를 좌표평면에 점으로 표현<br>• **양의 상관관계**: $x$↑ → $y$↑ (우상향 분포)<br>• **음의 상관관계**: $x$↑ → $y$↓ (우하향 분포)<br>• **상관관계 없음**: 원 모양 또는 축에 평행한 분포 |
| **주요 오개념** | • **⭐⭐ 인과관계 ↔ 상관관계 혼동**: "키와 몸무게가 양의 상관 → 키가 크기 때문에 몸무게가 많이 나간다"<br>  → 상관관계는 **경향성**일 뿐, 원인-결과 보장 안 함<br>• **선형성 오해**: "모든 점이 한 직선 위에 있어야 상관관계"<br>  → 분포의 **경향(Trend)**만으로도 성립 |
| **출제 포인트** | ✓ 산점도 위 보조선 긋고 특정 영역 학생 수 세기<br>✓ 실생활 예시로 상관관계 유형 판별<br>  - 양: 기온 ↔ 아이스크림 판매량<br>  - 음: 기온 ↔ 패딩 판매량<br>  - 무: 신발 크기 ↔ 지능 지수 |
