## 중학교 1학년

### 1단원: 소인수분해
- **주요 유형**
  - 소수 판별 (10%, 30초, ⭐)
  - 소인수분해 (15%, 1분, ⭐)
  - 약수의 개수/합 (20%, 1-2분, ⭐⭐)
  - 최대공약수/최소공배수 (25%, 2분, ⭐⭐)
  - 조건 역추적 (30%, 3-4분, ⭐⭐⭐)

### 2단원: 정수와 유리수
- **주요 유형**
  - 부호 결정 (15%, 30초, ⭐)
  - 사칙연산 (25%, 1분, ⭐)
  - 혼합 계산 (30%, 2분, ⭐⭐)
  - 절댓값 포함 (20%, 2분, ⭐⭐)
  - 조건 추론 (10%, 3분, ⭐⭐⭐)

### 3단원: 문자와 식 / 일차방정식
- **주요 유형**
  - 식의 값 (대입) (15%, 1분, ⭐)
  - 일차방정식 풀이 (25%, 1-2분, ⭐)
  - 분수/소수 방정식 (20%, 2분, ⭐⭐)
  - 활용 문제 (기본) (25%, 3분, ⭐⭐)
  - 활용 문제 (심화) (15%, 4-5분, ⭐⭐⭐)

### 4단원: 좌표와 그래프 / 정비례·반비례
- **주요 유형**
  - 좌표 읽기/찍기 (10%, 30초, ⭐)
  - 대칭점 (15%, 1분, ⭐)
  - 정비례/반비례 판별 (15%, 1분, ⭐)
  - 그래프 해석 (30%, 2분, ⭐⭐)
  - 비례상수 조건 (30%, 2-3분, ⭐⭐)

### 5단원: 기본 도형 / 작도와 합동
- **주요 유형**
  - 각의 관계 (맞꼭지각 등) (15%, 1분, ⭐)
  - 평행선과 각 (25%, 1-2분, ⭐⭐)
  - 합동 조건 판별 (20%, 1분, ⭐)
  - 작도 과정 (20%, 2분, ⭐⭐)
  - 증명 문제 (20%, 3-4분, ⭐⭐⭐)

### 6단원: 입체도형
- **주요 유형**
  - 겉넓이/부피 기본 (30%, 2분, ⭐)
  - 전개도 (20%, 2분, ⭐⭐)
  - 회전체 (25%, 2-3분, ⭐⭐)
  - 복합 입체 (25%, 3-4분, ⭐⭐⭐)

## 단원 구조 요약

```
중1: 6개 단원 (소인수 → 정수 → 방정식 → 좌표 → 도형 → 입체)
중2: 6개 단원 (순환소수 → 부등식/연립 → 일차함수 → 사각형 → 닮음 → 확률)
중3: 6개 단원 (제곱근 → 인수분해 → 이차방정식 → 이차함수 → 삼각비 → 원)
```
