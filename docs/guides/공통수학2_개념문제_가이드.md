## 공통수학2의 특징

- **위상**: 고등학교 1학년 2학기 과정, 미적분학으로 나아가는 **징검다리**
- **학문적 전환**:
  - 공통수학1의 순수 대수·이산수학 → **기하학적 조건의 대수적 번역(Analytic Geometry)** + **논리적 엄밀성(Logical Rigor)** + **함수적 변환(Functional Transformation)**
  - 좌표 기하 → 집합·명제 → 함수의 3중 나선 구조로 **수학적 성숙도** 완성
  - 시각적 직관(기하) + 논리적 언어(명제) + 변화의 관계(함수)를 통합적으로 다루는 과목
- **핵심 변화**:
  - **선분의 외분 삭제**: 계산 복잡성 완화, 내분점·무게중심에 집중
  - **도형의 방정식**: 공통수학1에서 이동 수용 (기하 영역 분리)
  - **코시-슈바르츠 부등식 비중 감소**: AM-GM 중심으로 절대부등식 재편
  - **행렬 → 공통수학1 이동**: 대수/기하 영역 분리 완료
- **2022 개정 핵심 의도**:
  - 해석기하학적 사고력과 논리적 증명 능력의 동시 함양
  - 함수의 구조적 이해를 통한 미적분 준비 완성
  - 계산 복잡도 완화 → 개념 간 연결과 구조 파악 능력 강화

> 공통수학2는 **"기하학적 조건의 대수적 번역"** 능력과 **"함수적 세계관"** 구축이 핵심

### 1단원: 도형의 방정식 - 기하학적 직관의 대수적 번역

> 좌표평면이라는 무대 위에서 도형의 기하학적 성질을 대수적 식으로 번역하는 **해석기하학(Analytic Geometry)**의 핵심. 점, 직선, 원의 방정식을 통해 기하와 대수의 교차점을 경험.

#### 세부 개념 A: 평면좌표와 직선의 방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **두 점 사이의 거리**: 피타고라스 정리의 좌표평면 일반화 $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$<br>- **선분의 내분점**: $m:n$ 내분 = $\left(\frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n}\right)$<br>- **삼각형의 무게중심**: $\left(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3}\right)$ (세 중선의 교점)<br>- **2022 개정: 선분의 외분 삭제** → 내분점 심화와 무게중심 응용에 집중<br>- **직선의 결정 조건**: 기울기+한 점 $y-y_1=m(x-x_1)$, 일반형 $ax+by+c=0$ (수직선 $x=k$ 포괄)<br>- **두 직선의 위치 관계**: 평행(기울기 같고 절편 다름), 수직($mm'=-1$ 또는 $aa'+bb'=0$)<br>- **점과 직선 사이의 거리**: $\frac{\|ax_1+by_1+c\|}{\sqrt{a^2+b^2}}$ |
| **교육과정 변화** | 외분 삭제로 계산 부담 완화, **거리 공식의 기하학적 해석**과 **내분점 활용** 심화 |
| **주요 오개념** | - **거리 공식에서 뺄셈 순서 착각**: $\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$에서 제곱하므로 순서 무관임을 인식 못함<br>- **수직 조건 단순 암기**: $mm'=-1$만 외우고 유도 과정(피타고라스 정리 기반) 이해 못함<br>- **수직선 $x=k$를 $y=mx+n$에 맞추려는 시도**: 기울기 정의 불가능한 경우의 예외 처리 누락 |
| **출제 포인트** | - $\sqrt{(x-a)^2+(y-b)^2}$를 보고 "점 $(x,y)$와 $(a,b)$ 사이의 거리"로 역추적<br>- 최단 거리 문제: 대칭이동 활용 (기하학적 직관)<br>- 수직 조건을 피타고라스 정리로 유도하는 서술형<br>- **지양**: 복잡한 외분점 계산 문제 |

#### 세부 개념 B: 원의 방정식과 도형의 이동

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **원의 정의**: 한 정점으로부터 일정한 거리에 있는 점들의 **자취**<br>- **표준형**: $(x-a)^2+(y-b)^2=r^2$, **일반형**: $x^2+y^2+Ax+By+C=0$<br>- **원이 되는 조건**: $A^2+B^2-4C>0$ (실원), $=0$ (점원), $<0$ (허원)<br>- **원과 직선의 위치 관계**: 전략 A(대수적: 대입 후 판별식 $D$), 전략 B(기하학적: 중심과 직선 사이 거리 $d$와 반지름 $r$ 비교)<br>- **원의 접선**: (i) 접점 주어진 경우, (ii) 기울기 주어진 경우, (iii) 외부 점에서 그은 경우<br>- **평행이동**: 점 $(x,y) \to (x+a,y+b)$, 도형 $f(x,y)=0 \to f(x-a,y-b)=0$ **(부호 반대!)**<br>- **대칭이동**: $x$축 $(y \to -y)$, $y$축 $(x \to -x)$, 원점 $(x \to -x,y \to -y)$, $y=x$ $(x \leftrightarrow y)$<br>- **$y=x$ 대칭과 역함수**: 3단원 역함수의 기하학적 의미와 직결 |
| **주요 오개념** | - **전략 선택 오류**: 모든 문제에 판별식만 고집 → 계산 과부하 (거리 공식이 효율적인 경우 많음)<br>- **실원/점원/허원 구별 실패**: $A^2+B^2-4C$의 부호 의미 미인식<br>- **접선 유형 혼동**: 세 가지 접선 유형(접점·기울기·외부 점)의 풀이 전략 구분 못함<br>- **점의 이동 vs 도형의 이동 부호 역설(Sign Paradox)**: 점은 $+a$, 도형은 $-a$<br>  → "새 좌표 $x'=x+a$이면 기존 좌표 $x=x'-a$를 대입" 설명이 핵심<br>- **자취의 방정식 난해**: 조건에서 매개변수를 소거하여 $x, y$ 관계식을 도출하는 과정 미숙 |
| **출제 포인트** | - 판별식보다 **거리 공식**이 효율적인 문제 설계<br>- 원의 접선 세 유형 구분<br>- "왜 도형의 이동에서 부호가 반대인가?" 서술형<br>- $y=x$ 대칭과 역함수의 기하학적 의미 연결 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 거리 역추적 | $\sqrt{(x-a)^2+(y-b)^2}$ 식 해석 | 중 |
| 내분점·무게중심 | 좌표 계산 및 도형 성질 활용 | 중 |
| 직선의 수직·평행 | 두 직선의 위치 관계 조건 | 중 |
| 원-직선 위치 관계 | 판별식 vs 거리 공식 전략 선택 | 중상 |
| 실원/점원/허원 | 일반형에서 원 조건 판별 | 중상 |
| 접선 유형 | 접점·기울기·외부 점 구분 | 상 |
| 이동 부호 역설 | 점 vs 도형의 이동 부호 비교 | 상 |
| 대칭이동 | $x$축·$y$축·원점·$y=x$ 대칭 | 중상 |

### 2단원: 집합과 명제 - 논리적 언어의 구축

> 수학적 주장을 엄밀하게 표현하고 검증하는 **논리적 언어 체계**의 형성. 집합을 통해 대상을 분류하고, 명제를 통해 참·거짓을 판별하며, 절대부등식을 통해 항상 참인 관계를 증명.

#### 세부 개념 A: 집합과 명제

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **집합의 정의**: 명확한 기준으로 구별되는 대상들의 모임 → "아름다운 꽃들"은 기준 모호 → 집합 아님<br>- **원소 vs 부분집합**: $\in$ (원소) vs $\subset$ (부분집합) → 문맥에 따라 같은 기호가 다른 의미<br>- **공집합 $\emptyset$**: 모든 집합의 부분집합 (공리적 약속, 대우 증명으로 설명)<br>- **집합 연산**: 교집합 $\cap$, 합집합 $\cup$, 여집합 $A^c$, 차집합 $A-B$<br>- **드 모르간의 법칙**: $(A \cup B)^c = A^c \cap B^c$, $(A \cap B)^c = A^c \cup B^c$<br>- **포함-배제의 원리**: $n(A \cup B) = n(A) + n(B) - n(A \cap B)$<br>- **명제**: 참/거짓이 명확한 문장, **조건과 진리집합**<br>- **$p \to q$의 핵심 원리**: $P \subset Q$이면 참<br>- **역, 이, 대우**: 대우($\lnot q \to \lnot p$)만 원 명제와 진리값 일치<br>- **충분조건과 필요조건**: $P \subset Q$ → $p$는 $q$의 충분조건, $q$는 $p$의 필요조건<br>  → "조건이 까다로울수록(진리집합 작을수록) **충분**조건" |
| **주요 오개념** | - **$\in$과 $\subset$ 혼동**: $A=\{1,2,\{1,2\}\}$일 때 $\{1,2\} \in A$ (원소) vs $\{1,2\} \subset A$ (부분집합) 모두 참이지만 의미 다름<br>- **공집합의 존재론적 지위**: $\emptyset \subset A$가 항상 참인 이유(대우 증명으로 설명) 이해 못함<br>- **진부분집합 개수 오류**: $2^n-1$에서 **자기 자신** 제외 (공집합 제외 아님!)<br>- **충분/필요조건 언어적 함정**: 일상 언어와 수학적 정의의 괴리<br>  → "$x=1$이면 $x^2=1$이다" → $\{1\} \subset \{-1,1\}$ → 충분조건<br>- **역과 대우 혼동**: 역이 참이면 원 명제도 참이라고 착각<br>- **'모든'과 '어떤'의 부정**: $\lnot(\forall x, p(x)) = \exists x, \lnot p(x)$ |
| **출제 포인트** | - $\in$과 $\subset$ 구분 함정 문제<br>- 충분/필요조건 판별 (진리집합 포함 관계)<br>- 대우를 이용한 증명, 귀류법<br>- 드 모르간의 법칙 활용 |

#### 세부 개념 B: 절대부등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **절대부등식**: 변수의 모든 값에 대해 항상 성립하는 부등식<br>- **증명 기본 원리**: $A \geq B \iff A-B \geq 0$, $A^2 \geq 0$ 활용<br>- **산술-기하 평균 부등식(AM-GM)**: $\frac{a+b}{2} \geq \sqrt{ab}$ (단, $a>0, b>0$)<br>  → 등호 성립: $a=b$일 때<br>- **코시-슈바르츠 부등식**: 2022 개정에서 비중 감소, 기본 원리만 이해 |
| **주요 오개념** | - **AM-GM 양수 조건 무시** (가장 치명적): 음수에 적용하면 부등식 방향 역전<br>  → $a, b$가 음수일 때는 $\sqrt{ab}$가 실수이더라도 AM-GM 부등식 부호 방향 역전<br>- **등호 성립 조건 망각**: 최솟값 구할 때 $a=b$가 **실제로 가능한지** 확인 안 함<br>  → 제약 조건 아래서 $a=b$가 불가능하면 그 최솟값은 달성 불가<br>- **함수의 최솟값과 혼동**: AM-GM은 **각 변수의 관계**에서 도출, 함수의 정의역 제한과 별개 |
| **출제 포인트** | - 음수 포함 식의 최대·최소 (AM-GM 사용 불가 상황) 함정 문제<br>- 등호 성립 조건이 존재하는지 확인하는 문제<br>- 도형의 넓이 최적화 (실생활 응용)<br>- $A^2 \geq 0$ 활용 절대부등식 증명 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| $\in$ vs $\subset$ | 원소·부분집합 기호 구분 | 중 |
| 공집합 성질 | 공집합의 부분집합 관계 | 중 |
| 드 모르간 | 여집합 연산 법칙 활용 | 중 |
| 충분/필요조건 | 진리집합 포함 관계로 판별 | 중상 |
| 대우 증명 | 대우를 이용한 간접 증명 | 중상 |
| AM-GM 함정 | 양수 조건 미충족 상황 | 상 |
| 등호 성립 | 등호 조건 가능성 확인 | 상 |
| 절대부등식 증명 | $A-B \geq 0$ 변형 | 중상 |

### 3단원: 함수 - 대응과 변환의 세계

> 함수의 엄밀한 정의에서 출발하여 합성함수, 역함수, 유리·무리함수까지 다루는 **함수적 세계관(Functional Worldview)**의 완성. 미적분학 진입 전 마지막 함수 준비 과정.

#### 세부 개념 A: 함수의 뜻과 합성함수·역함수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **함수의 정의**: 정의역의 모든 원소가 **오직 하나의** 공역 원소에 대응<br>  → 세로선 판별법: 세로선이 그래프와 2점 이상에서 만나면 함수 아님<br>- **일대일 함수**: 서로 다른 정의역 원소 → 서로 다른 함숫값<br>- **일대일 대응**: 일대일 함수 + 치역=공역 → **역함수 존재 조건**<br>- **합성함수**: $(f \circ g)(x)=f(g(x))$, 교환법칙 **불성립**: $f \circ g \neq g \circ f$ (일반적)<br>- **역함수**: 존재 조건=일대일 대응, $f(f^{-1}(x))=f^{-1}(f(x))=x$<br>  → 기하학적 의미: $y=x$에 대해 대칭 (1단원 $y=x$ 대칭이동과 직결)<br>- **합성함수의 분해**: 주어진 합성함수에서 원래 함수 역추적 |
| **주요 오개념** | - **일대일 함수 ↔ 일대일 대응 혼동**: 치역 $\neq$ 공역이면 일대일 대응 아님<br>- **합성함수 교환법칙 착각**: $f \circ g = g \circ f$라고 생각<br>- **합성 순서 혼동**: $(f \circ g)(x)$에서 $f$를 먼저 적용하는 오류 → $g$ 먼저!<br>- **역함수 교점 패러독스(Intersection Paradox)**:<br>  "대부분의 학생 오답: $f$와 $f^{-1}$의 교점은 **반드시** $y=x$ 위에만 있다"<br>  → **거짓!** 증가함수에서는 $y=x$ 위에만 존재하지만,<br>  **감소함수**(예: $y=-x^3$, $y=1/x$)에서는 $y=x$ **밖에서도** 교점 $(a,b)$, $(b,a)$ 쌍 존재 가능<br>  → 이 패러독스는 고난도 문항의 단골 소재 |
| **출제 포인트** | - 세로선 판별법으로 함수 여부 판단<br>- $f \circ g \neq g \circ f$ 반례 찾기<br>- 역함수 존재 조건 판별<br>- $f$와 $f^{-1}$의 교점이 $y=x$ 밖에 있는 경우 탐구 (고난도) |

#### 세부 개념 B: 유리함수와 무리함수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **유리함수** $y = \frac{k}{x-p} + q$: 점근선 $x=p$ (수직), $y=q$ (수평), 점근선 교점 $(p,q)$에 대해 **점대칭**<br>- **유리함수 그래프**: $k>0$이면 1·3사분면형, $k<0$이면 2·4사분면형 (점근선 교점 기준)<br>- **무리함수** $y = \sqrt{ax+b} + c$: 정의역 $ax+b \geq 0$, 시작점에서 한 방향으로 뻗어나감<br>- **무리함수의 역함수 관계**: $y=\sqrt{x}$ $\leftrightarrow$ $y=x^2$ ($x \geq 0$)는 $y=x$ 대칭 |
| **주요 오개념** | - **무리함수와 직선의 교점에서 무연근(Extraneous Root) 발생**:<br>  양변 제곱 시 부호 정보 소실 → 존재하지 않는 근 등장<br>  → 반드시 **원래 방정식에 대입 검증** 또는 **그래프로 시각적 확인** 필요<br>- **유리함수 점근선 착각**: 그래프가 점근선에 **닿을 수 있다**고 생각 → 점근선은 한없이 가까워질 뿐 도달 불가<br>- **유리함수 사분면 배치 오류**: $k$의 부호와 점근선 교점 위치에 따른 그래프 배치 혼동<br>- **정의역 조건 누락**: $\sqrt{ax+b}$에서 $ax+b \geq 0$ 조건 빠뜨림 |
| **출제 포인트** | - 무리함수와 직선의 위치 관계: 대수적 계산 + **그래프 확인** 병행<br>- 유리함수 대칭성 활용 (최단 거리, 넓이)<br>- 점근선 구하기와 그래프 개형<br>- 정의역·치역 구하기 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 함수 판별 | 세로선 판별법 적용 | 하 |
| 일대일 대응 | 역함수 존재 조건 판별 | 중 |
| 합성함수 순서 | $f \circ g$에서 $g$ 먼저 적용 | 중 |
| 합성 분해 | 합성함수에서 원래 함수 역추적 | 중상 |
| 역함수 교점 | $y=x$ 밖의 교점 존재 여부 | 상 |
| 유리함수 그래프 | 점근선, 대칭성, 사분면 배치 | 중상 |
| 무리함수 무연근 | 양변 제곱 후 검증 필수 | 상 |

## 문제 개발 전략

### 1. 도형의 이동: 부호 역설(Sign Paradox) 완전 정복

| 이동 유형 | 점의 이동 | 도형의 이동 |
|-----------|-----------|-------------|
| 평행이동 $(a, b)$ | $(x, y) \to (x+a, y+b)$ | $f(x, y)=0 \to f(x-a, y-b)=0$ |
| 부호 | 직관적 덧셈 | **반대** |

> **핵심 설명**: 새 좌표 $x'=x+a$이면 기존 좌표 $x=x'-a$를 대입해야 함
> → "도형의 모든 점이 이동한 결과, 새 좌표계에서 기존 관계식을 만족시키려면 역변환 필요"

### 2. 원과 직선의 위치 관계: 전략 선택 가이드

| 상황 | 권장 전략 | 이유 |
|------|-----------|------|
| 미지수 $k$가 복잡하게 포함 | **거리 공식** | 계산량 대폭 감소 |
| 교점의 좌표가 필요 | 판별식 + 근과 계수의 관계 | 좌표값 직접 산출 |
| 접선의 존재성만 판단 | **거리 공식** = $r$ | 가장 간결 |

> 판별식만 고집하면 4차식 등 계산 과부하 발생 → **전략 선택 능력**이 핵심

### 3. AM-GM 적용 완전 체크리스트

| 확인 사항 | 체크 |
|-----------|------|
| 1. $a > 0$, $b > 0$ (양수 조건) | - |
| 2. 등호 성립 조건 $a=b$가 가능한가? | - |
| 3. 그때의 최솟값/최댓값 계산 | - |
| 4. 제약 조건 아래서 등호 달성 가능한가? | - |

> **함정 문제 설계**: 양수 조건 불만족 또는 등호 조건 달성 불가능 상황 제시

### 4. 역함수 교점 패러독스: 완전 정리

| 함수 유형 | $f$와 $f^{-1}$ 교점 위치 | 수학적 근거 |
|-----------|---------------------------|-------------|
| **증가함수** | $y=x$ 위에만 존재 | $f(a)=b, f(b)=a$이면 $f$ 증가 → $a=b$ |
| **감소함수** | $y=x$ **밖에도** 존재 가능 | $f(a)=b, f(b)=a$ ($a \neq b$) 가능 |

> 반례: $f(x)=-x^3$, $f(x)=\frac{1}{x}$ ($x<0$)
> → 이 패러독스를 **정확히 서술하는 고난도 문항** 설계 가능

### 5. 무리함수 무연근 방지 체크리스트

| 단계 | 내용 |
|------|------|
| 1. 양변 제곱하여 $x$ 구하기 | 기계적 대수 조작 |
| 2. 원래 방정식에 대입 검증 | **부호 정보 복원** |
| 3. 그래프로 교점 위치 확인 | **시각적 검증** (최종 확인) |

> "제곱하면 무연근이 생길 수 있다" → **대수적 계산 + 그래프 확인** 병행 필수

### 6. 역량 기반 문항 설계
