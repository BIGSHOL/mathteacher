## 📘 단원별 정리

### 1단원: 도형의 방정식

#### 1-1. 평면좌표

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **두 점 사이의 거리**: 피타고라스 정리의 좌표평면 일반화<br>  - $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$<br>• **선분의 내분**: $m:n$으로 나누는 점<br>  - 내분점: $\left(\frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n}\right)$<br>• **삼각형의 무게중심**: 세 중선의 교점<br>  - $\left(\frac{x_1+x_2+x_3}{3}, \frac{y_1+y_2+y_3}{3}\right)$ |
| **교육과정 변화** | ⚠️ **선분의 외분 삭제**<br>  - 삭제 배경: 계산 복잡성이 수학적 흥미 저해<br>  - 대신 **내분점의 심화**와 **무게중심 응용**에 집중 |
| **출제 포인트** | ✓ $\sqrt{(x-a)^2 + (y-b)^2}$를 보고 "점 $(x,y)$와 $(a,b)$ 사이의 거리"로 역추적<br>✓ 최단 거리 문제: 대칭이동 활용 (기하학적 직관)<br>⚠️ **지양**: 복잡한 외분점 계산 문제 |

---

#### 1-2. 직선의 방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **직선의 결정 조건**:<br>  - 기울기 + 한 점: $y - y_1 = m(x - x_1)$<br>  - 일반형: $ax + by + c = 0$ (수직선 $x=k$ 포괄)<br>• **두 직선의 위치 관계**:<br>  - 평행: 기울기 같고 $y$절편 다름<br>  - 수직: $mm' = -1$ 또는 $aa' + bb' = 0$<br>• **점과 직선 사이의 거리**: $\frac{|ax_1 + by_1 + c|}{\sqrt{a^2+b^2}}$ |
| **주요 오개념** | • **수직 조건 단순 암기**: 공식만 외우고 피타고라스 정리 기반 유도 이해 못함 |
| **출제 포인트** | ✓ 수직 조건을 피타고라스 정리로 유도하는 과정 서술<br>✓ 점과 직선 사이의 거리 공식 활용 (원의 접선 등) |

#### 1-3. 원의 방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **원의 정의**: 한 정점으로부터 일정한 거리에 있는 점들의 자취<br>• **표준형**: $(x-a)^2 + (y-b)^2 = r^2$ (중심 $(a,b)$, 반지름 $r$)<br>• **일반형**: $x^2 + y^2 + Ax + By + C = 0$<br>  - 원이 되는 조건: $A^2 + B^2 - 4C > 0$<br>• **원과 직선의 위치 관계**:<br>  - 전략 A (대수적): 대입 후 판별식 $D$<br>  - 전략 B (기하학적): 중심과 직선 사이 거리 $d$와 반지름 $r$ 비교 |
| **주요 오개념** | • **전략 선택 오류**: 모든 문제에 판별식 사용 → 계산 과부하<br>  → 거리 공식이 효율적인 경우 많음 |
| **출제 포인트** | ✓ 판별식보다 **거리 공식**이 효율적인 문제 설계<br>✓ 원의 접선 (접점 주어진 경우, 기울기 주어진 경우, 외부 점에서 그은 경우 구분) |

#### 1-4. 도형의 이동

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **평행이동**:<br>  - 점의 이동: $(x, y) \to (x+a, y+b)$<br>  - 도형의 이동: $f(x, y) = 0 \to f(x-a, y-b) = 0$ **(부호 반대!)**<br>• **대칭이동**:<br>  - $x$축 대칭: $(x, y) \to (x, -y)$<br>  - $y$축 대칭: $(x, y) \to (-x, y)$<br>  - 원점 대칭: $(x, y) \to (-x, -y)$<br>  - $y=x$ 대칭: $(x, y) \to (y, x)$ → **역함수와 연결** |
| **주요 오개념** | • **⭐⭐⭐ 점의 이동 vs 도형의 이동 부호 혼동**:<br>  점: $(x, y) \to (x+a, y+b)$ (직관적 덧셈)<br>  도형: $f(x-a, y-b) = 0$ (부호 반대)<br>  → 새 좌표 $(x', y')$와 기존 좌표 $(x, y)$ 관계식으로 설명 필요 |
| **출제 포인트** | ✓ "왜 도형의 이동에서 부호가 반대인가?" 서술형<br>✓ $y=x$ 대칭과 역함수의 기하학적 의미 연결 |

### 2단원: 집합과 명제

#### 2-1. 집합

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **집합의 정의**: 명확한 기준으로 구별되는 대상들의 모임<br>  → "아름다운 꽃들의 모임"은 집합이 아님 (기준 모호)<br>• **원소 vs 부분집합 기호**: $\in$ vs $\subset$<br>• **공집합**: 모든 집합의 부분집합 (공리적 약속)<br>• **집합의 연산**: 교집합, 합집합, 여집합, 차집합<br>• **드 모르간의 법칙**: $(A \cup B)^c = A^c \cap B^c$ 등<br>• **포함-배제의 원리**: $n(A \cup B) = n(A) + n(B) - n(A \cap B)$ |
| **주요 오개념** | • **⭐⭐ $\in$과 $\subset$ 혼동**:<br>  $A = \{1, 2, \{1, 2\}\}$일 때<br>  - $\{1, 2\} \in A$ ✓ (원소로서 포함)<br>  - $\{1, 2\} \subset A$ ✓ (부분집합으로서 포함)<br>  - $\{1\} \in A$ ✗ / $\{1\} \subset A$ ✓<br>• **진부분집합 개수**: $2^n - 1$에서 **자기 자신**을 제외 (공집합 제외 아님!) |
| **출제 포인트** | ✓ $\in$과 $\subset$ 구분 함정 문제<br>✓ 드 모르간의 법칙 증명 (벤 다이어그램 또는 논리적) |

---

#### 2-2. 명제

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **명제**: 참/거짓이 명확한 문장<br>• **조건과 진리집합**: 조건 $p$를 참이 되게 하는 모든 원소의 집합 $P$<br>• **명제 $p \to q$의 핵심 원리**: $P \subset Q$이면 참<br>• **역과 대우**:<br>  - 대우($\neg q \to \neg p$)는 원 명제와 참/거짓 일치<br>  - 역($q \to p$)은 원 명제와 논리적 관계 없음<br>• **충분조건과 필요조건**:<br>  - $P \subset Q$일 때: $p$는 $q$의 **충분조건**, $q$는 $p$의 **필요조건** |
| **주요 오개념** | • **⭐⭐ 충분/필요조건 혼동**: 조건이 까다로울수록(진리집합 작을수록) **충분**조건<br>• **역과 대우 혼동**: 역이 참이면 원 명제도 참이라고 착각 |
| **출제 포인트** | ✓ $x=1$은 $x^2=1$이기 위한 무슨 조건? → 진리집합 $\{1\} \subset \{-1, 1\}$ → 충분조건<br>✓ 대우를 이용한 증명, 귀류법 ($\sqrt{2}$가 무리수임 증명) |

---

#### 2-3. 절대부등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **절대부등식**: 항상 성립하는 부등식<br>• **증명 기본 원리**: $A \geq B \iff A - B \geq 0$, $A^2 \geq 0$<br>• **산술평균-기하평균 관계 (AM-GM)**:<br>  $\frac{a+b}{2} \geq \sqrt{ab}$ (단, $a > 0$, $b > 0$)<br>  - 등호 성립: $a = b$일 때 |
| **주요 오개념** | • **⭐⭐⭐ 양수 조건 무시**: AM-GM은 **양수**에서만 성립<br>  → 음수에 적용하는 치명적 오류<br>• **등호 성립 조건 미확인**: 최솟값 구할 때 $a=b$가 실제로 가능한지 확인 안 함 |
| **출제 포인트** | ✓ 음수 포함된 식의 최대·최소 (AM-GM 사용 불가 상황) 함정 문제<br>✓ 등호 성립 조건이 존재하는지 확인하는 문제<br>✓ 도형의 넓이 최적화 문제 (실생활 응용) |

### 3단원: 함수

#### 3-1. 함수의 뜻과 그래프

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **함수의 정의**: 모든 정의역 원소가 **오직 하나의** 공역 원소에 대응<br>  → 세로선 판별법: 세로선이 그래프와 2점 이상에서 만나면 함수 아님<br>• **일대일 함수**: 서로 다른 정의역 원소 → 서로 다른 함숫값<br>• **일대일 대응**: 일대일 함수 + 치역 = 공역 → **역함수 존재 조건** |
| **주요 오개념** | • **일대일 함수 ↔ 일대일 대응 혼동**: 치역 ≠ 공역이면 일대일 대응 아님 |
| **출제 포인트** | ✓ 세로선 판별법으로 함수 여부 판단<br>✓ 일대일 함수이지만 일대일 대응은 아닌 사례 |

---

#### 3-2. 합성함수와 역함수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **합성함수**: $(f \circ g)(x) = f(g(x))$<br>  - 교환법칙 **불성립**: $f \circ g \neq g \circ f$ (일반적으로)<br>• **역함수**:<br>  - 존재 조건: **일대일 대응**<br>  - 기하학적 의미: $y = x$에 대해 대칭<br>  - $f(f^{-1}(x)) = f^{-1}(f(x)) = x$ |
| **주요 오개념** | • **⭐ 합성함수 교환법칙 착각**: $f \circ g = g \circ f$라고 생각<br>• **⭐⭐⭐ 역함수 교점 위치 착각**:<br>  "$f$와 $f^{-1}$의 교점은 **반드시** $y=x$ 위에만 있다"<br>  → **거짓!** 감소함수($y=-x^3$, $y=1/x$ 등)의 경우<br>  $y=x$ 밖에서도 교점 $(a, b)$, $(b, a)$ 쌍 존재 가능 |
| **출제 포인트** | ✓ $f \circ g \neq g \circ f$ 반례 찾기<br>✓ $f$와 $f^{-1}$의 교점이 $y=x$ 밖에 있는 경우 탐구 (고난도) |

---

#### 3-3. 유리함수와 무리함수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **유리함수** $y = \frac{k}{x-p} + q$:<br>  - 점근선: $x = p$ (수직), $y = q$ (수평)<br>  - 대칭: 점근선 교점 $(p, q)$에 대해 **점대칭**<br>• **무리함수** $y = \sqrt{ax+b}$:<br>  - 정의역: $ax + b \geq 0$<br>  - $y = \sqrt{x}$는 $y = x^2$ ($x \geq 0$)의 역함수 |
| **주요 오개념** | • **⭐⭐⭐ 무리함수와 직선의 교점에서 무연근 발생**:<br>  양변 제곱하면 부호 정보 소실 → 존재하지 않는 근 등장<br>  → 반드시 **그래프로 시각적 확인** 필요<br>• **점근선 착각**: 그래프가 점근선에 닿을 수 있다고 생각 |
| **출제 포인트** | ✓ 무리함수와 직선의 위치 관계: 대수적 계산 + **그래프 확인** 병행<br>✓ 유리함수 대칭성 활용 문제 (최단 거리, 넓이 등) |

## 💡 문제 개발 전략

### 1. 도형의 이동: 부호 혼동 방지

| 이동 유형 | 점의 이동 | 도형의 이동 |
|-----------|-----------|-------------|
| 평행이동 $(a, b)$ | $(x, y) \to (x+a, y+b)$ | $f(x, y)=0 \to f(x-a, y-b)=0$ |
| 부호 | 직관적 덧셈 | **반대** |

> **핵심 설명**: 새 좌표 $x' = x + a$이면 기존 좌표 $x = x' - a$를 대입해야 함

---

### 2. 원과 직선의 위치 관계: 전략 선택

| 상황 | 권장 전략 |
|------|-----------|
| 미지수 $k$가 복잡하게 포함 | **거리 공식** (계산량 감소) |
| 교점의 좌표가 필요 | 판별식 + 근과 계수의 관계 |

> 판별식만 고집하면 4차식 등 계산 과부하 발생

---

### 3. AM-GM 적용 체크리스트

| 확인 사항 | 체크 |
|-----------|------|
| ① $a > 0$, $b > 0$ (양수 조건) | ☐ |
| ② 등호 성립 조건 $a = b$가 가능한가? | ☐ |
| ③ 그때의 최솟값/최댓값 계산 | ☐ |

> **함정 문제 설계**: 양수 조건 불만족 상황 제시

---

### 4. 역함수 교점: 반례 정리

| 함수 유형 | $f$와 $f^{-1}$ 교점 위치 |
|-----------|-------------------------|
| 증가함수 | $y = x$ 위에만 존재 |
| **감소함수** | $y = x$ **밖**에도 존재 가능 |

> 반례: $f(x) = -x^3$, $f(x) = \frac{1}{x}$ ($x < 0$)

### 5. 개념 문항 설계 3단계

| 단계 | 목표 | 예시 |
