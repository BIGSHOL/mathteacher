## 중1 수학의 특징

- **위상**: 초등 수학(직관적 단계) -> 중등 수학(추상적/논리적 단계)의 결정적 전환기
- **인지적 도약**: 산술적 사고(Arithmetic) -> 대수적 사고(Algebraic)
- **수 체계 확장**: 자연수 -> 정수 -> 유리수
- **2022 개정 핵심 변화**:
  - 최대공약수/최소공배수 복잡한 활용 문제 **삭제**
  - 대푯값, 산점도 **중3 -> 중1로 이동** (데이터 리터러시 강화)
  - '문자와 식' + '함수' -> **'변화와 관계'** 영역으로 통합
  - **디지털 소양** 강화: 공학 도구 활용 권장

> 개념 문제는 단순 연산 숙달이 아닌 **'개념적 이해(Conceptual Understanding)'** 평가로 전환

### 1단원: 소인수분해

#### 세부 개념 A: 소수와 합성수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **소수(Prime)**: 1보다 큰 자연수 중 1과 자기 자신만을 약수로 가지는 수 (2, 3, 5, 7, 11...)<br>- **합성수(Composite)**: 1보다 큰 자연수 중 소수가 아닌 수<br>- **단위수 1**: 소수도 합성수도 아님 (산술의 기본 정리 - 유일성 조건)<br>- **서로소(Relatively Prime)**: 최대공약수가 1인 두 자연수의 **관계** |
| **주요 오개념** | - **1을 소수로 분류**: "약수가 하나뿐이므로 가장 단순한 소수"<br>- **서로소 의미 혼동**: "서로소인 두 수는 둘 다 소수여야 한다" (반례: 4와 9는 서로소이지만 둘 다 합성수)<br>- **2가 소수가 아니라고 착각**: "짝수니까 합성수" (2는 유일한 짝수 소수) |
| **출제 포인트** | - "왜 1은 소수에서 제외되는가?" 심층 이해<br>- 서로소는 '두 수 사이의 관계'임을 강조<br>- 소수 판별법: 주어진 수의 제곱근 이하의 소수로 나누어 보기 |

#### 세부 개념 B: 거듭제곱과 소인수분해

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **거듭제곱**: $a^n$에서 $a$는 밑(base), $n$은 지수(exponent)<br>- **소인수분해 방법**: 가지치기(Factor Tree) / 나눗셈(Repeated Division)<br>- **소인수분해의 유일성**: 곱하는 순서와 관계없이 소인수들의 곱은 유일하게 결정됨<br>- **약수의 개수**: $N=a^m \times b^n$일 때, 약수의 개수 = $(m+1)(n+1)$ |
| **주요 오개념** | - **지수와 밑 혼동**: $2^3 = 2 \times 3 = 6$으로 계산 (정답: $2^3 = 8$)<br>- **약수의 개수 공식 적용 실수**: 지수에 1을 더하지 않고 그대로 곱함 |

| 문항 유형 | 설명 | 핵심 전략 | 난이도 |
|-----------|------|-----------|--------|
| 기본 연산 | 주어진 자연수를 소인수분해하여 $a^m \times b^n$ 꼴로 표현 | 작은 소수(2, 3, 5...)부터 순차적으로 나누기 | 하 |
| 약수의 개수 | 소인수분해를 이용하여 약수의 개수 구하기 | $(m+1)(n+1)$ 공식 적용 | 중 |
| 제곱수 만들기 | 자연수 $N$에 가장 작은 수를 곱하여 제곱수로 만들기 | 모든 소인수의 지수가 짝수가 되도록 부족한 소인수를 곱함 | 중상 |
| 약수 역추적 | 약수의 개수가 주어졌을 때 원래 수 $N$ 또는 미지수 찾기 | 약수의 개수를 분해하여 가능한 지수 조합 추론 | 상 |
| 응용 심화 | 소인수분해를 이용한 자연수의 나누어 떨어짐 판정 | 특정 수로 나누어 떨어지려면 그 수의 소인수를 모두 포함해야 함 | 상 |

#### 세부 개념 C: 최대공약수와 최소공배수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **최대공약수(GCD)**: 공약수 중 가장 큰 수. 두 수의 공약수는 GCD의 약수와 같음<br>- **최소공배수(LCM)**: 공배수 중 가장 작은 수. 두 수의 공배수는 LCM의 배수와 같음<br>- **핵심 관계식**: $A \times B = GCD(A,B) \times LCM(A,B)$<br>- 소인수분해 이용: GCD는 공통 소인수의 **최소 지수**, LCM은 모든 소인수의 **최대 지수** |
| **주요 오개념** | - **최대/최소 직관 오류**: "최대공약수니까 큰 수", "최소공배수니까 작은 수"<br>- **$A \times B = GCD \times LCM$ 관계 미인지** |
| **출제 포인트** | - "두 수의 공배수는 최소공배수의 배수이다" 같은 **수의 성질** 문항<br>- $A \times B = GCD \times LCM$ 활용 문항 |

**GCD/LCM 활용 문항 유형**

| 구분 | 하위 유형 | 상황 | 모델링 전략 |
|------|-----------|------|------------|
| GCD 활용 | 직사각형 채우기 | 가로, 세로가 다른 직사각형을 가장 큰 정사각형 타일로 빈틈없이 채우기 | 가로와 세로의 GCD 구하기 |
| GCD 활용 | 똑같이 나누어 주기 | 사탕과 초콜릿을 최대한 많은 학생에게 남김없이 나누어 주기 | 물건 개수의 GCD 구하기 |
| GCD 활용 | 나무 심기/가로등 | 일정한 간격으로 나무를 심을 때 간격 최대화 | 거리의 GCD. 주의: 나무 수 = (거리/간격) + 1 |
| LCM 활용 | 동시 출발 | 배차 간격이 다른 버스가 동시 출발 후 다시 만나는 시각 | 배차 간격의 LCM 구하기 |
| LCM 활용 | 톱니바퀴 | 톱니 수가 다른 두 톱니바퀴가 처음 맞물린 톱니가 다시 만나는 회전수 | 톱니 수의 LCM 구하기 |
| LCM 활용 | 정육면체 만들기 | 직육면체 벽돌을 쌓아 가장 작은 정육면체 만들기 | 가로, 세로, 높이의 LCM 구하기 |
| 나머지 문제 | 나머지 역추적 | $N$으로 나누면 $a$가 남고, $M$으로 나누면 $b$가 남는 수 구하기 | (어떤 수 - 나머지)는 나누는 수의 공배수임을 이용 |

### 2단원: 정수와 유리수

#### 세부 개념 A: 정수의 분류와 수직선

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **정수**: 양의 정수(자연수), 0, 음의 정수<br>- **음수의 도입**: 서로 반대되는 성질 표현 (이익/손해, 영상/영하)<br>- **수직선 모델**: 수의 크기와 순서 시각화, 오른쪽이 항상 큼<br>- **대소 관계**: 음수는 절대값이 클수록 작음. 음수 < 0 < 양수 |
| **주요 오개념** | - **0을 양의 정수에 포함**: 0은 양도 음도 아님<br>- **음수끼리 대소 비교 실수**: "절대값이 큰 수가 항상 더 크다"고 착각 |

#### 세부 개념 B: 절대값과 유리수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **절대값**: 원점으로부터의 거리 (기하학적 정의). 항상 0 이상<br>- **유리수**: 분수 $\frac{a}{b}$ ($a, b$는 정수, $b \neq 0$) 꼴로 나타낼 수 있는 수<br>- **수 체계 포함 관계**: 자연수 $\subset$ 정수 $\subset$ 유리수<br>- **유리수의 조밀성**: 두 유리수 사이에 무수히 많은 유리수 존재 |
| **주요 오개념** | - **"$-a$는 반드시 음수이다"** (가장 치명적): $a = -3$이면 $-a = +3$. 마이너스 기호는 '부호를 반대로 바꿈'의 연산자<br>- **분수 형태 = 정수 아님**: "$\frac{6}{2}$는 분수이므로 정수 아니다" (약분하면 정수 3)<br>- **절대값이 같고 부호가 반대인 두 수의 합**: 항상 0임을 놓침 |
| **출제 포인트** | - "절대값이 $a$인 수는 $\pm a$ 두 개" 활용 문제<br>- "1과 2 사이에 존재하는 유리수는 몇 개인가?" (무수히 많다)<br>- 수 체계 분류 벤 다이어그램 활동 |

#### 세부 개념 C: 유리수의 사칙연산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **덧셈/뺄셈**: 부호가 같으면 절대값의 합, 다르면 절대값의 차. 뺄셈은 덧셈으로 변환<br>- **곱셈/나눗셈**: 마이너스 부호 개수가 홀수면 $(-)$, 짝수면 $(+)$<br>- **거듭제곱 부호 처리**: $(-2)^2 = 4$ vs $-2^2 = -4$ (매우 중요한 구분)<br>- **분배법칙**: $a \times (b+c) = ab + ac$. 효율적 계산의 필수 도구<br>- **연산 법칙**: 교환법칙, 결합법칙(덧셈/곱셈), 분배법칙 |
| **주요 오개념** | - **$-3^2$을 $9$로 계산**: 정답은 $-9$ ($-(3^2)$이므로)<br>- **뺄셈에서 교환법칙 적용**: $3 - 5 \neq 5 - 3$<br>- **나눗셈 역수 처리 오류**: $\frac{2}{3} \div \frac{4}{5}$에서 역수를 잘못 취함 |
| **출제 포인트** | - 계산 법칙 성립 여부: "정수의 뺄셈에서 교환법칙이 성립하지 않음을 보이는 반례"<br>- 괄호(소, 중, 대)와 거듭제곱이 혼합된 복잡한 혼합 계산 (PEMDAS 적용)<br>- 분배법칙을 이용한 효율적 계산: $99 \times 37 = (100-1) \times 37$ |

### 3단원: 문자의 사용과 식의 계산

#### 세부 개념 A: 문자의 사용과 대수적 관습

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **대수적 언어 습득**: 수량 관계를 문자로 표현 (일반화, 간결성)<br>- **대수적 관습**: 곱셈 기호 생략, 숫자를 문자 앞에, 알파벳 순서, 1의 생략<br>- **나눗셈의 분수 표현**: $a \div b = \frac{a}{b}$<br>- **항, 계수, 차수**: 식의 구성 요소 용어 정리 |
| **주요 오개념** | - **곱셈 기호 생략 규칙 혼동**: $x \times 1 = x$ (1 생략), $0.1 \times x = 0.1x$ (소수는 그대로)<br>- **음수 대입 시 괄호 미사용**: $2x$에 $x = -3$ 대입 시 $2 \times -3$으로 표기 (정답: $2 \times (-3)$) |

#### 세부 개념 B: 동류항과 식의 계산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **동류항(Like Terms)**: 문자와 차수가 같은 항. 동류항끼리만 덧셈/뺄셈 가능<br>- **분배법칙**: 괄호를 풀어 식을 간단히 함<br>- **일차식의 정리**: 동류항 정리 -> 내림차순 배열 |
| **주요 오개념** | - **분배법칙 부호 오류**: $-(2x - 3) = -2x - 3$ (정답: $-2x + 3$). 괄호 안 **모든 항**에 $-1$ 곱해야 함<br>- **동류항 판별 오류**: $2x^2$와 $2x$를 동류항으로 착각 (차수가 다름)<br>- **상수항 무시**: $3x + 5 - 2x - 3$에서 상수항 $5 - 3 = 2$를 빠뜨림 |

#### 세부 개념 C: 대입과 식의 값

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **대입**: 문자에 구체적인 수를 넣어 식의 값을 구하는 과정<br>- **변수(Variable)** 개념 정착: 문자는 '미지수'가 아니라 '변하는 수'<br>- **식(Expression) vs 방정식(Equation)**: 등호가 없으면 식, 있으면 등식 |
| **주요 오개념** | - **식과 방정식 혼동**: $2x + 4$를 보고 무의식적으로 $= 0$ 붙여 $x = -2$로 품<br>- **역수 취하기 오류**: $\frac{2}{3}x$의 역수를 $\frac{3}{2}x$로 착각 ($\frac{2}{3}x = \frac{2x}{3}$) |
| **출제 포인트** | - "식을 간단히 하시오" vs "방정식을 푸시오" 구분 문항<br>- 두 식의 합/차를 구한 후 대입하여 값을 구하는 복합 문제 |

### 4단원: 일차방정식

#### 세부 개념 A: 방정식과 등식의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **방정식의 정의**: 미지수를 포함하는 등식<br>- **등식의 성질 4가지**: 양변에 같은 수를 더하기/빼기/곱하기/나누기(0 제외)해도 등식 성립<br>- **방정식의 해**: 방정식을 참이 되게 하는 미지수의 값<br>- **해의 검증**: 구한 해를 원래 방정식에 대입하여 참이 되는지 확인 |
| **주요 오개념** | - **계수 처리 오류**: $3x = 6$에서 $x = 6 - 3$으로 계산 (나눗셈이 아닌 뺄셈 적용) |

#### 세부 개념 B: 일차방정식의 풀이

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **이항**: 등식의 성질에서 파생된 결과. 항을 반대편으로 옮기면 부호가 바뀜<br>- **소수 계수 처리**: 양변에 10, 100 등을 곱하여 정수로 변환<br>- **분수 계수 처리**: 양변에 분모의 최소공배수를 곱하여 정수로 변환<br>- **비례식 형태**: $A:B = C:D \rightarrow AD = BC$ (내항의 곱 = 외항의 곱) |
| **주요 오개념** | - **이항 시 부호 유지**: 항을 옮길 때 부호를 안 바꿈<br>- **소수/분수 계수 처리**: $0.1x = 2$에서 $x = 0.2$로 착각 (양변에 10을 곱해야) |

| 풀이 유형 | 설명 | 핵심 전략 | 난이도 |
|-----------|------|-----------|--------|
| 기본 풀이 | $ax + b = c$ 형태 | 이항 -> 정리 -> 계수로 나누기 | 하 |
| 소수 계수 | $0.3x - 0.2 = 0.1x + 0.6$ | 양변에 10 곱하여 정수화 | 중 |
| 분수 계수 | $\frac{x}{2} - \frac{x-1}{3} = 1$ | 분모의 LCM을 양변에 곱하기 | 중 |
| 비례식 | $3:x = 5:10$ | 내항의 곱 = 외항의 곱 | 중 |
| 해 대입 | $x = -2$가 해일 때 $ax + 3 = 7$에서 $a$ 구하기 | 해를 대입하여 미정계수 결정 | 중상 |
| 해의 특수성 | $ax = b$에서 해가 없거나 무수히 많은 경우 | $a=0, b=0$: 항등식(무수히 많음), $a=0, b \neq 0$: 불능(해 없음) | 상 |

#### 세부 개념 C: 항등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **항등식**: 어떤 값을 대입해도 항상 참인 등식<br>- **항등식 조건**: 양변의 동류항의 계수가 각각 같아야 함<br>- **미정계수법**: "$3(x-1) = ax + b$가 항등식이 되도록 하는 $a, b$는?" |

#### [심화] 일차방정식의 활용 문항 유형

> 일차방정식의 활용은 중1 수학의 가장 큰 난관(Killer Contents).
> 2022 개정에서는 문해력을 요하는 긴 지문 형태의 문제가 증가하는 추세.

**유형 1: 농도 문제**

핵심 원리: **소금의 양 보존** ($\frac{\%}{100} \times \text{소금물 양} = \text{소금 양}$)

| 하위 유형 | 상황 | 모델링 핵심 |
|-----------|------|------------|
| 물 추가 (희석) | 소금물에 물만 더 넣음 | 처음 소금 = 나중 소금. 소금물 양 증가 |
| 증발 (농축) | 물을 증발시킴 | 처음 소금 = 나중 소금. 소금물 양 감소 |
| 소금 추가 | 소금을 더 넣음 | 처음 소금 + 추가 소금 = 나중 소금. 소금물 양도 증가 |
| 두 소금물 혼합 | 농도가 다른 A, B 혼합 | $\text{소금}_A + \text{소금}_B = \text{소금}_\text{혼합}$ |

교수 전략: "비커 그림(Beaker Diagram)" - 변화 전후의 소금물 양과 농도를 시각화

**유형 2: 거리/속력/시간 (거속시)**

핵심 공식: $\text{거리} = \text{속력} \times \text{시간}$

| 하위 유형 | 상황 | 핵심 전략 |
|-----------|------|-----------|
| 왕복 이동 | 갈 때/올 때 속력이 다름 | 거리=$x$로 놓고, 갈 때 시간 + 올 때 시간 = 총 시간 |
| 등산 문제 | 올라갈 때/내려올 때 코스가 다름 | 올라간 거리 $x$, 내려온 거리는 총 거리로 표현. 시간 합 식 세우기 |
| 터널/기차 통과 | 기차가 터널을 완전히 통과 | 이동 거리 = 터널 길이 + 기차 길이 |
| 추격 문제 | 먼저 출발한 A를 B가 따라잡음 | A의 이동 거리 = B의 이동 거리 |
| 호수/트랙 돌기 | 반대 방향(만남) / 같은 방향(추격) | 반대: 거리 합 = 둘레 / 같은: 거리 차 = 둘레 |

**유형 3: 일(Work) 문제**

핵심 원리: 전체 일의 양 = **1**. A가 혼자 $a$일 -> 하루 작업량 = $\frac{1}{a}$

**유형 4: 원가/정가/할인**

핵심 관계: 이익 = 판매가 - 원가. 정가 = $\text{원가} \times (1 + \frac{a}{100})$

**유형 5: 시계 문제 (심화)**

핵심 원리: 시침은 1분에 $0.5°$, 분침은 1분에 $6°$ 이동

모델링: $x$분 후 두 바늘의 각도 차이 = $|6x - (30 \times \text{시} + 0.5x)|$

### 1단원: 기본 도형과 작도

| **주요 오개념** | - **꼬인 위치 찾기 실패**: 직육면체에서 평행한 모서리를 꼬인 위치로 착각<br>- **반직선의 방향성 무시**: $\overrightarrow{AB} = \overrightarrow{BA}$로 착각 |

#### 세부 개념 B: 평행선의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **동위각**: 같은 위치에 있는 각. 평행할 때 크기가 같음<br>- **엇각**: 엇갈린 위치에 있는 각. 평행할 때 크기가 같음<br>- **역**: 동위각(또는 엇각)의 크기가 같으면 두 직선은 평행<br>- **꺾인 선 문제**: 평행선 사이 꺾인 점에서 평행한 보조선을 그어 해결 |
| **주요 오개념** | - **"동위각은 항상 같다"**: **평행할 때만** 같음! 평행 아닌 경우 반례 필수<br>- 동위각과 엇각의 위치를 혼동 |
| **출제 포인트** | - 평행하지 않은 두 직선의 동위각 반례 그림<br>- 보조선을 긋고 동위각/엇각을 활용하여 미지의 각도 구하기 |

#### 세부 개념 C: 작도와 삼각형 합동

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **작도 도구**: 눈금 없는 자(선 긋기, 연장) + 컴퍼스(원 그리기, 거리 옮기기)<br>- **기본 작도**: 길이가 같은 선분, 크기가 같은 각, 각의 이등분선, 수직이등분선<br>- **삼각형 합동 조건**: SSS, SAS, ASA |

| 합동 조건 | 표기 | 설명 및 주의사항 |
|-----------|------|------------------|
| SSS | Side-Side-Side | 세 변의 길이가 모두 같음 |
| SAS | Side-Angle-Side | 두 변과 **끼인각(Included Angle)**이 같음. 끼인각이 아닌 다른 각은 불가 |
| ASA | Angle-Side-Angle | 한 변과 양 끝각이 같음. (AAS도 180도 이용시 ASA로 환원 가능) |

| **주요 오개념** | - **SSA 합동 오류**: 끼인 각이 아닌 다른 한 각이 같을 때도 합동이라 판단. 작도 반례(Ambiguous Case) 제시 필수<br>- **대응 요소 혼동**: 합동 기호 순서가 대응점 순서임을 모름 ($\triangle ABC \equiv \triangle DEF$ -> $A \leftrightarrow D$) |
| **출제 포인트** | - SAS에서 **'끼인 각'** 중요성 각인<br>- 복합 도형에서 숨겨진 합동 삼각형 찾기 (공통변, 맞꼭지각 활용) |

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **히스토그램**: 직사각형 막대로 자료의 분포 시각화. 막대 높이 = 도수<br>- **도수분포다각형**: 히스토그램의 각 막대 윗변 중점을 연결한 꺾은선<br>- **넓이 관계**: 도수분포다각형과 가로축으로 둘러싸인 넓이 = 히스토그램 넓이의 합 = (계급의 크기) $\times$ (총 도수) |
| **주요 오개념** | - **히스토그램 축 혼동**: 막대 높이가 '변량의 크기'라고 착각 -> 높이는 **'도수(빈도)'** |

#### 세부 개념 C: 상대도수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **상대도수**: $\frac{\text{해당 계급의 도수}}{\text{전체 도수}}$<br>- **성질**: 상대도수의 총합은 항상 1<br>- **활용**: 도수의 총합이 다른 두 집단의 분포를 비교할 때 사용 |
| **출제 포인트** | - 상대도수를 이용한 두 집단 비교 (예: A중학교와 B중학교 키 분포)<br>- 계급값의 의미와 평균 추정 과정 이해 |

### 5단원: 대푯값 (NEW: 중3->중1 이동)

#### 세부 개념 A: 평균/중앙값/최빈값

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **평균(Mean)**: 총합 / 개수. 극단값(Outlier)에 취약<br>- **중앙값(Median)**: 크기순 나열 시 가운데 값. 짝수 개면 가운데 두 값의 평균. 극단값 영향 안 받음<br>- **최빈값(Mode)**: 가장 자주 나타나는 값. 2개 이상이거나 없을 수도 있음 |
| **주요 오개념** | - **중앙값 정렬 누락**: 주어진 순서대로 가운데 값 찾음 (크기순 정렬 안 함)<br>- **짝수 개 데이터**: "가운데 숫자 없으므로 중앙값 없다"<br>- **대표성 오해**: "평균이 항상 가장 정확한 대푯값이다" |
| **출제 포인트** | - 대푯값 계산 (정렬 후 중앙값, 최빈값 식별)<br>- 데이터 하나가 변할 때 평균/중앙값 변화 추론 |

#### 세부 개념 B: 상황별 대푯값 선택

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **극단값이 있을 때**: 평균은 왜곡됨 -> 중앙값이 더 대표적<br>- **맥락적 판단**: 어떤 대푯값이 상황에 가장 적절한지 선택하는 능력<br>- **데이터 리터러시**: 계산보다 **의미와 맥락적 판단** 중시 |
| **출제 포인트** | - "재벌 2세(용돈 1억)가 있는 반의 평균 vs 중앙값 중 어느 것이 더 대표적인가?"<br>- 데이터 리터러시 관점의 서술형 문항 |

### 6단원: 산점도와 상관관계 (NEW: 중3->중1 이동)

#### 세부 개념 A: 산점도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **산점도(Scatter Plot)**: 두 변량 $x, y$를 좌표평면에 점으로 표현<br>- **양의 상관관계**: $x$가 증가하면 $y$도 증가 (우상향 분포)<br>- **음의 상관관계**: $x$가 증가하면 $y$는 감소 (우하향 분포)<br>- **상관관계 없음**: 원 모양 또는 축에 평행한 분포 |
| **출제 포인트** | - 산점도 위 보조선 긋고 특정 영역 학생 수 세기<br>- 실생활 예시로 상관관계 유형 판별 (양: 기온-아이스크림, 음: 기온-난방비, 무: 신발-지능) |

#### 세부 개념 B: 상관관계와 인과관계

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **상관관계 $\neq$ 인과관계**: 상관관계는 경향성일 뿐, 원인-결과를 보장하지 않음<br>- **선형성 오해 교정**: "모든 점이 직선 위에 있어야 상관관계"가 아님. 분포의 경향(Trend)만으로도 성립 |
| **주요 오개념** | - **인과관계 혼동**: "키와 몸무게가 양의 상관 -> 키가 크기 **때문에** 몸무게가 많이 나간다"<br>- **제3변수 무시**: 아이스크림 판매량과 익사 사고는 양의 상관 -> 진짜 원인은 '기온'(제3변수) |
| **출제 포인트** | - 인지적 갈등 유발하는 반례 + 토론 수업<br>- "상관관계가 있으면 인과관계도 있다" 참/거짓 문항 |
