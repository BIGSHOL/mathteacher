### 2단원: 약수와 배수 ⭐ (정수론의 기초)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **약수·배수 정의**: $A = B \times C$일 때<br>  - $A$는 $B, C$의 **배수**<br>  - $B, C$는 $A$의 **약수**<br>• **최대공약수(GCD)**: 공약수 중 가장 큰 수<br>  → "두 수의 공약수 = 최대공약수의 약수"<br>• **최소공배수(LCM)**: 공배수 중 가장 작은 수<br>  → "두 수의 공배수 = 최소공배수의 배수"<br>• **관계식**: (두 수의 곱) = GCD × LCM |
| **주요 오개념** | • **⭐ 약수 누락**: 1과 자기 자신 빠뜨림<br>• **배수의 유한성 착각**: 배수는 **무한**함을 모름<br>• **⭐ 최대공배수/최소공약수 오류**: 존재하지 않는 개념 구하려 함<br>  → 최소공약수 = 항상 1, 최대공배수 = 무한대<br>• **용어 혼동**: '최대'공약수가 크다고 착각, '최소'공배수가 작다고 착각 |
| **출제 포인트** | ✓ "12를 나누어떨어지게 하는 수를 모두 찾으시오" (정의 기반 발문)<br>✓ "왜 최대공배수는 구할 수 없는가?" 서술형<br>✓ 곱과 GCD만 알 때 LCM 구하기 (관계식 활용) |

### 5단원: 분수의 덧셈과 뺄셈 ⭐ (1학기 결정판)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **이종 분모 연산 원리**: 분모가 다르면 '단위'가 다름 → 통분으로 단위 일치<br>• **대분수 연산 전략**:<br>  ① 자연수끼리, 분수끼리 따로 계산<br>  ② 가분수로 고쳐서 계산<br>• **받아올림/받아내림**: 분수끼리 합이 가분수 → 받아올림, 뺄 수 없으면 → 받아내림 |
| **주요 오개념** | • **⭐⭐⭐ 분모끼리/분자끼리 덧셈**: $\frac{1}{2} + \frac{1}{3} = \frac{2}{5}$<br>  (가장 흔하고 치명적인 오개념)<br>  → 피자 조각/수직선 모델로 $\frac{2}{5}$가 $\frac{1}{2}$보다 작음을 시각적 증명<br>• **⭐ 받아내림 시 십진법 혼동**: 자연수에서 빌려온 1을 분자에 **10**으로 더함<br>  (정답: **분모만큼** 더해야 함)<br>• **가분수/대분수 처리**: 답을 대분수로 안 고침 |
| **출제 포인트** | ✓ 피자 조각 그림으로 $\frac{1}{2} + \frac{1}{3} \neq \frac{2}{5}$ 시각적 증명<br>✓ 틀린 계산 과정 제시 → 오류 찾기(Error Correction) 문항 |

### 6단원: 다각형의 둘레와 넓이

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **단위넓이**: $1cm^2, 1m^2$ = 단위 정사각형의 개수<br>• **등적 변형과 공식 유도**:<br>  - 평행사변형: 잘라 붙여 직사각형 → 밑변 × 높이<br>  - 삼각형: 2개 붙여 평행사변형 → 밑변 × 높이 ÷ 2<br>  - 마름모: 한 대각선 × 다른 대각선 ÷ 2<br>  - 사다리꼴: (윗변 + 아랫변) × 높이 ÷ 2<br>• **둘레 ↔ 넓이 독립성**: 둘레 길어도 넓이 넓다고 할 수 없음 |
| **주요 오개념** | • **⭐ ÷2 누락**: 삼각형, 마름모, 사다리꼴에서 마지막 2로 나누기 잊음<br>  → 공식 유도 과정(배적 변형) 잊고 결과만 암기했을 때 발생<br>• **⭐ 높이 오인**: 평행사변형·삼각형에서 '높이'를 빗변 길이로 착각<br>  → 높이 = 밑변과 **수직**인 거리<br>• **직사각형 편향**: 모든 사각형을 가로 × 세로로 계산 |
| **출제 포인트** | ✓ "삼각형 넓이 공식을 평행사변형으로 설명하시오" 서술형<br>✓ 밑변·빗변만 주고 높이 안 주는 **함정 문제**<br>✓ 밑변·높이 같은 여러 삼각형 → 넓이 같은 것 찾기 (카발리에리 원리) |

### 1단원: 수의 범위와 어림하기

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **범위 용어**:<br>  - 이상($\ge$), 이하($\le$): 기준수 **포함** (●)<br>  - 초과($>$), 미만($<$): 기준수 **미포함** (○)<br>• **어림 3가지**:<br>  - **올림**: 부족하면 안 되는 상황 (텐트 개수)<br>  - **버림**: 넘치면 안 되는 상황 (구매 가능 개수)<br>  - **반올림**: 가장 가까운 근삿값 (0~4 버림, 5~9 올림) |
| **주요 오개념** | • **⭐ 경계값 포함 여부 혼동**: "50 이상"에 50 포함 여부<br>• **⭐⭐ 자릿수 표현 언어 혼동**:<br>  - "십의 자리**에서** 반올림" → 십의 자리 보고 판단 → **백의 자리**까지 나타냄<br>  - "십의 자리**까지** 나타내시오" → 일의 자리에서 반올림 |
| **출제 포인트** | ✓ 상황 판단: 올림/버림/반올림 중 적절한 것 선택<br>✓ 수직선에 범위 정확히 작도 |

### 2단원: 분수의 곱셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **연산자로서의 분수**: $6 \times \frac{1}{3}$ = 6을 3등분 한 것 중 1<br>• **면적 모델**: 진분수 × 진분수 = 가로·세로 쪼개어 겹치는 부분<br>  → 전체의 부분의 부분(part of part) → 전체보다 작아짐<br>• **약분의 효율성**: 곱셈 중 대각선 방향으로 미리 약분 |
| **주요 오개념** | • **⭐⭐ 대분수 곱셈 오류**: $1\frac{1}{2} \times 1\frac{1}{2}$를<br>  자연수끼리, 분수끼리 따로 곱해서 $1\frac{1}{4}$이라 답함<br>  → 덧셈/뺄셈 습관 전이, 곱셈은 **반드시 가분수로 변환**<br>  (정답: $\frac{3}{2} \times \frac{3}{2} = \frac{9}{4} = 2\frac{1}{4}$)<br>• **축소 변환 불신**: 진분수 곱하면 작아지는 것을 의심 |
| **출제 포인트** | ✓ 면적 모델로 진분수 × 진분수 시각화<br>✓ 대분수 곱셈 → 가분수 변환 필수 강조 |

### 3단원: 합동과 대칭

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **합동**: 모양과 크기가 같아 포개면 완전히 겹침<br>• **대응점, 대응변, 대응각**: 합동 도형에서 겹치는 요소 (길이·크기 같음)<br>• **선대칭도형**: 대칭축을 따라 접으면 완전히 겹침<br>• **점대칭도형**: 대칭의 중심을 기준으로 180° 돌리면 겹침 |
| **주요 오개념** | • **방향에 따른 합동 인지 오류**: 뒤집히거나 회전된 도형을 합동 아니라고 착각<br>• **⭐⭐ 대칭축 오류**:<br>  - 평행사변형의 대각선을 대칭축으로 착각 (선대칭 아님!)<br>  - 직사각형 대칭축 = 대각선(×), 각 변의 중점 연결(○)<br>• **점대칭 작도 오류**: 대응점을 대칭 중심 통과 반대편 같은 거리에 정확히 안 찍음 |
| **출제 포인트** | ✓ "선대칭도형의 대칭축을 잘못 그린 것을 모두 고르시오" (평행사변형 함정)<br>✓ 도형 일부만 주고 대칭 성질로 완성하기 |

### 6단원: 평균과 가능성

|------|------|
| **핵심 개념** | • **평균의 의미**: 자료를 고르게 하여 전체 대표 (Leveling out)<br>  → "튀어나온 곳 깎고 부족한 곳 채우는" 평탄화 개념<br>• **가능성**: 불가능하다 ~ 확실하다를 $0, \frac{1}{2}, 1$ 등 수치로 대응 (확률의 직관적 도입) |
| **주요 오개념** | • **평균의 대표성 오해**: 평균 80점이면 모든 과목 80점 근처라 착각<br>  → 극단값(0점)이 평균에 미치는 영향 이해 필요<br>• **⭐ 0의 처리**: 0점인 과목도 **자료의 수(나누는 수)**에 포함해야 함<br>• **가능성의 주관적 해석**: 수학적 확률이 아닌 희망사항으로 판단 |
| **출제 포인트** | ✓ 극단값이 평균에 미치는 영향 문제<br>✓ 0이 포함된 자료의 평균 계산 |
