## 초5 수학의 특징

- **위상**: 한국 수학교육의 **가장 결정적인 분수령**, '수포자' 대거 발생 시기
- **인지 발달**: 구체적 조작기 → 형식적 조작기 **이행 과도기(Critical Transition Period)**
- **학문적 전환**:
  - 자연수 사칙연산 완성 → **유리수(분수, 소수)** 영역으로 본격 확장
  - 직관적 탐구 → **논리적 정당화(Justification)**로 발전
  - 산술적 사고(Arithmetic) → **대수적 사고(Algebraic Thinking)** 태동
- **2022 개정 핵심 변화**:
  - **5대 핵심 역량**: 문제해결, 추론, 의사소통, 연결, 정보처리로 재구조화
  - **수학적 모델링**: 실생활 문제를 수학적으로 변환·해결·해석
  - **정당화(Justification)** 및 **공학 도구 활용** 강조
  - **과정 중심 평가**: 결과 중심 → 사고 과정 추적, 서술형 강화
  - **디지털 소양**: 알지오매스(AlgeoMath) 등 공학 도구 전면 활용

> 개념 문제는 단순 연산 숙달이 아닌 **'수학적 소양(Mathematical Literacy)'** 함양에 초점

### 1단원: 자연수의 혼합 계산 - 연산 위계의 구조화

> 덧셈, 뺄셈, 곱셈, 나눗셈이 섞인 식을 다루며, 수학적 약속(Convention)으로서의 연산 순서를 내면화하는 과정. 대수학에서 연산자 우선순위를 이해하는 첫걸음이며, 다항식 계산의 필수 절차적 지식.

#### 세부 개념 01: 연산 우선순위와 좌결합성 (`mixed-calc-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **연산의 위계성**: ① 괄호 ( ) 가장 먼저 → ② 곱셈·나눗셈 → ③ 덧셈·뺄셈<br>- **좌결합성(Left Associativity)**: 동순위는 왼쪽 → 오른쪽 순서로 계산<br>  (예: $20 \div 4 \times 2 = 5 \times 2 = 10$, 40 아님!)<br>- 연산 순서는 임의가 아닌 **수학적 약속(Convention)** |
| **주요 오개념** | - **선형적 계산 오류** (가장 빈출): 위계 무시하고 왼쪽→오른쪽 순서대로만 계산<br>  (예: $10 + 5 \times 2 = 15 \times 2 = 30$, 정답: $10 + 10 = 20$)<br>- **곱셈·나눗셈 간 우선순위 착각**: "곱셈이 나눗셈보다 먼저"라고 오해 (같은 위계!)<br>- **괄호 무용론**: 괄호 있어도/없어도 같다고 생각 |
| **문항 설계 지침** | ✓ 계산 순서 표시 문항: 식에 번호 매기고 단계별 계산<br>✓ 반례 제시: $8 + 2 \times 3 = 14$ vs $(8 + 2) \times 3 = 30$<br>✓ 좌결합성 확인: $24 \div 4 \div 2$ (왼쪽부터 6 → 3)<br>✓ 계산 순서 비교: 같은 숫자·연산자인데 결과 다른 식 대조 |

#### 세부 개념 02: 괄호와 문장제 모델링 (`mixed-calc-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 문장제 문제를 하나의 통합된 식(One-line expression)으로 모델링<br>- **괄호의 기능**: 연산 순서를 명시적으로 조정<br>- 부분 계산 없이 전체 구조를 파악하는 정보처리 역량<br>- "어떤 수" 구하기: 역연산 논리 이해 |
| **주요 오개념** | - **역연산 순서 혼동**: "어떤 수에 5를 곱하고 3을 더하면 23" → $\frac{23-3}{5}$를 $\frac{23}{5}-3$으로 착각<br>- **식 세우기와 계산 분리 못함**: 문장을 읽으며 중간에 계산해버림<br>- 괄호 누락: 복잡한 상황에서 괄호 안 쓰고 순서 의존 |
| **문항 설계 지침** | ✓ 문장제 → 하나의 식으로 표현 후 계산<br>✓ 목표수 만들기: 숫자 카드(3,5,8,2)와 연산 기호·괄호로 24 만들기<br>✓ 오류 수정형: 잘못 계산된 식 → 오류 단계 찾아 고치기<br>✓ 괄호 위치 비교: 괄호 위치만 바꿨을 때 결과 비교 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 계산 순서 표시 | 계산 순서 번호 매기고 단계별 계산 | 하 |
| 좌결합성 확인 | 곱셈·나눗셈 혼합에서 순서대로 계산 | 중 |
| 괄호 비교 | 괄호 위치 변화에 따른 결과 비교 | 중 |
| 오류 수정형 | 잘못 계산된 식에서 오류 단계 찾기 | 중상 |
| 목표수 만들기 | 숫자 카드와 연산 기호로 특정 값 만들기 | 상 |

### 2단원: 약수와 배수 - 정수론의 기초

> 자연수의 구성 원리를 탐구하는 단원. 수의 성질을 분석하고 분수의 사칙연산(약분, 통분)으로 연결하는 핵심 고리.

#### 세부 개념 01: 약수·배수의 정의와 관계 (`divisor-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **약수**: 어떤 수를 나누어떨어지게 하는 수<br>- **배수**: 어떤 수를 1배, 2배, 3배... 한 수<br>- **상호 관계**: $21 = 3 \times 7$에서 3, 7은 21의 약수이고, 21은 3, 7의 배수<br>- **유한성·무한성**: 약수는 유한, 배수는 무한<br>- 1은 모든 자연수의 약수, 자기 자신은 자신의 약수이자 배수 |
| **주요 오개념** | - **약수 누락**: 1과 자기 자신을 빠뜨림<br>- **배수의 유한성 착각**: 배수를 몇 개만 쓰고 끝<br>- **약수·배수 언어 혼동**: "12는 3의 약수다"처럼 관계 역전<br>- **0의 처리**: 0을 약수로 포함하거나 배수에서 제외 |
| **문항 설계 지침** | ✓ 정의 기반 발문: "12를 나누어떨어지게 하는 수를 모두 찾으시오"<br>✓ 직사각형 배열 모델: 바둑돌 12개를 배열하여 약수 발견<br>✓ 약수·배수 관계 문장 완성: "24는 ( )의 배수이고 ( )의 약수다" (없음)<br>✓ 배수의 무한성 설명: "6의 배수는 왜 무한한가?" |

#### 세부 개념 02: 최대공약수와 최소공배수 (`divisor-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **최대공약수(GCD)**: 공약수 중 가장 큰 수. "두 수의 공약수 = GCD의 약수"<br>- **최소공배수(LCM)**: 공배수 중 가장 작은 수. "두 수의 공배수 = LCM의 배수"<br>- **핵심 관계식**: $A \times B = \text{GCD}(A,B) \times \text{LCM}(A,B)$<br>- 구하는 방법: 나열법, 공통 약수/배수 찾기 |
| **주요 오개념** | - **최대공배수/최소공약수 오류**: 존재하지 않는 개념을 구하려 함<br>  (최소공약수=항상 1, 최대공배수=무한대)<br>- **직관 오류**: '최대'공약수가 두 수보다 크다고 착각, '최소'공배수가 두 수보다 작다고 착각<br>- **관계식 미인지**: 곱과 GCD만 알 때 LCM 못 구함 |
| **문항 설계 지침** | ✓ 존재하지 않는 개념 구분: "최대공배수가 존재하는가?"<br>✓ 수의 성질 활용: "두 수의 공배수는 최소공배수의 배수다"<br>✓ 관계식 적용: $A \times B = 360$, $\text{GCD}=12$일 때 $\text{LCM}$ 구하기<br>✓ 실생활 활용 제한: 과도하게 복잡한 문장제 지양 |

| 활용 유형 | 상황 | 모델링 핵심 |
|-----------|------|------------|
| GCD 활용 | 타일 붙이기 (가장 큰 정사각형) | 가로·세로의 GCD |
| GCD 활용 | 사탕 똑같이 나누기 | 물건 개수의 GCD |
| LCM 활용 | 버스 배차 간격 (동시 출발) | 배차 간격의 LCM |
| LCM 활용 | 톱니바퀴 맞물리기 | 톱니 수의 LCM |
| 관계식 | 곱과 GCD만 알 때 LCM 구하기 | $\text{GCD} \times \text{LCM} = A \times B$ |

### 3단원: 규칙과 대응 - 함수적 사고의 도입

> '변화와 관계' 영역의 핵심. 중학교 함수($y=f(x)$) 개념을 초등 수준에서 직관적으로 다루는 대수적 사고(Algebraic Thinking)의 출발점.

#### 세부 개념 01: 대응 관계 파악 (`corresp-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **변수의 도입**: 변화하는 두 양을 ○, △, □로 표현<br>- **대응표 읽기**: 가로적 관계(한 양이 변할 때 다른 양의 변화) 파악<br>- **세로적 vs 가로적**: 세로(Recursive, +3씩) vs 가로(Functional, 3배 관계)<br>- 패턴 인식: 규칙적으로 변하는 두 양의 관계 발견 |
| **주요 오개념** | - **세로적 관계 편향** (가장 빈출): 가로적 관계보다 한 양의 변화 패턴에만 주목<br>  (예: y가 3, 6, 9일 때 "3씩 커진다"만 답하고 "y는 x의 3배"를 못 도출)<br>- **덧셈 규칙 ↔ 곱셈 규칙 혼동**: $y = x + 3$ vs $y = 3x$ 구분 못함<br>- 표에서 규칙 찾기는 하지만 일반화하지 못함 |
| **문항 설계 지침** | ✓ 표의 빈칸 채우기: 일부만 주고 규칙 유추<br>✓ "x와 y의 관계를 말로 설명하시오" (가로적 관계 유도)<br>✓ 세로적/가로적 관계 모두 쓰기: 두 관점 비교<br>✓ 함수 상자 모델: 입력→변환→출력 시각화 |

#### 세부 개념 02: 대응식 표현과 해석 (`corresp-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **대응 관계의 일반화**: "자전거 수 × 2 = 바퀴 수" → ○ × 2 = △<br>- **식의 의미 해석**: 기호로 나타낸 식을 문장으로 설명<br>- **복합 규칙**: $y = 2x + 2$ (책상 연결하기) 같은 비단순 패턴<br>- 큰 수 예측: "100번째 값은?" 질문으로 식의 필요성 정당화 |
| **주요 오개념** | - **복합 규칙 오류**: 책상 연결하기($2n+2$)를 $4n$으로 오인<br>- **식 해석 불가**: ○ × 3 + 1 = △를 "동그라미에 3을 곱하고 1을 더하면 세모"로만 읽고 구체적 의미 못 설명<br>- **역대응 오류**: △=15일 때 ○ 구하기 (역연산 적용 못함) |
| **문항 설계 지침** | ✓ 식으로 나타내기: 대응표 → ○와 △의 관계식<br>✓ 식 해석하기: ○ × 5 = △ → "세모는 동그라미의 몇 배인가?"<br>✓ 100번째 값 예측: 일일이 세기 불가능한 큰 수 질문<br>✓ 실생활 맥락: 물건 값, 운동회 좌석 배열 등 연결 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 표 완성 | 규칙 찾아 빈칸 채우기 | 하 |
| 관계 설명 | 두 양의 관계를 말로 표현 | 중 |
| 식으로 나타내기 | 대응 관계를 기호식으로 표현 | 중상 |
| 복합 규칙 | $y = 2x + 1$ 같은 비단순 패턴 | 상 |
| 역대응 | y값 주고 x값 구하기 | 상 |

### 4단원: 약분과 통분 - 동치분수의 이해

> 수학적 동치류(Equivalence Class)를 초등 수준에서 경험하는 단원. 분수의 크기가 같음을 이해하고 변형하는 핵심 기술 습득.

#### 세부 개념 01: 약분과 기약분수 (`reduce-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **분수의 기본 성질**: 분모·분자에 0이 아닌 같은 수를 곱하거나 나누어도 크기 불변<br>- **약분**: 공약수로 나눠 간단히 만들기<br>- **기약분수**: 분모·분자의 공약수가 1뿐인 분수<br>- 약분 전략: 최대공약수로 한 번에 약분 vs 단계적 약분 |
| **주요 오개념** | - **가법적 오류(Additive Error)**: 분모·분자에 같은 수를 '더하거나 빼도' 크기 같다고 착각<br>  (예: $\frac{1}{2} = \frac{1+1}{2+1} = \frac{2}{3}$ ← 틀림!)<br>- **약분 불완전**: 2로 한 번 나누고 멈춤 (기약분수까지 계속해야)<br>- 0으로 나누기: 분모·분자에 0을 곱해도 같다고 착각 |
| **문항 설계 지침** | ✓ 가법적 오류 반례: 영역 모델로 $\frac{1}{2} \neq \frac{2}{3}$ 시각적 증명<br>✓ 기약분수 판별: 주어진 분수가 기약분수인지 확인<br>✓ 약분 과정 단계별 서술<br>✓ 종이접기 활동: 전체 양 보존 체험 |

#### 세부 개념 02: 통분과 크기 비교 (`reduce-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **통분**: 이분모 분수의 크기 비교나 연산을 위해 공통 단위(공통분모) 설정<br>- **공통분모 선택**: 두 분모의 곱 vs 최소공배수(효율적)<br>- **크기 비교 전략**: 통분 후 분자 크기 비교 또는 수직선 활용<br>- 분수의 순서: 세 개 이상 분수 크기순 배열 |
| **주요 오개념** | - **통분 시 분자 연산 누락** (매우 빈출): 분모만 바꾸고 분자는 그대로<br>  (예: $\frac{1}{2} \rightarrow \frac{1}{6}$ ← 분자에도 ×3 해야 $\frac{3}{6}$)<br>- **크기 비교 직관 오류**: 분모·분자 숫자가 크면 큰 수라고 착각<br>- 공통분모를 항상 두 분모의 곱으로만 선택 (비효율) |
| **문항 설계 지침** | ✓ 통분 과정에서 분자 변환 확인 문항<br>✓ 수직선 위에 분수 표시하여 크기 비교<br>✓ 세 분수 이상 크기순 배열<br>✓ 통분 방법 비교: 두 분모의 곱 vs LCM (효율성 논의) |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 약분 기본 | 공약수로 나누어 기약분수로 | 하 |
| 기약분수 판별 | 주어진 분수가 기약분수인지 확인 | 중 |
| 통분 과정 | 분자·분모 모두 올바르게 변환 | 중 |
| 크기 비교 | 통분 후 분자 크기 비교 | 중상 |
| 순서 배열 | 세 분수 이상 크기순 배열 | 상 |

### 5단원: 분수의 덧셈과 뺄셈 - 연산의 확장 ⭐ (1학기 핵심)

> 통분을 도구로 사용하여 이분모 분수의 사칙연산을 수행. 전 세계적으로 가장 보편적인 오개념이 집중된 단원.

#### 세부 개념 01: 이분모 분수의 덧뺄셈 원리 (`frac-add-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **핵심 원리**: 분모가 다르면 '단위'가 다름 → 통분으로 단위 일치 후 연산<br>- **알고리즘**: 통분 → 분자끼리 연산 → 약분(기약분수)<br>- **단위의 통일성**: "사과 1개 + 배 1개 = 과일 2개, 사과 2개 아님" 비유<br>- 대분수 덧셈: 자연수끼리·분수끼리 분리 또는 가분수 변환 |
| **주요 오개념** | - **분모끼리/분자끼리 덧셈** (전 세계 가장 보편적 오개념):<br>  $\frac{1}{2} + \frac{1}{3} = \frac{2}{5}$ ← 분수를 양(Quantity)이 아닌 두 자연수의 쌍으로 인식<br>- **단위 무시**: 서로 다른 단위(분모)에서 직접 연산<br>- 통분 없이 "분모가 작은 쪽에 맞춤" 같은 잘못된 규칙 |
| **문항 설계 지침** | ✓ 피자 조각/수직선 모델로 $\frac{2}{5}$가 $\frac{1}{2} + \frac{1}{3}$보다 작음을 시각적 증명<br>✓ 오류 수정형: $\frac{1}{2} + \frac{1}{3} = \frac{2}{5}$ 틀린 이유 설명<br>✓ 단위 비유 활용: "2미터 + 3센티미터 = 5미터센티미터?" 유사 오류<br>✓ 통분 과정 단계별 서술 |

#### 세부 개념 02: 대분수의 받아올림·받아내림 (`frac-add-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **받아올림**: 분수끼리 합이 가분수가 되면 자연수 부분으로 올림<br>- **받아내림**: 분수끼리 뺄 수 없으면 자연수에서 1을 빌려옴<br>- **핵심**: 빌려온 1 = $\frac{\text{분모}}{\text{분모}}$ (분모만큼 더해야 함)<br>- 가분수/대분수 변환 숙달 필수 |
| **주요 오개념** | - **받아내림 시 십진법 혼동** (매우 빈출): 자연수에서 빌려온 1을 분자에 **10**으로 더함<br>  (정답: **분모만큼** 더해야. 분모가 5이면 $\frac{5}{5}$)<br>- 가분수/대분수 변환 미처리: 답을 대분수로 안 고침<br>- 자연수 - 분수 계산 오류: $3 - \frac{2}{5}$ 처리 못함 |
| **문항 설계 지침** | ✓ 받아내림 과정 단계별 서술<br>✓ 틀린 계산 과정 제시 → 오류 찾기(Error Correction)<br>✓ 자연수 - 분수 형태 문제<br>✓ 시각 모델: 막대 그림으로 받아내림 과정 표현 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 동분모 연산 | 분모 같은 분수 덧뺄셈 | 하 |
| 이분모 기본 | 통분 후 분자끼리 연산 | 중 |
| 받아올림 | 분수 합이 1 넘을 때 처리 | 중상 |
| 받아내림 | 자연수에서 빌려와 계산 | 상 |
| 오류 수정 | 잘못된 계산에서 오류 찾기 | 상 |

### 6단원: 다각형의 둘레와 넓이 - 측정과 정당화

> 2022 개정 교육과정에서 가장 강조하는 '정당화' 활동이 활발히 일어나는 기하 영역. 공식 암기 지양, 공식 유도 과정 경험 필수.

#### 세부 개념 01: 평행사변형과 삼각형의 넓이 (`poly-area-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **등적 변형**: 도형을 잘라 붙여 모양은 다르지만 넓이 같은 도형으로<br>  (평행사변형 → 직사각형)<br>- **배적 변형**: 도형을 복사하여 2배로 만든 뒤 넓이 구하고 ÷2<br>  (삼각형 2개 → 평행사변형)<br>- **공식 유도**:<br>  - 평행사변형: 밑변 × 높이<br>  - 삼각형: 밑변 × 높이 ÷ 2<br>- **높이의 정의**: 밑변과 수직인 거리 (빗변 아님!) |
| **주요 오개념** | - **÷2 누락** (가장 빈출): 삼각형에서 마지막 ÷2 잊음<br>  → 공식 유도 과정 잊고 결과만 암기했을 때 발생<br>- **높이 오인**: 평행사변형·삼각형에서 '높이'를 빗변 길이로 착각<br>- **직사각형 공식 적용**: 모든 사각형을 가로 × 세로로 계산 |
| **문항 설계 지침** | ✓ "삼각형 넓이 공식을 평행사변형으로 설명하시오" 서술형<br>✓ 밑변·빗변만 주고 높이 안 주는 **함정 문제**<br>✓ 등적 변형 활동: 종이 오려 직사각형으로 만들기<br>✓ 같은 밑변·높이 다른 모양 삼각형 → 넓이 같음 확인 |

#### 세부 개념 02: 사다리꼴·마름모와 둘레 (`poly-area-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **사다리꼴**: (윗변 + 아랫변) × 높이 ÷ 2 (뒤집어 붙여 평행사변형 → ÷2)<br>- **마름모**: 한 대각선 × 다른 대각선 ÷ 2 (대각선 잘라 직사각형 → ÷2)<br>- **둘레와 넓이의 독립성**: 둘레 길어도 넓이 크다고 할 수 없음<br>- **단위넓이**: $1cm^2$, $1m^2$ = 단위 정사각형 개수 |
| **주요 오개념** | - **÷2 누락**: 사다리꼴, 마름모에서도 마지막 ÷2 잊음<br>- **둘레-넓이 혼동**: 둘레 길면 넓이도 크다는 직관적 오류<br>- **마름모를 대각선으로만 계산**: 밑변×높이 공식도 가능함을 모름<br>- 대각선을 빗변으로 착각 |
| **문항 설계 지침** | ✓ 공식 유도 과정 서술: "왜 사다리꼴은 ÷2를 하는가?"<br>✓ 둘레 같은 도형 여러 개 → 넓이 비교<br>✓ 마름모 넓이 두 가지 방법으로 구하기<br>✓ 공학 도구(알지오매스): 평행사변형 밀기(Shear) → 넓이 일정 확인 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 공식 적용 | 밑변·높이 주고 넓이 구하기 | 하 |
| 높이 함정 | 빗변만 주고 높이 없는 문제 | 중 |
| 공식 유도 | "왜 삼각형은 ÷2를 하는가?" 서술 | 중상 |
| 복합 도형 | 여러 도형 조합된 넓이 구하기 | 상 |
| 둘레-넓이 독립성 | 둘레 같아도 넓이 다른 경우 | 상 |

## 2학기 단원별 정리

> 2학기는 수의 범위를 확장하여 실생활 문제 해결력을 높이고, 합동과 대칭으로 공간 감각을 기르며, 자료와 가능성으로 통계적 소양을 함양한다.

### 1단원: 수의 범위와 어림하기 - 실용적 수 감각

> 연속적인 양(Continuous Quantity)을 다루기 위한 근삿값의 개념 도입. 중고등학교 부등식 영역의 기초.

#### 세부 개념 01: 수의 범위 (부등식의 기초) (`range-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **이상($\ge$)**: 기준수 **포함**, "그 수보다 크거나 같다"<br>- **이하($\le$)**: 기준수 **포함**, "그 수보다 작거나 같다"<br>- **초과($>$)**: 기준수 **미포함**, "그 수보다 크다"<br>- **미만($<$)**: 기준수 **미포함**, "그 수보다 작다"<br>- 수직선 표현: 꽉 찬 점($\bullet$) vs 빈 점($\circ$) |
| **주요 오개념** | - **경계값 포함 여부 혼동**: "50 이상"에 50 포함되는지 모름<br>- **이상/초과, 이하/미만 용어 혼동**: 일상 언어와 수학 언어 차이<br>- 수직선에 범위 표시할 때 점 종류 혼동 |
| **문항 설계 지침** | ✓ 경계값 확인: "50 이상인 수 중 가장 작은 자연수는?"<br>✓ 수직선에 범위 정확히 작도<br>✓ 실생활 맥락: "만 12세 이상 입장 가능" 해석<br>✓ 용어 구분: 네 개념 비교 문항 |

#### 세부 개념 02: 어림하기 (올림, 버림, 반올림) (`round-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **올림**: 부족하면 안 되는 상황 (텐트 개수, 화물차 대수)<br>- **버림**: 넘치면 안 되는 상황 (구매 가능 개수, 엘리베이터 정원)<br>- **반올림**: 가장 가까운 근삿값 (0~4 버림, 5~9 올림)<br>- **자릿수 표현 언어**:<br>  - "십의 자리**에서** 반올림" → 십의 자리 숫자 보고 판단 → **백의 자리**까지<br>  - "십의 자리**까지** 나타내시오" → 일의 자리에서 반올림 |
| **주요 오개념** | - **자릿수 표현 언어 혼동** (가장 빈출): "~에서" vs "~까지" 구분 못함<br>- **반올림 연속 적용 오류**: 하위 자리부터 차례로 올리는 오류<br>- **5 처리 혼동**: 5를 올림하는지 버림하는지 헷갈림 (반올림: 올림) |
| **문항 설계 지침** | ✓ 상황 판단: 올림/버림/반올림 중 적절한 것 선택<br>✓ 자릿수 언어 명확히: "백의 자리에서 반올림" vs "백의 자리까지"<br>✓ 실생활 데이터: 인구, 예산 등 큰 수의 어림<br>✓ 어림의 필요성과 오차 허용 범위 토론 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 범위 표현 | 이상·이하·초과·미만 구분 | 하 |
| 수직선 작도 | 범위를 수직선에 표시 | 중 |
| 상황 판단 | 어림 방법 선택 (올림/버림/반올림) | 중상 |
| 자릿수 언어 | "~에서" vs "~까지" 구분 | 상 |

### 2단원: 분수의 곱셈 - 연산 모델의 전환

> 덧셈적 사고에서 승법적 사고(Multiplicative Thinking)로 완전히 전환되는 지점.

#### 세부 개념 01: 분수×자연수와 동수누가 (`frac-mul-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **(분수) × (자연수)**: 분자에 자연수를 곱함<br>  $\frac{2}{5} \times 3 = \frac{2 \times 3}{5} = \frac{6}{5}$<br>- **(자연수) × (분수)**: 동수누가 개념<br>  $6 \times \frac{1}{3}$ = "6을 3등분 한 것 중 1" = $\frac{6}{3} = 2$<br>- 교환법칙 성립: $\frac{2}{5} \times 3 = 3 \times \frac{2}{5}$<br>- 약분 전략: 곱셈 전 약분 vs 곱셈 후 약분 |
| **주요 오개념** | - **분모에도 곱하기**: $\frac{2}{5} \times 3 = \frac{6}{15}$ (분자만 곱해야)<br>- **자연수를 분수로 변환 못함**: $3 = \frac{3}{1}$ 인식 못함<br>- 곱셈 결과가 작아질 수 있음을 이해 못함 (진분수 × 진분수) |
| **문항 설계 지침** | ✓ 동수누가 시각 모델: 피자 조각 반복<br>✓ 교환법칙 확인: 두 순서 모두 계산하여 같음 확인<br>✓ 약분 전략 비교: 효율성 논의<br>✓ 분수를 가분수로 나타내기 연습 |

#### 세부 개념 02: 분수×분수와 대분수 곱셈 (`frac-mul-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **(분수) × (분수)**: 분자끼리, 분모끼리 곱함<br>  $\frac{2}{3} \times \frac{4}{5} = \frac{2 \times 4}{3 \times 5} = \frac{8}{15}$<br>- **넓이 모델(Area Model)**: '전체의 부분의 부분'<br>  가로·세로 분할 → 겹치는 부분 = 결과<br>- **연산자 의미 확장**: 분수 곱하기 = 대상 축소(Scaling down)<br>- **대분수 곱셈**: **반드시 가분수로 변환** 후 계산<br>- 대각선 약분: 곱셈 중 교차 약분으로 효율화 |
| **주요 오개념** | - **"곱하면 무조건 커진다"**: 진분수 곱셈 결과 작아지는 것 불신<br>- **대분수 곱셈 오류** (매우 빈출): 자연수끼리·분수끼리 따로 곱함<br>  $1\frac{1}{2} \times 1\frac{1}{2}$에서 $1 \times 1 = 1$, $\frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$ → 오답: $1\frac{1}{4}$<br>  정답: $\frac{3}{2} \times \frac{3}{2} = \frac{9}{4} = 2\frac{1}{4}$<br>- 넓이 모델 이해 부족 |
| **문항 설계 지침** | ✓ 넓이 모델로 $\frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$ 시각적 증명<br>✓ 대분수 곱셈 → **가분수 변환 필수** 강조<br>✓ "곱하면 커진다" 반례: $\frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$ (더 작음)<br>✓ 대각선 약분 전략 소개 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 분수×자연수 | 분자에 자연수 곱하기 | 하 |
| 분수×분수 | 분자끼리, 분모끼리 곱하기 | 중 |
| 대분수 곱셈 | 가분수로 변환 후 계산 | 중상 |
| 넓이 모델 | 시각적으로 곱셈 이해 | 상 |
| 대각선 약분 | 교차 약분으로 효율화 | 상 |

### 3단원: 합동과 대칭 - 공간 감각과 변환

> 도형의 이동(Rigid Motion)을 기반으로 합동과 대칭의 성질을 탐구. 중학교 삼각형 합동 조건(SSS, SAS, ASA)의 기초.

#### 세부 개념 01: 도형의 합동과 대응 (`congru-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **합동**: 모양과 크기가 같아 완전히 포개어지는 도형<br>- **대응점, 대응변, 대응각**: 합동 도형에서 겹치는 요소 (길이·크기 같음)<br>- **대응 관계 표기**: ∆ABC ≡ ∆DEF → A↔D, B↔E, C↔F<br>- 뒤집거나 회전해도 합동 관계 유지 |

| **문항 설계 지침** | ✓ "선대칭도형의 대칭축을 잘못 그린 것 모두 고르시오" (평행사변형 함정)<br>✓ 도형 일부만 주고 대칭 성질로 완성하기<br>✓ 대칭축·대칭 중심 개수 세기 (정다각형)<br>✓ 공학 도구(알지오매스): 도형 회전·뒤집기 시뮬레이션 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 합동 판별 | 합동인 도형 찾기 | 하 |
| 대응 요소 | 대응변·대응각 찾기 | 중 |
| 대칭축 찾기 | 선대칭도형의 대칭축 그리기 | 중 |
| 평행사변형 함정 | 평행사변형 대칭축 오류 | 중상 |

### 4단원: 소수의 곱셈 - 알고리즘의 형식화

> 분수의 곱셈 원리를 바탕으로 소수의 연산 체계를 완성.

#### 세부 개념 01: 소수×자연수와 소수점 이동 (`dec-mul-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **(소수) × (자연수)**: 세로셈으로 계산, 소수점 위치 유지<br>  $2.3 \times 4 = 9.2$<br>- **소수점 이동 규칙**: 곱하는 수가 10배 → 소수점 오른쪽으로 1칸<br>  $0.5 \times 10 = 5$<br>- 분수 변환으로 원리 확인: $2.3 = \frac{23}{10}$ |
| **주요 오개념** | - **소수점 위치 오류**: 계산 후 소수점 찍는 위치 혼동<br>- **0 처리 오류**: $0.5 \times 2 = 0.10$에서 0을 먼저 지우고 소수점 찍으면 $0.01$ 오류<br>- 어림 부재: 결과 범위 예측 못함 |
| **문항 설계 지침** | ✓ 소수점 이동 규칙 발견: ×10, ×100 패턴<br>✓ 분수로 변환하여 원리 확인<br>✓ 어림으로 결과 범위 예측 습관화<br>✓ 세로셈 정렬 확인 |

#### 세부 개념 02: 소수×소수와 소수점 위치 (`dec-mul-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **(소수) × (소수)**: 자연수처럼 계산 후, **두 수의 소수점 아래 자릿수 합**만큼 소수점 찍기<br>  $0.5 \times 0.4 = 0.20$ (1+1=2자리)<br>- **순서**: 소수점 먼저 찍고, 그 뒤 맨 끝 0 지우기<br>- 소수를 분수로 변환하면 원리 명확: $\frac{5}{10} \times \frac{4}{10} = \frac{20}{100} = 0.20$ |
| **주요 오개념** | - **0 처리 순서 오류**: $0.5 \times 0.4$에서 $5 \times 4 = 20$ → 0 먼저 지우면 $2$ → 소수점 찍으면 $0.02$ (틀림!)<br>  정답: $20$ → 소수점 먼저 찍어 $0.20$ → 맨 끝 0 지워 $0.2$<br>- **어림 부재**: $4.1 \times 5.2 \approx 20$인데 $213.2$ 답하고 이상함 못 느낌<br>- 자릿수 합 계산 실수 |
| **문항 설계 지침** | ✓ 계산 전 어림(Estimation)으로 결과 범위 예측<br>✓ 소수점 위치 확인 문제<br>✓ 소수 → 분수 변환으로 원리 확인<br>✓ 0 처리 순서 명확히 제시 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 소수×자연수 | 세로셈 소수점 위치 | 하 |
| 소수점 이동 | ×10, ×100 규칙 | 중 |
| 소수×소수 | 자릿수 합으로 소수점 위치 | 중상 |
| 어림 확인 | 결과 범위 예측 | 상 |

### 6단원: 평균과 가능성 - 불확실성의 수학

> 통계적 대표값과 확률의 기초. 2022 개정에서 중요도가 높아진 영역.

#### 세부 개념 01: 평균(Mean)의 개념 (`avg-01`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **계산적 정의**: (자료의 값을 모두 더함) ÷ (자료의 수)<br>- **개념적 의미(Leveling out)**: 자료를 평평하게 고르는 것<br>  → "튀어나온 곳 깎고 부족한 곳 채우는" 평탄화<br>- **블록 모델**: 들쑥날쑥한 높이의 블록을 평평하게 맞추기 |
| **주요 오개념** | - **평균의 대표성 오해**: 평균 80점이면 모든 과목 80점 근처라 착각<br>- **극단값(Outlier) 영향 무시**: 0점이 평균에 미치는 영향 이해 못함<br>- **0의 처리**: 0점인 과목도 **자료의 수에 포함** 해야 함<br>- 평균값이 자료 내에 반드시 존재해야 한다고 착각 (예: 2.5명) |
| **문항 설계 지침** | ✓ 극단값이 평균에 미치는 영향 문제<br>✓ 0이 포함된 자료의 평균 계산<br>✓ 블록 모델로 평탄화 시각화<br>✓ 평균의 함정: "평균 연봉" 오용 사례 토론 |

#### 세부 개념 02: 가능성의 표현 (`avg-02`)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 사건이 일어날 가능성을 **5단계 척도**로 수치화:<br>  불가능하다(0) - 아닐 것 같다 - 반반이다($\frac{1}{2}$) - 일 것 같다 - 확실하다(1)<br>- 수직선에 가능성 위치 표시<br>- 고등학교 확률과 통계의 기초 개념 |
| **주요 오개념** | - **가능성의 주관적 해석**: 수학적 확률이 아닌 개인 믿음으로 판단<br>- **0과 1 사이 수치 대응 거부**: "반반이다"를 $\frac{1}{2}$로 표현 못함<br>- 불가능(0)과 아닐 것 같다 구분 못함 |
| **문항 설계 지침** | ✓ 회전판 돌리기 실험 → 가능성 수치화<br>✓ 상황별 가능성 판단: "내일 해가 뜬다" (확실), "동전 앞면" (반반)<br>✓ 수직선에 가능성 위치시키기<br>✓ 가능성 크기 순서 배열 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 평균 계산 | 자료의 합 ÷ 개수 | 하 |

## 문제 개발 전략

### 1. "왜?"를 묻는 문항 비중 확대 (정당화 중심)

| 나쁜 예 | 좋은 예 |
|---------|---------|
| "$\frac{1}{2} + \frac{1}{3}$을 계산하시오." | "$\frac{1}{2} + \frac{1}{3} = \frac{2}{5}$가 틀린 이유를 그림으로 설명하시오." |
| "평행사변형의 넓이를 구하시오." | "왜 평행사변형 넓이는 직사각형과 같은 방법으로 구할 수 있는가?" |
| "대분수를 곱하시오." | "왜 대분수 곱셈은 가분수로 변환해야 하는가?" |

### 2. 분수 오개념 교정 핵심

| 오개념 | 교정 도구 |
|--------|-----------|
| $\frac{1}{2}+\frac{1}{3}=\frac{2}{5}$ | 피자 조각 그림, 수직선 모델, "단위 통일" 비유 |
| 받아내림 시 10 더함 | 단계별 과정 제시 → 오류 찾기 |
| 대분수 곱셈 따로 계산 | 가분수 변환 후 계산 강제, 분배법칙과 효율성 비교 |
| 통분 시 분자 누락 | 영역 모델 재분할, 시각적 검증 |

### 3. 도형 넓이 공식 유도 과정 강조

| 도형 | 변형 유형 | 유도 과정 |
|------|-----------|-----------|
| 평행사변형 | 등적 변형 | 잘라 붙여 직사각형 → 밑변 × 높이 |
| 삼각형 | 배적 변형 | 2개 붙여 평행사변형 → ÷2 |
| 마름모 | 등적 변형 | 대각선 잘라 직사각형 → ÷2 |
| 사다리꼴 | 배적 변형 | 뒤집어 붙여 평행사변형 → ÷2 |

> **÷2 누락 방지**: 공식 암기 아닌 **유도 과정 경험** 필수

### 4. 역량 기반 문항 설계

| 핵심 역량 | 문항 설계 포인트 | 예시 |
|-----------|------------------|------|
| **문제해결** | 정형화되지 않은 문제 | 숫자 카드로 목표수 만들기 |
| **추론** | 논리적 정당화 | "왜 삼각형 넓이는 ÷2인가?" |
| **의사소통** | 표현 변환 (말↔식↔그림) | 대응 관계를 말, 표, 식으로 |
| **연결** | 단원 간 연결 | 약수→약분, 직사각형→삼각형 넓이 |
| **정보처리** | 데이터 해석 | 평균 오용 상황 토론 |

>
> 자연수 세계에서 **유리수 세계**로 수 감각 확장
> 직관적 인식에서 **논리적 규명**으로 사고 전환
> '수학의 벽'을 넘어 '도약의 시기'로

### 학기별 핵심 키워드

| 학기 | 핵심 |
|------|------|
| **1학기** | 연산 위계 확립, 약수·배수(정수론 기초), **함수적 사고 태동**, **분수 연산 완성**, 넓이 공식 유도(정당화) |
| **2학기** | 수의 범위·어림(부등식 기초), 분수·소수 곱셈(승법적 사고), **공간 지각력**, **통계적 소양** |

### 2022 개정 핵심 변화 3가지

1. **5대 핵심 역량 재구조화** → 문제해결·추론·의사소통·연결·정보처리
2. **정당화(Justification) 강화** → 공식 암기 지양, 유도 과정 경험 필수
3. **디지털 소양 전면 배치** → 알지오매스 등 공학 도구의 인지적 도구화

### 5학년 분수 지도 3대 핵심

1. **통분의 필요성**: 분모가 다르면 '단위'가 다름 → 직접 연산 불가
2. **시각적 모델 활용**: 피자 조각, 수직선, 넓이 모델로 오개념 교정
3. **대분수 곱셈 주의**: 덧셈 습관 전이 방지, **반드시 가분수 변환**
