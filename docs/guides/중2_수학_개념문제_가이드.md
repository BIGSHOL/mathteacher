## 📗 1학기 단원별 정리

### 1단원: 유리수와 순환소수

#### `concept-m2-rational-01` 유한소수 판별

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **유리수 정의**: $\frac{a}{b}$ ($a, b$는 정수, $b \neq 0$) 꼴로 나타낼 수 있는 수<br>• **유한소수 조건**: 기약분수의 분모 소인수가 **2 또는 5뿐**일 때<br>• **유한소수 판별 3단계**: ① 약분(기약분수) → ② 분모 소인수분해 → ③ 2와 5만 있는지 판단 |
| **출제 유형** | ✓ 주어진 분수가 유한소수인지 판별하기<br>✓ 유한소수가 되도록 하는 자연수 $a$의 값 구하기<br>✓ 약분 → 소인수분해 → 판단의 3단계 알고리즘 적용 |
| **주요 오개념** | • **약분 없이 바로 소인수분해**: $\frac{6}{15}$에서 15를 바로 소인수분해 → 분모에 3이 있어 무한소수라고 판단<br>  → 반드시 기약분수 $\frac{2}{5}$로 만든 후 판별<br>• **분모에 2 또는 5 '포함'과 '뿐'의 혼동**: 분모에 2가 있으면 유한소수라고 착각 (2 외에 다른 소인수가 있어도 유한소수라고 판단) |

---

#### `concept-m2-rational-02` 순환소수·분수 변환

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **순환소수**: 무한소수이지만 순환마디가 있어 분수로 표현 가능 → **유리수**<br>• **순환마디 표기**: 양 끝에 점 ($0.\dot{1}4\dot{2}$)<br>• **순환소수 → 분수 변환법**:<br>  - 순환마디 1자리: $x = 0.\dot{3}$ → $10x - x = 9x = 3$ → $x = \frac{1}{3}$<br>  - 순환마디 2자리: $x = 0.\dot{2}\dot{7}$ → $100x - x = 99x = 27$ → $x = \frac{3}{11}$<br>• **$0.999... = 1$ 증명**: $x = 0.999...$, $10x = 9.999...$, $9x = 9$, $x = 1$ |
| **출제 유형** | ✓ 순환소수의 분수 표현: 공식 암기 지양, **등식의 성질 이용 과정** 경험<br>✓ 계산기로 $\frac{1}{7}, \frac{2}{7}, \frac{3}{7}$ 등 순환마디 패턴 발견<br>✓ $0.999... = 1$ 증명 서술 |
| **주요 오개념** | • **$0.999... < 1$ 직관적 저항**: "한없이 가까워질 뿐, 결코 1이 될 수 없다"<br>  → 대수적 증명으로 논리적 확인<br>• **순환마디 결정 오류**: $0.297297...$에서 순환마디를 972나 729로 착각<br>  → 소수점 이후 처음 반복 시작 위치를 정확히 찾기<br>• **$\pi$와 순환소수 혼동**: "무한히 계속되는 소수 = 순환소수"라는 잘못된 일반화<br>  → $\pi$는 순환마디가 없는 무한소수(무리수) |

### 2단원: 식의 계산

#### `concept-m2-expr-01` 지수법칙

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **지수법칙 3가지**:<br>  - $a^m \times a^n = a^{m+n}$ (같은 밑의 곱 → 지수의 합)<br>  - $(a^m)^n = a^{mn}$ (거듭제곱의 거듭제곱 → 지수의 곱)<br>  - $(ab)^n = a^n \cdot b^n$ (곱의 거듭제곱 → 각각 거듭제곱)<br>• **나눗셈**: $a^m \div a^n$: $m>n$이면 $a^{m-n}$, $m=n$이면 $1$, $m<n$이면 $\frac{1}{a^{n-m}}$<br>※ 중학교에서는 **음의 지수, 0지수 미도입** → 경우의 수 분류 필수 |
| **출제 유형** | ✓ 지수 정의로 돌아가 시각적 확인: $a^3 \times a^2 = (a \cdot a \cdot a) \times (a \cdot a)$<br>✓ 특정 수($x=1, y=2$) 대입하여 **검산 습관**<br>✓ 지수법칙 혼합 계산 |
| **주요 오개념** | • **연산 기호와 지수법칙 불일치**:<br>  - $a^3 \times a^2 = a^6$ (곱셈 기호에 이끌려 지수도 곱함)<br>  - $(a^3)^2 = a^5$ (거듭제곱인데 지수끼리 더함)<br>• **나눗셈 혼동**: $a^2 \div a^5 = a^3$이라고 답함<br>  → 분수 꼴로 풀어 약분: $\frac{a \cdot a}{a \cdot a \cdot a \cdot a \cdot a} = \frac{1}{a^3}$<br>• **계수 거듭제곱 누락**: $(2x^2y)^3 = 2x^6y^3$ (계수 2를 거듭제곱 안 함)<br>  → 올바른 답: $(2x^2y)^3 = 8x^6y^3$ |

| 구분 | 내용 |

#### 3-1. 일차부등식

##### `concept-m2-ineq-01` 부등식 성질·풀이

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **부등식**: 부등호($<, >, \le, \ge$)로 대소 관계 표현<br>• **부등식의 성질**:<br>  - 양변에 같은 수를 더하거나 빼도 부등호 방향 유지<br>  - 양변에 같은 **양수**를 곱하거나 나눠도 부등호 방향 유지<br>  - 양변에 같은 **음수**를 곱하거나 나누면 **부등호 방향 반전**<br>• **수직선 표현**: ○(미포함, $<, >$), ●(포함, $\le, \ge$)<br>• **복합 부등식**: $a < x \le b$ 형태 |

##### `concept-m2-simul-02` 연립일차방정식 해의 해석

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **해의 기하학적 의미**: 대수적 해 $(x, y)$ = 두 직선의 **교점**<br>• **해의 특수성**:<br>  - 해가 1개 (일반): 두 직선이 한 점에서 만남<br>  - 해가 없음 (**불능**): 두 직선이 **평행** (기울기 같고, y절편 다름)<br>  - 해가 무수히 많음 (**부정**): 두 직선이 **일치** (기울기·y절편 모두 같음) |
| **출제 유형** | ✓ 해의 개수 판별 문제 (불능/부정/일반)<br>✓ 그래프와 연립방정식 해의 관계<br>✓ 미정계수를 포함한 연립방정식의 해의 조건 |
| **주요 오개념** | • **불능/부정 혼동**: "해가 없다"와 "해가 무수히 많다"를 구분 못함<br>  → 불능 = 평행(만날 수 없음), 부정 = 일치(모든 점이 해)<br>• **대수적 해 ↔ 그래프 교점 분리**: 연립방정식의 해가 두 직선의 교점이라는 기하학적 의미를 연결 못함|

### 4단원: 일차함수

#### `concept-m2-linfn-01` 일차함수 그래프

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **일차함수 정의**: $y = ax + b$ ($a \neq 0$)<br>• **기울기($a$)**: $\frac{\Delta y}{\Delta x}$ = 변화율, 그래프의 가파른 정도와 방향<br>  - $a > 0$: 우상향, $a < 0$: 우하향<br>  - $|a|$가 클수록 가파름<br>• **절편**: $x$절편($y=0$일 때 $x$값), $y$절편($x=0$일 때 $y$값, 즉 $b$)<br>• **두 점을 지나는 직선의 방정식**: 기울기 $= \frac{y_2 - y_1}{x_2 - x_1}$ 구한 후 한 점 대입 |
| **출제 유형** | ✓ **공학 도구 필수**: 슬라이더로 $a, b$ 변화 → 그래프 회전/평행이동 동적 관찰<br>✓ 기울기의 부호와 절댓값에 따른 그래프 변화 설명<br>✓ 두 점의 좌표로 직선의 방정식 구하기<br>✓ 상수함수 $y = 3$과 $x = 2$의 차이 |
| **주요 오개념** | • **기울기 크기 비교 혼동**: $-2$와 $-5$ 중 어느 그래프가 더 가파른지?<br>  → "가파름" = 절댓값 $|a|$, 부호 = 방향(우상향/우하향)<br>• **절편과 좌표 혼동**: $x$절편 3을 좌표평면에 찍을 때 엉뚱한 곳에 표시<br>  → 절편은 '값'이자 그래프에서는 '위치' (x절편 3 = 점 $(3, 0)$) |

#### `concept-m2-linfn-02` 일차함수와 방정식 관계

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **일차함수 ↔ 일차방정식**: $ax + by + c = 0$의 그래프는 직선<br>  → 방정식의 해 = 직선 위의 점<br>• **$x = k$ (세로선)**: 함수가 **아님** (하나의 $x$에 무한히 많은 $y$ 대응)<br>• **$y = k$ (가로선)**: 일차함수가 아닌 **상수함수**<br>• **연립방정식과 그래프**: 두 일차함수의 교점 = 연립방정식의 해 |
| **출제 유형** | ✓ 일차방정식의 그래프 그리기<br>✓ 두 일차함수의 교점을 연립방정식으로 구하기<br>✓ 실생활 데이터 산점도 → 추세선을 일차함수로 모델링 |
| **주요 오개념** | • **$x = 2$를 함수로 착각**: 함수의 정의(모든 $x$가 하나의 $y$에 대응)와 연결하여 **함수 아님** 설명<br>• **그래프의 교점 의미 불이해**: 두 직선의 교점 좌표가 연립방정식의 해임을 모름<br>• **일차함수와 일차방정식의 관계 혼동**: $y = ax + b$와 $ax + by + c = 0$의 관계를 정리하지 못함 |

### 7단원: 피타고라스 정리

#### `concept-m2-pytha-01` 피타고라스 정리·증명

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **피타고라스 정리**: 직각삼각형에서 $a^2 + b^2 = c^2$ ($c$는 **빗변**)<br>• **피타고라스 정리의 역**: $a^2 + b^2 = c^2$이면 직각삼각형<br>  - $a^2 + b^2 > c^2$ → 예각삼각형<br>  - $a^2 + b^2 < c^2$ → 둔각삼각형<br>• **피타고라스 수**: 3-4-5, 5-12-13, 8-15-17 등<br>※ 무리수 학습 전이므로 피타고라스 수 위주 출제 |
| **출제 유형** | ✓ 유클리드 증명법(풍차 모양): 합동 → 넓이 이동 단계별 서술<br>✓ 피타고라스 정리의 역으로 직각삼각형 판별<br>✓ 피타고라스 수 활용 문제 |
| **주요 오개념** | • **빗변 아닌 변을 $c$에 대입**: $a^2 + b^2 = c^2$에서 **가장 긴 변(빗변)**을 $c$로 설정해야 함<br>  → 반드시 빗변(직각의 대변, 가장 긴 변)을 확인하고 $c$에 대입<br>• **역 적용 시 가장 긴 변 미확인**: 예각/둔각 판별 시 **가장 긴 변을 $c$로 설정** 안 함 |

#### `concept-m2-pytha-02` 피타고라스 정리 활용·무게중심

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **좌표평면 두 점 사이 거리**: $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ (피타고라스 정리 응용)<br>• **특수 직각삼각형**:<br>  - 직각이등변삼각형: $1:1:\sqrt{2}$<br>  - 정삼각형의 반: $1:\sqrt{3}:2$<br>• **삼각형의 무게중심**: 세 중선의 교점, 중선을 꼭짓점에서 **2:1**로 내분<br>• **무게중심의 넓이 등분**: 6개 작은 삼각형의 넓이가 모두 같음 |

### 8단원: 확률

#### `concept-m2-prob-01` 경우의 수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **합의 법칙**: A **또는** B가 일어나는 경우 = $m + n$ (동시에 일어나지 않을 때)<br>• **곱의 법칙**: A **그리고** B가 동시에 일어나는 경우 = $m \times n$<br>• **'동시에'의 수학적 의미**: 시간적 동시성이 아닌 **논리적 AND** (둘 다 일어남) |
| **출제 유형** | ✓ 합의 법칙 vs 곱의 법칙 구분 문제<br>✓ 경우의 수 구하기 (나열, 수형도, 표)<br>✓ **인지적 갈등 유발하는 반례** + 토론 수업 |
| **주요 오개념** | • **합/곱의 법칙 혼동**: "A 또는 B"에 곱의 법칙, "A 그리고 B"에 합의 법칙 적용<br>  → "또는" = 더하기, "그리고(동시에)" = 곱하기로 대응<br>• **'동시에'의 언어적 오해**: 시간적 동시성으로만 해석<br>  → 주사위를 순서대로 던져도 곱의 법칙 적용 |

---

#### `concept-m2-prob-02` 확률 계산·성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **수학적 확률**: $P(A) = \frac{n(A)}{n(S)} = \frac{사건의\ 경우의\ 수}{모든\ 경우의\ 수}$<br>  ※ 전제: 각 근원사건이 **동등하게(Equiprobable)** 일어남<br>• **확률의 성질**: $0 \le P(A) \le 1$<br>• **여사건의 확률**: $P(A') = 1 - P(A)$ ("~이 아닌" 경우를 구할 때 유용)<br>• **복원 추출 vs 비복원 추출**: 꺼낸 것을 다시 넣느냐 아니냐에 따라 확률 달라짐 |
| **출제 유형** | ✓ 근원사건 동등성 확인 문제<br>✓ 여사건 활용: "적어도 하나" 유형<br>✓ 복원/비복원 추출 구별 문제 |
| **주요 오개념** | • **근원사건의 동등성 무시**:<br>  흰 공 2개, 검은 공 1개 → 흰 공 나올 확률을 $\frac{1}{2}$로 착각<br>  → 겉보기 같아도 **흰1, 흰2**로 구분해야 동등성 보장<br>  → 윷놀이: 각 면이 나올 확률 $\neq \frac{1}{2}$<br>• **여사건 활용 미숙**: "적어도 1개" 문제를 직접 세려다 빠뜨림<br>  → $1 - P(하나도\ 없음)$으로 구하는 것이 효율적<br>• **복원/비복원 구별 실패**: 문제에서 "꺼낸 후 다시 넣지 않고"를 무시하고 복원 추출로 계산 |

## 💡 문제 개발 전략

### 1. 지수법칙 오류 교정 (`concept-m2-expr-01`)

| 오류 유형 | 교정 전략 |
|-----------|-----------|
| $a^3 \times a^2 = a^6$ | 지수 정의로 풀어서 확인: $(a \cdot a \cdot a) \times (a \cdot a) = a^5$ |
| $(a^3)^2 = a^5$ | 괄호 의미 강조: $a^3$을 2번 곱함 = $a^3 \times a^3 = a^6$ |
| $a^2 \div a^5 = a^3$ | 분수로 약분: $\frac{a \cdot a}{a \cdot a \cdot a \cdot a \cdot a} = \frac{1}{a^3}$ |
| $(2x^2y)^3 = 2x^6y^3$ | 계수도 거듭제곱: $(2x^2y)^3 = 2^3 \cdot (x^2)^3 \cdot y^3 = 8x^6y^3$ |

---

### 2. 정의와 성질 구분 훈련 (`concept-m2-tri-01`, `concept-m2-quad-01`)

| 도형 | 정의 | 성질 (증명 대상) |
|------|------|------------------|
| 이등변삼각형 | 두 변의 길이가 같음 | 두 밑각이 같음 |
| 직사각형 | 네 각이 모두 직각인 평행사변형 | 대각선의 길이가 같음 |
| 마름모 | 네 변의 길이가 모두 같은 평행사변형 | 대각선이 서로 수직이등분 |
| 평행사변형 | 두 쌍의 대변이 평행 | 대각선이 서로를 이등분 |

> "정의는 증명 없이 사용하는 전제, 성질은 증명해야 하는 대상"

---

### 3. 확률 오개념 교정: 반례 활용 (`concept-m2-prob-01`, `concept-m2-prob-02`)

| 오개념 상황 | 반례 |
|-------------|------|
| "흰 공 나올 확률 = $\frac{1}{2}$" (종류만 셈) | 흰 공 2개, 검은 공 1개 → 정답 $\frac{2}{3}$ |
| "윷놀이 각 면 확률 = $\frac{1}{2}$" | 윷의 구조상 평평한 면과 둥근 면의 확률 다름 |
| "'동시에' = 시간적 동시" | 주사위를 순서대로 던져도 곱의 법칙 적용 |
| "적어도 1개를 직접 세기" | 여사건 활용: $1 - P(하나도\ 없음)$이 효율적 |

---

### 4. 부등식·연립방정식 오개념 교정 (`concept-m2-ineq-01`, `concept-m2-simul-02`)

| 오개념 상황 | 교정 전략 |
|-------------|-----------|
| $-2x > 4$ → $x > -2$ (부등호 방향 유지) | 음수 나눗셈 시 반드시 부등호 반전: $x < -2$ |
| 불능과 부정 혼동 | 불능 = 평행(만남 불가), 부정 = 일치(모든 점이 해) |
| 대수적 해와 그래프 교점 분리 | 연립방정식의 해 $(x, y)$ = 두 직선의 교점 시각화 |

---

### 5. 닮음비·넓이비·부피비 오류 교정 (`concept-m2-simil-02`)

| 닮음비 | 둘레비 | 넓이비 | 부피비 |
|--------|--------|--------|--------|
| $m:n$ | $m:n$ | $m^2:n^2$ | $m^3:n^3$ |
| $2:3$ | $2:3$ | $4:9$ | $8:27$ |
| $1:2$ | $1:2$ | $1:4$ | $1:8$ |

> 핵심: 닮음비의 **제곱**이 넓이비, **세제곱**이 부피비

### 6. 공학 도구 활용 (2022 개정 필수)

| 단원 | 개념 ID | 활용 방법 |
|------|---------|-----------|
| 일차함수 | `linfn-01` | 슬라이더로 $a, b$ 변화 → 그래프 동적 관찰 |
| 삼각형의 성질 | `tri-02` | 공학 도구로 도형 변형 → 외심/내심 위치 변화 관찰 |
| 사각형의 성질 | `quad-01` | 평행사변형 꼭짓점 드래그 → 불변 성질 관찰 |

## ✅ 핵심 원칙

> **중2 수학의 본질**
>
> **대수**: 유리수 체계 완성, 지수법칙과 다항식의 **구조적 사고**
> **기하**: 직관 기하 → **논증 기하** 전환, 정의와 성질 구분
> **확률**: 결정론적 세계 → **비결정론적 세계**로 사고 전환
