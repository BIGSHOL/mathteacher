# 초등학교 3학년 수학 개념 문제 개발 가이드
> 2022 개정 교육과정 기반

## 📌 3학년 수학의 특징

- **위상**: 자연수 연산의 완성 + 추상적 수 개념(분수, 소수) 도입의 교차점
- **인지 발달**: 구체적 조작기 중반부
- **핵심 전환**: 직관적·구체적 활동 → 형식적 알고리즘과 엄밀한 정의로 전환
- **주의**: 분수 도입은 수포자 발생의 분수령이 되는 핵심 단원

> 💡 3학년 개념 문항은 단순 '계산 수행 능력'을 넘어, **'개념적 이해(Conceptual Understanding)'** 평가로 전환되어야 함

## 📗 1학기 단원별 정리

### 1단원: 덧셈과 뺄셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 세 자리 수의 덧셈과 뺄셈<br>• **받아올림**: 각 자리의 합이 10 이상일 때 상위 자릿수로 1 올림<br>• **받아내림(재구조화)**: 상위 자리에서 1을 빌려 10으로 바꾸어 계산<br>• 계산 결과의 어림 (수 감각) |
| **주요 오개념** | • **⭐ 감산 방향성 오류**: $423-157$에서 일의 자리 $3-7$이 안 되니까 $7-3=4$로 계산<br>  (원인: '큰 수-작은 수' 규칙을 위치 무관하게 적용)<br>• 받아올림 누락: 십의 자리 합이 12인데 백의 자리로 1 안 올림<br>• 자릿수 정렬 오류: 가로셈→세로셈 변환 시 왼쪽 정렬 |
| **출제 포인트** | ✓ 받아내림의 필요성 인식 문항<br>  (예: "십의 자리에서 가져온 수는 실제로 얼마를 의미하는가?" → 답: 10)<br>✓ **오류 찾기(Error Analysis)** 유형 적극 활용<br>✓ 중간 계산 과정 빈칸으로 받아올림 표기 강제 |

### 2단원: 평면도형

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **선분**: 두 점을 잇는 가장 짧은 선 (길이 유한)<br>• **반직선**: 한 점에서 시작해 한쪽으로 끝없이 뻗은 선 (시작점+방향)<br>• **직선**: 양쪽으로 끝없이 뻗은 선<br>• **각**: 한 점에서 그은 두 반직선으로 이루어진 도형 (꼭짓점, 변)<br>• **직각**: 종이를 반듯하게 두 번 접었을 때 생기는 각 |
| **주요 오개념** | • **반직선 동일성 오해**: 반직선 AB = 반직선 BA라고 착각<br>  (선분은 양 끝점 같으면 같지만, 반직선은 시작점+방향 모두 같아야 함)<br>• **직각의 방향성 고착**: 수평/수직으로 놓인 직각만 직각으로 인식<br>  (기울어진 직각삼각형에서 직각 못 찾음)<br>• **변 길이 = 각 크기 혼동**: 변이 길면 각도 크다고 착각 |
| **출제 포인트** | ✓ 반직선 AB와 BA를 각각 다른 색으로 칠하기<br>✓ 다양한 각도로 회전된 도형에서 직각 모두 찾기<br>✓ 변의 길이는 다르지만 각의 크기는 같은 두 각 비교 |

---

### 3단원: 나눗셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **등분제**: 전체를 똑같은 수의 묶음으로 나눌 때 한 묶음의 개수<br>  (예: 사탕 12개를 3명에게 똑같이 나누기)<br>• **포함제**: 전체를 한 묶음씩 덜어낼 때 묶음의 개수<br>  (예: 사탕 12개를 3개씩 나누기)<br>• **곱셈과의 관계**: $12 \div 3 = 4$ ↔ $3 \times 4 = 12$ |
| **주요 오개념** | • **⭐ 교환법칙 적용 오류**: $12 \div 4$와 $4 \div 12$를 같다고 착각<br>  (덧셈·곱셈과 달리 나눗셈은 교환법칙 성립 안 함)<br>• **몫의 의미 혼동**: 몫 '4'가 '사람 수'인지 '사탕 개수'인지 단위 헷갈림<br>  (등분제/포함제 상황 구별 못함) |
| **출제 포인트** | ✓ 구체물 그림 제시 → 상황에 맞는 나눗셈식 고르기<br>✓ "사과 4개를 12명에게 나누는 상황" 가능 여부 묻기<br>✓ **관계 짓기 문항**: "그림을 보고 곱셈식 2개, 나눗셈식 2개 만드시오" |

---

### 4단원: 곱셈

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • (몇십몇) × (몇) 계산 원리<br>• **분배법칙의 기초**: $23 \times 2 = 20 \times 2 + 3 \times 2$<br>• 올림이 있는 경우와 없는 경우<br>• 곱셈 = 동수누가(Repeated Addition)의 효율적 표현 |
| **주요 오개념** | • **자릿수 혼동**: $23 \times 4$에서 $3 \times 4 = 12$, $2 \times 4 = 8$을 나열해 812로 표기<br>  (십의 자리 '2'가 실제로는 '20'임을 간과) |
| **출제 포인트** | ✓ 모눈종이(Area Model) 활용<br>✓ $20 \times 4$ 영역과 $3 \times 4$ 영역 구분하여 합산하는 과정 시각화 |

---

### 5단원: 길이와 시간

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **길이 단위**: 1mm, 1cm, 1m, 1km 도입 및 변환<br>• **시간**: '시각'과 '시간(양)' 구별, 초 단위까지 읽기<br>• 시간의 덧셈과 뺄셈 |
| **주요 오개념** | • **⭐ 60진법과 10진법 충돌**<br>  1시간 40분 + 30분 = 1시간 70분 = 1.7시간(?)으로 계산<br>  (60분이 되면 1시간으로 받아올림해야 함)<br>• **자 눈금 읽기 오류**: 항상 0에서 시작해야 한다고 착각<br>  자의 중간(3cm)에서 시작하면 끝 눈금을 길이로 오인 |
| **출제 포인트** | ✓ **부러진 자** 문제: 0 눈금 없는 자에서 물체 길이 구하기<br>  ($끝 눈금 - 시작 눈금$ 원리 이해 확인)<br>✓ 시간 덧셈에서 받아올림 기준(60) 강조 문항 |

### 6단원: 분수와 소수 ⭐ (최대 난관)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **분수의 정의**: 전체를 '똑같이' 나눈 것 중의 일부분<br>• **단위분수**: 분자가 1인 분수 ($\frac{1}{2}, \frac{1}{3}, \frac{1}{4}$...)<br>• **소수 도입**: $\frac{1}{10} = 0.1$<br>• 크기 비교: 분모가 같은 분수, 단위분수, 소수 |
| **주요 오개념** | • **⭐ '똑같이 나누기' 실패**: 크기가 다른 4조각 중 1개를 색칠하고 $\frac{1}{4}$이라 함<br>  (이산량적 개수 세기 경험이 연속량 면적에 간섭)<br>• **⭐ 단위분수의 역설**: $\frac{1}{3} > \frac{1}{2}$라고 착각<br>  (원인: 자연수 지식의 간섭 - 3이 2보다 크니까)<br>• **소수 0.1의 실체성 부재**: 0.1을 0보다 작은 수로 오해 |
| **출제 포인트** | ✓ 등분된 것과 안 된 것 섞어서 "$\frac{1}{4}$을 나타내는 그림 모두 고르기"<br>✓ 피자 2판: 2명 나눠 먹기 vs 3명 나눠 먹기 비교<br>✓ **수직선** 위에서 0과 1 사이를 10칸으로 나누고 0.1, 0.5 위치 찍기 |

## 📘 2학기 단원별 정리

### 1단원: 곱셈 (알고리즘의 완성)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • (세 자리 수) × (한 자리 수)<br>• (두 자리 수) × (두 자리 수): 4번의 부분 곱(Partial Products) 합산<br>  → 초등 연산 중 가장 복잡한 알고리즘 |
| **주요 오개념** | • **자릿수 맞춤 오류 심화**: $34 \times 12$에서<br>  $34 \times 10 = 340$을 써야 하는데 '0' 생략하고 34로 적음<br>  → $34 \times 2 + 34 \times 1$로 잘못 계산 |
| **출제 포인트** | ✓ 계산 과정에서 0을 쓰도록 '보조선'이나 '칸' 제공<br>✓ 자릿수 개념 강화 문항 |

---

### 2단원: 나눗셈 (나머지의 발견)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **나머지가 있는 나눗셈**: 나누어떨어지지 않는 경우<br>• **검산**: $A \div B = Q ... R$ → $A = B \times Q + R$<br>• **제약 조건**: 나머지 < 나누는 수 ($R < B$) |
| **주요 오개념** | • **⭐ 나머지 크기 조건 위반**: $14 \div 3 = 3 ... 5$라고 씀<br>  (덜어낼 수 있을 때까지 덜어내야 함을 모름)<br>• **⭐ 문맥적 해석 실패**<br>  "14명을 3명씩 텐트 배정 → 텐트 몇 개?" $4...2$ → 답 4개(×)<br>  (남는 2명 위해 1개 더 필요 → 정답 5개, 올림 상황) |
| **출제 포인트** | ✓ 잘못된 계산 제시 → "나머지의 성질을 이용해 틀린 이유 설명하기"<br>✓ **올림 상황** vs **버림 상황** 구별 문항 세트 출제 |

---

### 3단원: 원

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **원의 중심**: 원 그릴 때 고정된 점<br>• **반지름**: 중심에서 원 위의 한 점까지 선분 (한 원에서 모두 같음)<br>• **지름**: 원 위의 두 점을 잇는 선분 중 **중심을 지나는** 선분<br>• 지름 = 반지름 × 2<br>• 컴퍼스 사용법 |
| **주요 오개념** | • **반지름 방향 가변성 오해**: 위쪽/옆쪽 반지름 길이가 다르다고 착각<br>  (시각적으로 길어 보이는 것을 더 길다고 판단)<br>• **지름 정의 오류**: 원 내부를 지나는 아무 선분(현)을 지름이라 부름 |
| **출제 포인트** | ✓ 찌그러진 원(타원) vs 바른 원: "컴퍼스로 그린 원은? 이유는?"<br>✓ 원 안에 여러 선분 그려놓고 지름인 것/아닌 것 분류<br>  (기준: **'원의 중심을 지나는가?'**) |

---

### 4단원: 분수 (분수의 다양한 형태)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **진분수**: 분자 < 분모<br>• **가분수**: 분자 ≥ 분모<br>• **대분수**: 자연수 + 진분수<br>• 대분수 ↔ 가분수 변환<br>• 분모가 같은 분수의 덧셈과 뺄셈: 분모 그대로, 분자끼리 연산 |
| **주요 오개념** | • **⭐ 분모 덧셈 오류**: $\frac{2}{7} + \frac{3}{7} = \frac{5}{14}$<br>  (원인: 자연수 덧셈 도식의 간섭 - 분모와 분자 각각 더함) |
| **출제 포인트** | ✓ 파이 차트 그림으로 $\frac{2}{7}$ + $\frac{3}{7}$ 시각적 증명<br>  → 조각의 크기(분모)는 불변, 개수(분자)만 증가 |

### 5단원: 들이와 무게

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **들이 단위**: L, mL (1L = 1000mL)<br>• **무게 단위**: kg, g (1kg = 1000g)<br>• 같은 단위끼리 덧셈과 뺄셈 |
| **주요 오개념** | • **부피 = 무게 혼동**: 부피가 크면 무조건 더 무겁다고 착각<br>  (예: 솜뭉치 vs 쇠구슬) |
| **출제 포인트** | ✓ 시각적 부피와 실제 무게가 상충하는 상황 제시<br>  (예: "스티로폼 상자 vs 작은 아령, 어느 것이 더 무거운가?") |

### 6단원: 자료의 정리 (그림그래프)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • 자료 수집 → 표 정리 → 그림그래프(Picture Graph) 표현<br>• **단위 설정**: 그림 하나가 나타내는 수 (10, 100 등) |
| **주요 오개념** | • **⭐ 상징의 수량화 실패**: '큰 스마일 = 10명'인데 그림 3개를 보고 "3명"이라 답함<br>  (범례 무시하고 그림 개수만 셈) |
| **출제 포인트** | ✓ 범례 다양하게 변형 (큰 그림=100, 작은 그림=10)<br>✓ 그림 개수 ≠ 실제 수량임을 계산하는 문제 |

## 💡 문제 개발 전략

### 1. 오류 찾기(Error Analysis) 문항 활용

> 가상의 학생이 푼 **틀린 풀이 과정** 제시
> - "왜 틀렸는지 설명하시오"
> - "어느 단계에서 처음 실수가 발생했는지 찾으시오"

**효과**: 메타인지 능력 평가에 탁월

### 2. 시각화 문항 적극 활용

| 개념 | 시각화 도구 |
|------|-------------|
| 곱셈 원리 | 모눈종이(Area Model) |
| 분수 개념 | 파이 차트, 막대 모델 |
| 소수 개념 | 수직선(Number Line) |
