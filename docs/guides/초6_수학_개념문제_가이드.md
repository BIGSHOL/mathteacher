## 초6 수학의 특징

- **위상**: 초등 수학의 완성 + 중등 수학(대수, 기하)으로의 결정적 도약점
- **인지 발달**: 구체적 조작기 → 형식적 조작기 과도기
- **교육 방향**: 계산 숙달 → **'개념의 연결(Connection)'**과 **'수학적 모델링(Modeling)'** 능력 함양
- **2022 개정 핵심 변화**:
  - '규칙성' 영역 → **'변화와 관계'** 영역으로 개편 (함수적 사고 기초)
  - 원주율($\pi$) **3.14로 고정** (3, 3.1 등 복수 조건 사용 지양)
  - 미지수 표기: '$x$' 대신 **'$\square$' 또는 '어떤 수'** 사용 (단, 중등 방정식과 매끄럽게 연결되도록 설계)
  - **디지털 소양** 강화: 공학 도구 활용 권장

> 개념 문제는 단순 연산 숙달이 아닌 **'이해하고 설명하기'**, **'추론하기'**로 평가 주안점 이동

### 1단원: 분수의 나눗셈 (자연수÷자연수, 분수÷자연수)

> 자연수의 나눗셈 원리를 분수로 확장하며, 알고리즘 암기보다 '왜' 그렇게 계산되는지 시각적 모델로 이해하는 것이 우선.

#### 세부 개념 A: 자연수 ÷ 자연수 = 분수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 나눗셈의 몫을 분수로 표현: $a \div b = \frac{a}{b}$<br>- 나눗셈 기호(÷)와 분수 막대의 동일한 의미<br>- 나눗셈을 '등분할(Partitive)' 상황으로 해석 |
| **주요 오개념** | - **"나눗셈 하면 숫자가 작아진다"** (자연수 경험칙에 갇힘)<br>- 교환법칙 오류: 작은 수÷큰 수 상황에서 순서 뒤바꿈 (예: 3L를 4명→$4 \div 3$으로 계산) |
| **출제 포인트** | - 숫자 크기만 보고 식 세우면 틀리는 문항 (예: "2m 끈을 5등분"→정답 $\frac{2}{5}$m)<br>- "2L의 우유를 3명이 나누어 마신다" vs "우유 2L는 3L의 몇 배인가?" 상황 구분 |

#### 세부 개념 B: 분수 ÷ 자연수

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - $\frac{4}{5} \div 2$: 분자가 자연수의 배수인 경우 → 분자끼리 나눗셈<br>- $\frac{3}{4} \div 2$: 배수가 아닌 경우 → 크기가 같은 분수로 변형하거나 $\times \frac{1}{자연수}$로 변환<br>- **알고리즘**: $(분수) \div (자연수) = \frac{a}{b} \div c = \frac{a}{b} \times \frac{1}{c}$ |
| **주요 오개념** | - 나누는 수가 정수일 때 곱하기 역수($\times \frac{1}{n}$)로 바뀌는 이유를 이해하지 못함<br>- 분모와 자연수를 곱해야 하는데 분자와 곱하는 역방향 오류 |
| **출제 포인트** | - 넓이 모델(직사각형)로 원리 설명하기<br>- 주어진 식을 그림으로 표현하고 서술하는 수행평가형 문항 |

### 2단원: 각기둥과 각뿔

> 입체도형의 구성 요소를 분석하고 분류하는 기하학적 사고를 기르는 단원.

#### 세부 개념 A: 분류와 구성 요소

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **분류 기준**: 밑면의 모양(삼각형, 사각형...)에 따라 이름 결정<br>- **각기둥**: 두 밑면이 합동이고 평행, 옆면이 모두 직사각형인 다면체<br>- **각뿔**: 밑면이 다각형이고 옆면이 모두 삼각형이며 뿔 꼭짓점을 갖는 다면체<br>- **각기둥 높이**: 두 밑면 사이의 거리<br>- **각뿔 높이**: 꼭짓점에서 밑면에 내린 수선의 길이 |
| **주요 오개념** | - **"밑면 = 바닥에 있는 면"** (가장 빈출 오개념! 옆으로 눕혀진 기둥의 밑면을 찾지 못함)<br>- 각뿔의 높이(수직 거리) ↔ 옆 모서리(모선) 길이 혼동 |
| **출제 포인트** | - 옆으로 누운 삼각기둥에서 밑면/높이 찾기<br>- 각뿔 내부 수직 막대 vs 옆면 타고 내려오는 실 비교 그림 |

#### 세부 개념 B: 구성 요소의 수와 전개도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **$n$각기둥**: 면 $n+2$, 모서리 $3n$, 꼭짓점 $2n$<br>- **$n$각뿔**: 면 $n+1$, 모서리 $2n$, 꼭짓점 $n+1$<br>- 전개도를 접었을 때 맞닿는 선분이나 점 찾기 (공간지각력) |
| **주요 오개념** | - 오일러 공식을 단순 암기만 하고 구성 요소를 직접 세지 못함<br>- 전개도에서 접었을 때 겹치는 부분 파악 실패 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 기본 분류 | 입체도형 이름 붙이기 | 하 |
| 구성 요소 세기 | 면/모서리/꼭짓점 수 구하기 | 중 |
| 조건 추론 | "모서리의 합이 36cm인 정사각기둥"에서 도형 유추 | 중상 |
| 전개도 | 접었을 때 맞닿는 선분/점 찾기 | 상 |

### 3단원: 소수의 나눗셈 (소수÷자연수)

#### 세부 개념 A: 자릿수 원리

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 자연수 나눗셈의 응용: $246 \div 2 = 123 \rightarrow 24.6 \div 2 = 12.3$<br>- **자릿수 원리**: 나누어지는 수가 $\frac{1}{10}$배 되면 몫도 $\frac{1}{10}$배<br>- 몫의 소수점 위치: 피제수의 소수점 위치를 그대로 올려 찍기 |
| **주요 오개념** | - **몫의 중간 0 누락** (가장 치명적): $6.12 \div 3 = 2.4$ (X) → $2.04$ (O)<br>- 나머지 해석 오류: $13.4 \div 4 = 3$...나머지를 14나 1로 표기 (나머지의 소수점은 원래 피제수 위치 따름) |
| **출제 포인트** | - 몫의 중간에 0이 들어가는 문항 필수 포함 (예: $6.12 \div 3$, $8.32 \div 4$)<br>- 자릿값 개념 테스트 |

#### 세부 개념 B: 문장제와 실생활

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 정수 부분 나누기: 몫이 1보다 작은 경우(0.xxx), 소수점 위치 맞추기<br>- 단위가 포함된 실생활 문제 해결 |
| **출제 포인트** | - "철사 2.5m를 5명에게 나눌 때..." 등 단위 포함 문제<br>- 나머지가 있는 소수 나눗셈에서 나머지의 소수점 위치 확인 |

### 4단원: 비와 비율 ⭐ (최고 난도/핵심 영역)

> 2022 개정 '변화와 관계' 영역의 핵심. 중등 함수의 기초가 되는 가장 중요한 단원.

#### 세부 개념 A: 비(Ratio)의 구조

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **비**: 두 수의 양을 비교하는 것 ($a:b$)<br>- **용어의 엄밀성**: $a$는 '비교하는 양', $b$는 '기준량'<br>- **읽기**: "$a$ 대 $b$", "$a$의 $b$에 대한 비", "$b$에 대한 $a$의 비"<br>- 기호 ':'의 오른쪽이 기준 |
| **주요 오개념** | - **기준량 찾기 실패**: "~에 대한"이 붙은 대상이 기준량($b$)임을 파악하지 못함<br>- 문장 어순과 비의 순서 혼동 |
| **출제 포인트** | - 문장 어순과 비 순서가 반대인 문제 (예: "사과 3개, 배 4개. 배에 대한 사과의 비는?")<br>- "~에 대한" 기준량 찾기 반복 연습 |

#### 세부 개념 B: 비율(Rate)과 백분율(Percentage)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **비율**: 기준량을 1(단위)로 볼 때의 비교하는 양의 크기 ($\frac{a}{b}$ 또는 소수)<br>- **백분율**: 기준량을 100으로 할 때의 비율<br>- 실생활 비율: 타율, 속력, 인구 밀도, 진하기 등 |
| **주요 오개념** | - **농도 분모 오류**: $\text{소금물 농도} = \frac{\text{소금}}{\text{소금물}(\text{소금}+\text{물})}$인데 분모를 '물'로 잡는 오류<br>- 비율과 비의 혼동 |

| 문항 유형 | 설명 | 핵심 전략 | 난이도 |
|-----------|------|-----------|--------|
| 기본 비율 | 두 수의 비율 계산 | 비교하는 양 / 기준량 | 하 |
| 농도 문제 | 소금물 진하기 | 분모 = 전체(소금+물)! | 중 |
| 백분율 변환 | 비율 ↔ 백분율 상호 변환 | ×100 / ÷100 | 중 |
| 기준량 변화 | 기준량이 변할 때 비율 변화 추론 | 함수적 사고 연결 | 상 |
| 복합 비율 | 할인율, 이익률 등 실생활 | 여러 단계의 비율 적용 | 상 |

### 5단원: 여러 가지 그래프

#### 세부 개념 A: 띠그래프와 원그래프

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **비율 그래프**: 전체에 대한 각 부분의 비율(백분율)을 시각화<br>- **작도 순서**: 각 항목의 백분율 구하기 → 백분율 합계(100%) 확인 → 구획 나누기<br>- **그래프 해석**: 대소 비교, 변화 추이 읽기<br>- 데이터 특성에 따른 그래프 선택 |
| **주요 오개념** | - **원그래프 중심각 계산 오류**: 30%를 30°로 그림 (정답: $360° \times 0.3 = 108°$)<br>- 백분율 합계 불일치 당황 (반올림으로 99% 또는 101%) |
| **출제 포인트** | - 비율 → 중심각 변환 식 세우기<br>- 합계가 100%가 되지 않을 때 조정하는 통계적 약속 이해 확인 |

#### 세부 개념 B: 그래프 해석과 정보 판단

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **비율과 실제 양 구분**: 비율이 크다고 양이 큰 것은 아님 (전체가 다를 때)<br>- 적절한 그래프 유형 선택 능력 |
| **주요 오개념** | - 비율과 실제 양 혼동: 비율이 크면 양도 크다고 착각<br>- 전체가 다른 두 그래프 비교 시 오류 |
| **출제 포인트** | - 자료의 일부가 가려진 그래프 해석하기<br>- 통계적 오류 찾기 (합계가 100%가 아닌 경우의 조정 방법) |

### 6단원: 직육면체의 부피와 겉넓이

#### 세부 개념 A: 부피의 개념과 공식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **부피**: 단위 부피($1cm^3$)가 몇 개 들어가는가? (쌓기나무 개수)<br>- **공식 유도**: $V = (\text{가로}) \times (\text{세로}) \times (\text{높이}) = (\text{밑면의 넓이}) \times (\text{높이})$<br>- **단위**: $1cm^3$, $1m^3$의 양감 |
| **주요 오개념** | - **단위 변환 실수**: $1m^3 = 100cm^3$ (정답: $1,000,000cm^3$). '0'의 개수 실수<br>- 부피 공식과 겉넓이 공식 혼동 |
| **출제 포인트** | - 쌓기나무로 부피 세기<br>- 단위 변환 ($1m^3 = 1,000,000cm^3$) 시각화 문제 |

#### 세부 개념 B: 겉넓이와 부피-겉넓이 관계

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **겉넓이**: 전개도 상의 6개 면의 넓이의 합. ($(\text{밑넓이} \times 2) + (\text{옆넓이})$)<br>- 부피와 겉넓이는 독립적인 값 |
| **주요 오개념** | - **"부피가 커지면 겉넓이도 항상 커진다?"** (반례: 납작한 모양 vs 정육면체)<br>- **"부피가 같으면 겉넓이도 같다"** (쌓기나무 개수 동일, 겉넓이 다른 반례) |
| **출제 포인트** | - 부피는 같지만(쌓기나무 개수 동일) 겉넓이가 다른 두 입체 비교<br>- 겉넓이가 주어지고 모서리의 길이 구하기 |

## 2학기 단원별 정리

### 1단원: 분수의 나눗셈 (분수÷분수)

> 1학기에서 배운 '분수÷자연수'를 확장하여 분수의 나눗셈을 완성하는 단원.

#### 세부 개념 A: 통분을 이용한 방법

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **통분 이용법**: 분모를 같게 만들면 분자끼리 나눗셈<br>  $\frac{a}{m} \div \frac{b}{m} = a \div b$<br>- 동분모 분수 나눗셈의 직관적 이해 |
| **출제 포인트** | - 통분 후 분자끼리 나누는 과정을 서술하는 문항 |

#### 세부 개념 B: 역수를 이용한 알고리즘

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **알고리즘 형식화**: $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$<br>- **역수**: $\frac{a}{b}$의 역수는 $\frac{b}{a}$ (두 수의 곱이 1)<br>- 번분수 형태 또는 통분 과정을 통해 역수 곱셈 원리 유도 |
| **주요 오개념** | - **대분수 처리 미숙**: 대분수 상태에서 바로 역수를 취함 ($1\frac{1}{2} \rightarrow 1\frac{2}{1}$ (X)). **반드시 가분수로 변환 후 계산**<br>- **앞의 수(피제수) 역수 취하기**: 나누어지는 수를 뒤집는 오류<br>- 나누는 수가 1보다 작을 때 몫이 처음 수보다 커짐을 이해 못함 |

| 문항 유형 | 설명 | 핵심 전략 | 난이도 |
|-----------|------|-----------|--------|
| 진분수÷진분수 | $\frac{3}{4} \div \frac{2}{5}$ | 역수 곱셈 | 하 |
| 대분수÷대분수 | $2\frac{1}{2} \div 1\frac{1}{3}$ | 가분수 변환 필수 | 중 |
| 과정 서술 | 계산 과정의 빈칸 채우기 | 서술형 평가 | 중 |
| 오류 찾기 | 틀린 계산 과정 제시 → 틀린 이유 서술 | 오류 분석 능력 | 상 |
| 원리 설명 | "역수를 곱하는 이유를 그림으로 설명하시오" | 과정 중심 평가 | 상 |

| 구분 | 내용 |
|------|------|

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 몫을 자연수 부분까지만 구하고 나머지 구하기<br>- 특정 자리에서 반올림하기 |

### 4단원: 비례식과 비례배분

> 1학기 '비와 비율'의 확장. 비례적 추론(Proportional Reasoning)의 핵심 도구.

#### 세부 개념 A: 비례식의 성질

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **비례식**: 비율이 같은 두 비를 등호(=)로 연결한 식 ($a:b = c:d$)<br>- **비례식의 성질**: 외항의 곱 = 내항의 곱 ($a \times d = b \times c$). 방정식 해결의 핵심 도구<br>- **비의 성질**: 전항·후항에 0 아닌 같은 수를 곱/나눠도 비의 값 동일 |
| **주요 오개념** | - **덧셈적 사고** (가장 치명적): $2:3 = 4:\square$에서 2가 4가 되려 $+2$ 했으니, 3에도 $+2$ 해서 5라고 답함. **(비례는 곱셈적 관계)**<br>- 외항/내항 위치 혼동 |
| **출제 포인트** | - 덧셈적 추론이 통하지 않는 수 제시 (예: $4:6 = 10:\square$)<br>- 미지수가 포함된 비례식 풀기 |

#### 세부 개념 B: 비례배분

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 전체를 주어진 비로 나누기: $\text{전체} \times \frac{a}{a+b}$<br>- 비례배분 결과의 합이 전체와 같은지 검증 |
| **주요 오개념** | - **비례배분 분모 오류**: $2:3$으로 나눌 때 분모를 $3$으로 잡음 (정답: 분모는 $2+3=5$)<br>- 비의 합 = 전체라고 착각 |
| **출제 포인트** | - 톱니바퀴 문제 (반비례 관계의 비례식 표현)<br>- 재산 상속이나 이익 배분 등의 실생활 비례배분<br>- 전체 막대 시각적 모델로 비례배분 확인 |

### 5단원: 원의 넓이

> 2022 개정에서 원주율을 3.14로 통일. 계산의 복잡성을 줄이고 개념적 이해에 집중.

#### 세부 개념 A: 원주율과 원주

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **원주율($\pi$)**: 지름에 대한 원주의 비율 (근삿값 **3.14** 고정 사용)<br>- **원주**: $2\pi r$ 또는 $\pi \times \text{지름}$<br>- 원주율은 어떤 원이든 일정한 상수 |
| **주요 오개념** | - 반지름과 지름 혼동<br>- 원주율 $\pi$를 정확한 수로 착각 |

#### 세부 개념 B: 원의 넓이 공식 유도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **공식 유도 (무한 분할)**: 원을 잘게 잘라 이어 붙이면 직사각형이 됨<br>  - 가로 = 원주의 $\frac{1}{2}$ ($\pi r$), 세로 = 반지름 ($r$)<br>  - **넓이 = $\pi r \times r = \pi r^2$** |
| **주요 오개념** | - **공식 혼용**: 원주($2\pi r$) ↔ 원의 넓이($\pi r^2$) 혼동<br>- **"지름 × 지름 × 원주율"로 넓이 계산하는 실수**<br>- **반원/부채꼴 둘레 오류**: 곡선 부분만 계산, 직선(지름) 누락 |
| **출제 포인트** | - 관계적 사고 문항: "반지름이 2배 → 원주 몇 배? 넓이 몇 배?"<br>- 운동장 트랙(직사각형+반원 2개)의 둘레와 넓이 구하기<br>- 색칠한 부분(도넛 모양 등)의 넓이 구하기 |

### 6단원: 원기둥, 원뿔, 구

> 회전체라는 곡선 도형을 다루며, 평면도형과 입체도형의 관계를 이해하는 단원.

#### 세부 개념 A: 회전체의 이해

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **회전체**: 평면도형(직사각형, 직각삼각형, 반원)을 한 직선을 축으로 1회전시켜 만든 입체<br>- **단면**: 회전축에 수직인 단면(원)과 회전축을 포함하는 단면(선대칭도형) |
| **주요 오개념** | - 회전시키기 전의 평면도형을 역추론하지 못함<br>- 원뿔의 높이(수직 거리) ↔ 모선(옆면을 따라 내려가는 길이) 혼동 (각뿔과 동일한 오류 패턴) |

#### 세부 개념 B: 원기둥의 겉넓이와 전개도

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **전개도**: 옆면의 가로가 밑면의 원주와 같음<br>- **겉넓이**: 밑넓이 × 2 + 옆넓이<br>  - 옆넓이 = 원주 × 높이 = $2\pi r \times h$ |
| **주요 오개념** | - 원기둥 옆면 전개도가 직사각형인 것 이해 못함<br>- 옆면 가로 길이 = 밑면 원주 연결 실패 |

#### 세부 개념 C: 구의 특수성

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - 구: 중심에서 같은 거리인 점들의 집합<br>- 구는 평면으로 완벽히 펼칠 수 없음 (전개도 불가) |
| **주요 오개념** | - **"구도 전개도를 그릴 수 있다"**: 구는 평면으로 펼칠 수 없음 (지구본→세계지도 왜곡 예시) |
| **출제 포인트** | - 모선 길이, 높이, 밑면 반지름 모두 제시 → '높이' 정확히 선택하는지 확인<br>- 회전체의 단면 유추 문제<br>- 회전시키기 전의 평면도형 찾기 |

- **미지수 표기**: '$x$' 대신 '$\square$' 또는 '어떤 수' 사용하되, 식을 세우는 논리는 중등 방정식($ax=b$)과 매끄럽게 연결되도록 설계
