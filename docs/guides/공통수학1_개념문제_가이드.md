## 📘 단원별 정리

### 1단원: 다항식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **다항식의 연산**: 내림차순 정리, 덧셈/뺄셈/곱셈<br>  - 곱셈 공식의 변형: 고교 수학 전반의 핵심 도구<br>• **항등식(Identity)**: 모든 $x$에 대해 성립하는 등식<br>  - 미정계수법: 수치대입법, 계수비교법<br>• **나머지정리**: $P(x)$를 $(x-\alpha)$로 나눈 나머지 = $P(\alpha)$<br>  - 인수정리의 이론적 토대<br>• **인수분해**: 다항식을 인수들의 곱으로 표현 |
| **주요 오개념** | • **⭐⭐ 항등식과 방정식 혼동**:<br>  "$x$에 대한 식" 보면 무조건 $x$ 값을 구하려 함<br>  → "모든 실수 $k$에 대해 성립" 조건에서 **$k$가 주체 변수**임을 인식 못함<br>• **문자의 역할 혼동**: 변수 vs 상수 구분 실패<br>  → 특정 문자에 대해 내림차순 정리하는 습관 필요 |
| **출제 포인트** | ✓ 항등식 조건에서 미정계수 구하기<br>✓ 나머지정리 활용 (함숫값 해석)<br>⚠️ **지양**: 지나치게 복잡한 인수분해, 기계적 조립제법 반복 |

### 2단원: 방정식과 부등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **복소수**: 허수단위 $i$ ($i^2 = -1$), 켤레복소수<br>  → 이차방정식의 해 표현 도구 (극형식, 복소평면 X)<br>• **이차방정식과 이차함수**:<br>  - 판별식 $D$: 근의 개수 + 그래프와 $x$축 위치 관계<br>  - 근과 계수의 관계: 합 = $-\frac{b}{a}$, 곱 = $\frac{c}{a}$<br>  - **제한된 범위에서의 최대·최소**: 꼭짓점이 범위 안인지 확인<br>• **여러 가지 방정식/부등식**:<br>  - 삼차/사차방정식, 연립이차방정식<br>  - 연립일차부등식, 이차부등식<br>  - 이차부등식 → 그래프 개형과 $x$축 교점으로 해석 |
| **주요 오개념** | • **⭐⭐⭐ 판별식 오남용**:<br>  "모든 실수에 대해 $kx^2 + 2x + k > 0$" 조건<br>  → 학생: $D < 0$만 계산<br>  → **빠뜨리는 것**: 이차항 계수 $k > 0$ 전제 + $k = 0$일 때 별도 체크<br>• **⭐⭐ 다변수 최대·최소에서 변수 독립성 무시**:<br>  $x^2 + 4x + 2y^2 - 4y + 3z^2 + 6z - 4$의 최솟값<br>  → 완전제곱식 변형 후 "최솟값 = -13"이라 결론<br>  → **오류**: $x+y+z=1$ 같은 제약 조건 있으면 성립 안 함<br>• **이차항 계수 조건 간과**: 이차방정식 되려면 $a \neq 0$ |
| **출제 포인트** | ✓ "항상 성립" 조건 → 이차항 계수 부호 + 판별식 **동시** 확인<br>✓ 제한된 범위에서 최대·최소 (꼭짓점 위치 판단)<br>✓ 다변수 함수: 변수 간 종속성 확인 |

### 3단원: 경우의 수 ⭐ (1학기 이동)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **합의 법칙**: 배반사건일 때 $n(A) + n(B)$<br>• **곱의 법칙**: 잇달아/동시에 일어날 때 $n(A) \times n(B)$<br>• **순열** $_nP_r$: 서로 다른 $n$개에서 $r$개 택해 **나열** (순서 O)<br>• **조합** $_nC_r$: 서로 다른 $n$개에서 $r$개 **택함** (순서 X) |
| **주요 오개념** | • **⭐⭐⭐ 동적 선택지 축소 오류 (Dynamic Pool Resizing)**:<br>  0~9 중 4자리 자연수 만들기 (특정 숫자 제외)<br>  - 천의 자리: 0 제외 → 선택지 감소<br>  - 백의 자리: 0 부활, 천의 자리 숫자 제외<br>  → 각 자릿수마다 **사용 가능한 숫자 풀이 동적으로 변함**<br>• **⭐⭐ 여사건과 포함관계 오류**:<br>  "2의 배수도 아니고 5의 배수도 아닌 수"<br>  → 단순히 (2의 배수 개수 + 5의 배수 개수) 빼는 오류<br>  → **교집합(10의 배수) 두 번 빼는 실수** (포함-배제 원리 누락)<br>• **⭐⭐ '순서가 정해진' vs '이웃하는' vs '자리 고정' 혼동**:<br>  - "A가 B보다 반드시 앞" ≠ "A가 맨 앞 고정" ≠ "A와 B가 붙어있음"<br>  → "순서가 정해졌다 = 그들끼리 자리 바꿀 권한 박탈" |
| **출제 포인트** | ✓ 숫자 나열해 놓고 지워가며 카운팅 (시각적 전략)<br>✓ 수형도(Tree Diagram)로 '빠짐없이, 중복 없이' 세기<br>✓ 포함-배제 원리 적용 문제 |

### 4단원: 행렬 ⭐⭐ (신설)

| 구분 | 내용 |
|------|------|
| **핵심 개념** | • **행렬의 정의**: 행과 열로 이루어진 수의 배열<br>• **범위 제한**: **2×2 행렬**까지만<br>• **행렬의 연산**: 덧셈, 뺄셈, 실수배, **곱셈**<br>  - 행렬 곱셈: AI 신경망 연산의 기초<br>• **제외 내용**: 역행렬, 행렬식($ad-bc$), 연립방정식 풀이<br>  → 행렬은 방정식을 '풀' 수 없고 **'나타낼'** 수만 있음 |
| **주요 오개념** | • **⭐⭐⭐ 교환법칙 성립 착각**:<br>  실수 연산 습관으로 $AB = BA$라고 가정<br>  → $(A+B)^2 = A^2 + 2AB + B^2$ ✗<br>  → **정답**: $A^2 + AB + BA + B^2$ (교환 안 됨!)<br>• **⭐⭐ 영인자(Zero Divisor) 오해**:<br>  실수: $ab = 0$ → $a=0$ or $b=0$<br>  행렬: $A \neq O$, $B \neq O$인데 $AB = O$ **가능**<br>  → 반례: $\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} = O$<br>• **⭐⭐ 약분(Cancellation) 시도**:<br>  $AB = AC$, $A \neq O$ → 양변을 $A$로 나눠 $B = C$?<br>  → 역행렬 개념 없으므로 **불가능** |
| **출제 포인트** | ✓ $AB \neq BA$ 확인 문제<br>✓ $(A+B)^2$ 전개 시 $AB + BA$ 구분<br>✓ 영인자 반례 제시<br>⚠️ **제외**: 역행렬, 행렬식, 연립방정식 풀이 |

## 💡 문제 개발 전략

### 1. 판별식 문제 완전 체크리스트

| 조건 | 확인 사항 |
|------|-----------|
| "$ax^2 + bx + c > 0$이 항상 성립" | ① $a > 0$ (아래로 볼록) ② $D < 0$ (교점 없음) |
| "$ax^2 + bx + c \geq 0$이 항상 성립" | ① $a > 0$ ② $D \leq 0$ |
| "$a = 0$일 때" | 별도로 체크 (일차부등식으로 전환) |

> **핵심**: 판별식만 계산하면 안 됨, **이차항 계수 조건 필수**

---

### 2. 경우의 수: 동적 풀 관리 전략

```
[예시] 0, 4, 5, 7, 8, 9로 4자리 자연수 만들기

천의 자리: {4, 5, 7, 8, 9} → 5가지 (0 제외)
백의 자리: {0, 나머지 4개} → 5가지 (0 부활, 천의 자리 제외)
십의 자리: 4가지
일의 자리: 3가지

→ 5 × 5 × 4 × 3 = 300
```

> **핵심**: 각 자리마다 "사용 가능한 숫자 풀"이 **동적으로 변함**

### 3. 행렬 연산 핵심 규칙

| 실수에서 성립 | 행렬에서 성립 여부 |
|---------------|-------------------|
| $ab = ba$ | ❌ $AB \neq BA$ (일반적으로) |
| $ab = 0$ → $a=0$ or $b=0$ | ❌ 영인자 존재 |
| $ab = ac$, $a \neq 0$ → $b = c$ | ❌ 약분 불가 (역행렬 없음) |
| $(a+b)^2 = a^2 + 2ab + b^2$ | ❌ $(A+B)^2 = A^2 + AB + BA + B^2$ |

### 2022 개정 3대 변화

| 변화 | 의미 |
|------|------|
| **행렬 신설** | AI·데이터 과학 기초 소양 |

### 행렬 지도 핵심 3가지

1. **교환법칙 불성립**: $AB \neq BA$ (반례로 체화)
2. **영인자 존재**: $A \neq O$, $B \neq O$인데 $AB = O$ 가능
3. **약분 불가**: 역행렬 없으므로 양변 나누기 불가
