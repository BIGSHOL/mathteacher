## 공통수학1의 특징

- **위상**: 고등학교 수학의 **첫 관문**, 고교학점제 기반 공통 과목 (4학점)
- **학문적 전환**:
  - 기하학적 직관 → **수식의 논리적 전개와 패턴 파악** 중심
  - 연산 능력(Computation) → **논리적 추론(Reasoning)** → **구조적 파악(Structure)** 의 위계적 흐름
  - 시각적 보조 도구 의존 → **추상적 사고력** 요구
- **핵심 변화**:
  - **행렬 신설**: 2009 개정 이후 삭제되었다가 AI/데이터 과학 기초 소양 목적으로 **부활** (2x2 한정)
  - **경우의 수 전진 배치**: 2학기(수학 하) → **1학기로 이동** (이산수학적 사고 조기 정립)
  - **도형의 방정식**: 공통수학2로 이동 (대수/기하 분리)
  - **계산 복잡도 완화**: 기계적 문제 풀이 기술 → **구조 파악과 논리적 사고** 전환
- **2022 개정 핵심 의도**:
  - 4차 산업혁명/AI 시대의 **수학적 소양**과 **디지털 리터러시** 함양
  - 대수적(Algebraic) 소양 + 이산수학적(Discrete Math) 논리 체계를 고1 1학기부터 강화
  - 수학의 **'언어적, 논리적 도구'**로서의 성격 대폭 강화

> 공통수학1은 **'디지털 대전환 시대의 수학적 기초 체력'**을 기르는 데 초점

### 1단원: 다항식 - 연산을 넘어선 구조의 이해

> 대수적 조작의 기본기를 다지는 영역. 단순한 전개와 분해의 기계적 숙달을 넘어 $A = BQ + R$ 이라는 항등식의 구조를 깊이 있게 이해하는 것이 목표.

#### 세부 개념 A: 다항식의 연산과 항등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **다항식의 연산**: 내림차순 정리, 덧셈·뺄셈·곱셈, 곱셈 공식의 변형(고교 수학 전반의 핵심 도구)<br>- **항등식(Identity)**: 모든 $x$에 대해 성립하는 등식<br>- **미정계수법**: 수치대입법(특정 값 대입), 계수비교법(양변의 동류항 비교)<br>- 항등식의 본질: $A = BQ + R$은 '다항식 공간에서의 분해 가능성' 탐구 |
| **주요 오개념** | - **항등식과 방정식 혼동** (가장 빈출): '$x$에 대한 식' 보면 무조건 $x$ 값을 구하려 함<br>  → "모든 실수 $k$에 대해 성립" 조건에서 **$k$가 주체 변수**임을 인식 못함<br>- **문자의 역할 혼동**: 변수 vs 상수 구분 실패<br>  → 특정 문자에 대해 내림차순 정리하는 습관 필요 |
| **출제 포인트** | - 항등식 조건에서 미정계수 구하기<br>- 문자의 역할(변수 vs 상수) 구분 문항 |

#### 세부 개념 B: 나머지정리와 인수분해

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **나머지정리**: $P(x)$를 $(x-\alpha)$로 나눈 나머지 = $P(\alpha)$<br>  → 함수값 계산 = 나눗셈의 나머지를 구하는 것과 **동치**<br>- **인수정리**: $P(\alpha) = 0$ → $(x-\alpha)$는 $P(x)$의 인수<br>  → 고차방정식 해법의 기초<br>- **조립제법**: 간편한 다항식 나눗셈 알고리즘 |
| **주요 오개념** | - **삼차식 제수의 구조적 오류** (최상위권 변별 핵심):<br>  $f(x)$를 $(x-1)^2$으로 나눈 나머지가 $3x+2$이고, $(x+1)$로 나눈 나머지가 $3$일 때<br>  $(x-1)^2(x+1)$로 나눈 나머지 $R(x)$를 구하라<br>  → 오류: $R(x) = ax^2+bx+c$로 설정 (미지수 3개, 조건 2개로 풀 수 없는 딜레마)<br>  → 정답: **나머지의 재귀적 구조** 활용 → $R(x) = a(x-1)^2 + (3x+2)$로 구조화<br>- **조립제법 계수 오류**: $ax+b$ ($a \neq 1$)로 나눌 때 조립제법 결과를 다시 $a$로 나누어야 하는데 누락 |
| **출제 포인트** | - 나머지정리 활용 (함숫값 해석)<br>- 나머지의 재귀적 구조 추론 문항<br>- 지나치게 복잡한 인수분해 **지양**, 원리 중심 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 미정계수법 | 항등식 조건에서 상수 구하기 | 중 |
| 나머지정리 | 일차식으로 나눈 나머지 = 함숫값 | 중 |
| 나머지의 구조 | 삼차식 제수에서 나머지 재귀 구조 설계 | 상 |
| 인수정리 활용 | 고차식의 인수 판별 | 중상 |
| 조립제법 | $ax+b$ ($a \neq 1$) 제수에서 계수 보정 | 중상 |

### 2단원: 방정식과 부등식 - 해석기하학적 연결

> 복소수라는 수 체계의 완성과, 이차방정식 및 이차부등식을 이차함수의 그래프와 연결하여 해석하는 통합적 사고를 요구하는 단원.

#### 세부 개념 A: 복소수와 이차방정식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **복소수**: 실수 체계를 확장하여 $i^2 = -1$ 도입 → 이차방정식이 항상 해를 갖도록 하는 **대수적 닫힘(Algebraic Closure)** 완성<br>- **판별식 $D$**: 실근/허근 판별 + 이차함수 $y = ax^2 + bx + c$의 그래프가 $x$축과 만나는 **교점 개수**(위치 관계의 핵심 지표)<br>- **근과 계수의 관계**: 합 = $-b/a$, 곱 = $c/a$<br>- **켤레복소수**: $\alpha \pm \beta i$ 쌍의 성질 |
| **주요 오개념** | - **음수의 제곱근 연산 성질** (가장 치명적):<br>  $\sqrt{-2} \times \sqrt{-3} = \sqrt{6}$ (X) → 정답: $-\sqrt{6}$<br>  $a<0, b<0$일 때 허수단위 $i$가 두 번 곱해지면서 부호 반전<br>  → 문자 포함 식 $\sqrt{x-1}\sqrt{x-2} = -\sqrt{(x-1)(x-2)}$에서 $x$의 범위($1<x<2$) 추론 문항<br>- **이차항 계수 조건 간과**: 이차방정식이려면 $a \neq 0$ |
| **출제 포인트** | - 음수의 제곱근 성질과 $x$ 범위 추론<br>- 판별식의 기하학적 의미 (교점 개수) |

#### 세부 개념 B: 이차함수와 이차부등식

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **이차함수의 최대·최소**: 정의역이 제한된 경우($\alpha \le x \le \beta$), 반드시 대칭축 위치 확인<br>- **이차부등식**: 그래프 개형 + $x$축 교점으로 해석<br>- **'항상 성립' 조건**: 이차항 계수 부호 + 판별식 **동시** 확인<br>- **절댓값 부등식**: $|x-a| < b$ → 수직선 상 **거리**로 해석 |
| **주요 오개념** | - **판별식 $D$와 함숫값 $y$의 부호 혼동** (매우 빈출):<br>  "모든 실수 $x$에 대해 $y > 0$" → 학생: $D > 0$ (X)<br>  → 정답: 그래프가 $x$축 위에 **떠 있으려면** 교점이 **없어야** → $D < 0$<br>  "위에 있다(Positive position)" vs "교점 없다(Negative discriminant)"의 이중적 의미 혼동<br>- **제한된 범위 최대·최소 오류**: 꼭짓점이 범위 밖이면 경계값에서 최대·최소 발생<br>  → 꼭짓점을 맹목적으로 답으로 선택하는 오류 (정의역 인식 부족)<br>- **'항상 성립' 판별식에서 $a=0$ 별도 체크 누락**:<br>  $kx^2 + 2x + k > 0$ → $k > 0$ 전제 + $D < 0$ + $k = 0$일 때 별도 확인 필수 |
| **출제 포인트** | - '항상 성립' 조건 → 이차항 계수 부호 + 판별식 + $a=0$ 체크<br>- 제한된 범위에서 최대·최소 (꼭짓점 위치 판단)<br>- 절댓값 부등식의 기하학적 해석 |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 복소수 연산 | 음수의 제곱근 곱셈에서 부호 판단 | 중 |
| 범위 추론 | $\sqrt{x-a}\sqrt{x-b}$의 부호 결정 조건 | 상 |
| 항상 성립 | $D$ + 이차항 계수 + $a=0$ 동시 체크 | 상 |
| 제한 범위 | 대칭축 위치에 따른 최대·최소 분기 | 중상 |
| 이차부등식 | 그래프 개형으로 부등식 해 해석 | 중 |
| 절댓값 복합 | $|x-a| + |x-b| < k$ 기하학적 풀이 | 상 |

### 3단원: 경우의 수 - 논리적 분류의 미학

> 기존 2학기(수학 하)에서 1학기로 이동. 대수적 논리 학습과 병행하여 논리적 분류 및 조직화(Logical Organization) 능력을 조기에 배양.

#### 세부 개념 A: 합의 법칙과 곱의 법칙

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **합의 법칙(Rule of Sum)**: 두 사건이 동시에 일어나지 않을 때(배반사건, Mutually Exclusive) 경우의 수를 더함<br>- **곱의 법칙(Rule of Product)**: 사건이 연이어서/동시에 일어날 때 경우의 수를 곱함<br>- **'동시에'의 정확한 의미**: 물리적 시간 일치가 아니라 '하나의 사건이 완성되기 위해 일련의 과정이 모두 수행되어야 함'<br>- 직관적 룰: **'분류하면 더하고, 진행하면 곱한다'** |
| **주요 오개념** | - **'동시에(Simultaneously)'의 언어적 오해** (매우 빈출):<br>  "상의를 입고 나서 하의를 입는 것은 시간차가 있으니 동시가 아니다" → 합의 법칙 적용 시도(X)<br>  → 양립할 수 없는(배반) 사건까지 곱하는 과대 계상(Overcounting) 오류<br>- **분류와 진행의 구분 실패**:<br>  "십의 자리와 일의 자리 합이 홀수" → (홀+짝)과 (짝+홀)로 **분류** = 합의 법칙<br>  → 각 Case 내에서 자리별 숫자 **선택** = 곱의 법칙 |
| **출제 포인트** | - 합의 법칙/곱의 법칙 적용 상황 구분 문항<br>- 수형도(Tree Diagram)로 '빠짐없이, 중복 없이' 세기 |

#### 세부 개념 B: 순열과 조합

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **순열 $_nP_r$**: 서로 다른 $n$개에서 $r$개 택해 **나열** (순서 O)<br>- **조합 $_nC_r$**: 서로 다른 $n$개에서 $r$개 **택함** (순서 X)<br>- 2022 개정: 단순 공식 암기보다 **상황에 맞는 도구 선택 능력** 강조 |
| **주요 오개념** | - **0을 포함한 순열(The Zero Trap)** (매우 빈출):<br>  자연수 구성 시 맨 앞자리에 0 불가 → 특정 숫자 제외/포함 복합 조건과 결합 시 강력한 오류 유발<br>  → 여사건 사용 시 전체 집합을 '0이 맨 앞에 오는 경우를 포함한 순열'로 잘못 설정<br>- **동적 선택지 축소(Dynamic Pool Resizing)** (최다 오답 원인):<br>  0~9 중 특정 숫자 제외 후 4자리 자연수 만들기<br>  → 천의 자리: 0 제외(선택지 감소) → 백의 자리: 0 부활, 천의 자리 숫자 제외<br>  → 각 자릿수마다 **사용 가능한 숫자 풀이 동적으로 변함**<br>- **여사건과 포함-배제 원리 누락**:<br>  "2의 배수도 아니고 5의 배수도 아닌 수" → 단순히 (2의 배수 + 5의 배수) 빼기(X)<br>  → 교집합(10의 배수) 이중 제거 실수<br>- **'순서 정해진' vs '이웃하는' vs '자리 고정' 혼동**:<br>  "A가 B보다 반드시 앞" ≠ "A가 맨 앞 고정" ≠ "A와 B가 붙어있음"<br>  → "순서가 정해졌다 = 그들끼리 자리 바꿀 권한 박탈" |
| **출제 포인트** | - 숫자 나열하고 지워가며 카운팅 (시각적 전략)<br>- 포함-배제 원리 적용 문제<br>- 조건부 순열/조합 (고정, 이웃, 순서 제약) |

| 문항 유형 | 설명 | 난이도 |
|-----------|------|--------|
| 합/곱 구분 | 분류(합) vs 진행(곱) 판단 | 하 |
| 동적 풀 관리 | 자릿수별 선택지 변화 추적 | 중상 |
| 0 포함 순열 | 맨 앞자리 0 제외 + 여사건 | 중 |
| 포함-배제 | 교집합 이중 계산 방지 | 중상 |
| 조건부 나열 | 순서 고정/이웃/자리 제약 통합 | 상 |
| 여사건 활용 | 전체에서 빼기 전략 | 중 |

### 4단원: 행렬 - 디지털 시대의 새로운 언어 (신설)

> 2022 개정 교육과정에서 가장 주목받는 변화. AI와 데이터 과학의 기초가 되는 '데이터의 구조화'를 이해하기 위해 도입. 비가환적(Non-commutative) 연산 체계의 첫 경험.

#### 세부 개념 A: 행렬의 정의와 기본 연산

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **행렬의 정의**: 행과 열로 이루어진 수의 배열 (2x2 한정)<br>- **기본 연산**: 덧셈, 뺄셈, 실수배 → 성분별(Element-wise) 연산으로 직관적<br>- **범위 제한**: 역행렬, 행렬식($ad-bc$), 연립방정식 풀이 **제외**<br>  → 행렬은 방정식을 '풀' 수 없고 **'나타낼'** 수만 있음<br>- 데이터를 묶어서 처리할 수 있다는 **가능성** 인식 |
| **주요 오개념** | - 기본 연산에서는 오개념 적음 (성분별 연산이 직관적) |

#### 세부 개념 B: 행렬의 곱셈과 비가환성

| 구분 | 내용 |
|------|------|
| **핵심 개념** | - **행렬 곱셈**: (행) $\times$ (열)의 **내적** 구조, 선형 변환의 합성을 내포<br>- **비가환성**: 일반적으로 $AB \neq BA$ → 실수와 완전히 다른 연산 체계<br>- **영행렬 $O$과 단위행렬 $E$**: 덧셈의 항등원 / 곱셈의 항등원<br>- **행렬의 거듭제곱**: 규칙성 찾기 문제 (케일리-해밀턴 정리의 간접적 활용 소지) |
| **주요 오개념** | - **교환법칙 성립 착각** (가장 치명적, 서술형 최대 감점 요인):<br>  $(A+B)^2 = A^2 + 2AB + B^2$ (X)<br>  → 정답: $(A+B)^2 = A^2 + AB + BA + B^2$ ($AB$와 $BA$는 동류항이 아님!)<br>- **영인자(Zero Divisors)** (직관 파괴):<br>  실수: $ab = 0$ → $a=0$ or $b=0$<br>  행렬: $A \neq O$, $B \neq O$인데 $AB = O$ **가능**<br>  → '0'과 '방정식의 해'에 대한 기존 직관을 깨뜨리는 개념<br>- **약분(Cancellation) 시도**:<br>  $AB = AC$, $A \neq O$ → 양변을 $A$로 나눠 $B = C$? (X)<br>  → 역행렬 개념이 없으므로 **불가능**<br>  → $AB = AC$에서 $A \neq O$라도 $B \neq C$일 수 있음 (역행렬 존재하지 않는 경우) |
| **출제 포인트** | - $AB \neq BA$ 확인 문제 (반례 제시)<br>- $(A+B)^2$ 전개 시 $AB + BA$ 구분<br>- 영인자 반례 구성<br>- 진위 판정(ㄱ, ㄴ, ㄷ 합답형) 문항 |

| 실수에서 성립 | 행렬에서 성립 여부 | 교육적 핵심 |
|---------------|-------------------|-------------|
| $ab = ba$ | $AB \neq BA$ (일반적) | 비가환적 연산 체계 인식 |
| $ab = 0$ → $a=0$ or $b=0$ | 영인자 존재 | 실수 직관의 한계 인식 |
| $ab = ac$, $a \neq 0$ → $b = c$ | 약분 불가 | 역행렬 개념 부재 |
| $(a+b)^2 = a^2 + 2ab + b^2$ | $(A+B)^2 = A^2 + AB + BA + B^2$ | 곱셈 공식 재정립 |

## 문제 개발 전략

### 1. 구조적 사고(Structural Thinking) 훈련

| 나쁜 예 | 좋은 예 |
|---------|---------|
| "$f(x)$를 $(x-1)$로 나눈 나머지를 구하시오" | "$f(x)$를 $(x-1)^2(x+1)$로 나눈 나머지를 구하시오" (구조 설계 필수) |
| "$ax^2+bx+c > 0$이 항상 성립할 조건" (단순 판별식) | "$kx^2+2x+k > 0$이 항상 성립할 $k$의 범위" ($k=0$ 별도 체크 포함) |
| "조합 공식으로 $_5C_3$을 구하시오" | "조건부 선택(특정 원소 포함/제외/고정)에서 경우의 수" |

**전략**: 답을 구하기 전 **식의 형태와 문제의 구조를 먼저 설계**하는 훈련

### 2. 판별식 문제 완전 체크리스트

| 조건 | 확인 사항 |
|------|-----------|
| "$ax^2 + bx + c > 0$이 항상 성립" | 1. $a > 0$ (아래로 볼록) 2. $D < 0$ (교점 없음) |
| "$ax^2 + bx + c \geq 0$이 항상 성립" | 1. $a > 0$ 2. $D \leq 0$ |
| "$a = 0$일 때" | 별도로 체크 (일차부등식으로 전환) |

> **핵심**: 판별식만 계산하면 안 됨, **이차항 계수 조건 필수**

### 3. 경우의 수: 동적 풀 관리 전략

```
[예시] 0, 4, 5, 7, 8, 9로 4자리 자연수 만들기

천의 자리: {4, 5, 7, 8, 9} → 5가지 (0 제외)
백의 자리: {0, 나머지 4개} → 5가지 (0 부활, 천의 자리 제외)
십의 자리: 4가지
일의 자리: 3가지

→ 5 x 5 x 4 x 3 = 300
```

> **핵심**: 각 자리마다 "사용 가능한 숫자 풀"이 **동적으로 변함**

### 4. 정의의 엄밀성(Rigor of Definition) 강조

| 혼동 지점 | 정확한 구분 |
|-----------|-------------|
| $D > 0$ vs $y > 0$ | $D$는 교점 개수, $y$는 함숫값의 부호 |
| $AB = BA$ | 행렬 곱셈은 비가환 (반례 필수 제시) |
| "동시에" | 물리적 시간 동시가 아닌, 사건 완성을 위한 일련의 과정 |
| $\sqrt{a}\sqrt{b} = \sqrt{ab}$ | $a, b$ 모두 음수일 때는 $-\sqrt{ab}$ |

### 5. 역량 기반 문항 설계

| 핵심 역량 | 문항 설계 포인트 | 예시 |
|-----------|------------------|------|
| **구조적 사고** | 식의 형태 설계 후 계산 | 나머지의 재귀적 구조 |
| **추론** | 정의에서 논리적 결론 도출 | $AB = O$에서 $A = O$ 또는 $B = O$? |
| **메타인지** | 자기 풀이의 결함 점검 | "미지수 3개인데 조건 2개인 딜레마" 인식 |
| **정보처리** | 동적 조건 추적 | 자릿수별 선택지 변화 |
| **의사소통** | 풀이 과정 서술 | $(A+B)^2$ 전개 과정 설명 |

### 6. 연결성(Connectivity) 강화

| 공통수학1 | 연결 방향 | 후속 과목 |
|-----------|-----------|-----------|
| 다항식 (항등식, 나머지정리) | → | 고차방정식 해법, 다항함수의 미적분 |
| 방정식과 부등식 (판별식, 이차함수) | → | 공통수학2 도형의 방정식, 미적분 |
| 경우의 수 (순열, 조합) | → | 확률과 통계 (확률, 이항정리) |
| 행렬 (연산, 비가환성) | → | 기하(선형 변환), AI/데이터 과학, 선형대수학 |
| 다항식 + 행렬 | → | 연립일차방정식의 행렬 표현 |
| 방정식 + 경우의 수 | → | 부정방정식의 정수해 개수 (이산수학적 융합) |

### 7. 단원 간 융합 문항 소재

|-----------|------|-----------|
