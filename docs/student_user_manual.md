## 1. 시작하기 🎯

### 1.1 플랫폼 소개

**이게 뭐예요?**
수학을 쉽고 재미있게 공부할 수 있는 온라인 학습 플랫폼이에요.
게임처럼 레벨을 올리고 경험치를 모으면서 수학 실력도 함께 향상시킬 수 있어요!

**누가 사용하나요?**
- 초등학교 1학년 ~ 고등학교 1학년 학생
- 수학을 처음 배우는 학생부터 심화 학습을 원하는 학생까지

**뭘 할 수 있나요?**
- ✅ 내 수준에 맞는 수학 문제 풀기
- ✅ 레벨 업하고 경험치(XP) 모으기
- ✅ 연속 출석으로 콤보 쌓기
- ✅ 내 성적 확인하고 취약한 부분 찾기
- ✅ 빈칸 채우기로 개념 완벽 암기하기

---

### 1.2 필요한 것

**기기**
- 컴퓨터 (Windows, Mac)
- 태블릿 (iPad, 안드로이드)
- 스마트폰 (큰 화면 권장)

**인터넷**
- Wi-Fi 또는 데이터 연결 필요
- 안정적인 연결 권장

**계정**
- 선생님이 만들어주신 이메일과 비밀번호
- 처음 로그인 후 비밀번호 변경 가능

## 2. 로그인하기 🔐

### 2.1 첫 로그인

**STEP 1: 사이트 접속**
```
웹 브라우저(크롬, 사파리 등)를 열고 주소창에 입력하세요:
https://your-math-platform.vercel.app
```

**STEP 2: 로그인 정보 입력**
1. 이메일 입력란에 선생님이 알려준 이메일 주소를 입력하세요
   ```
   예: student01@school.com
   ```

2. 비밀번호 입력란에 초기 비밀번호를 입력하세요
   ```
   예: test1234
   ```

3. "로그인" 버튼을 클릭하세요

**STEP 3: 처음 로그인 확인**
- 로그인에 성공하면 메인 화면으로 이동해요
- 화면 오른쪽 위에 내 이름이 보이면 성공!

### 2.2 비밀번호 변경하기 (권장)

**왜 바꿔야 하나요?**
초기 비밀번호는 다른 친구들도 알 수 있어요. 내 성적과 정보를 지키려면 비밀번호를 바꾸는 게 좋아요!

**어떻게 바꾸나요?**
1. 오른쪽 위 내 이름 클릭
2. "설정" 선택
3. "비밀번호 변경" 클릭
4. 현재 비밀번호 입력
5. 새 비밀번호 입력 (8자 이상 추천)
6. 새 비밀번호 확인
7. "변경하기" 클릭

**TIP 💡**
- 쉬운 비밀번호: 생일, 이름 등은 피하세요
- 좋은 비밀번호 예: `Math2024!`, `Study@123`
- 비밀번호는 꼭 어딘가에 메모해두세요!

### 2.3 로그인 문제 해결

**"이메일 또는 비밀번호가 틀렸습니다"가 뜨면?**
1. 이메일 주소가 정확한지 확인하세요
2. 비밀번호를 다시 한번 천천히 입력해보세요
3. 대문자/소문자를 구분하는지 확인하세요
4. 그래도 안 되면 선생님께 문의하세요

**"서버에 연결할 수 없습니다"가 뜨면?**
1. 인터넷 연결을 확인하세요
2. 다른 웹사이트도 열리는지 확인하세요
3. 페이지를 새로고침(F5)해보세요
4. 잠시 후 다시 시도하세요

## 3. 메인 화면 둘러보기 🏠

### 3.1 화면 구성

로그인하면 제일 먼저 보이는 메인 화면이에요!

```
┌────────────────────────────────────────────────┐
│  📚 수학 학습 플랫폼        [내 이름] ▼ [로그아웃] │
├────────────────────────────────────────────────┤
│                                                │
│  🎯 내 정보                                     │
│  ┌──────────────────────────────────────────┐ │
│  │ Lv.5  ■■■■■□□□□□ 다음 레벨까지 45 XP   │ │
│  │ 🔥 현재 연속: 3일                          │ │
│  │ ⭐ 최고 연속: 7일                          │ │
│  └──────────────────────────────────────────┘ │
│                                                │
│  📝 사용 가능한 테스트                         │
│  ┌──────────────┐ ┌──────────────┐           │
│  │ [연산]        │ │ [개념]        │           │
│  │ 덧셈과 뺄셈   │ │ 분수의 기초   │           │
│  │ 10문제 | 15분 │ │ 8문제 | 10분  │           │
│  └──────────────┘ └──────────────┘           │
│                                                │
│  ┌──────────────┐ ┌──────────────┐           │
│  │ [빈칸]        │ │ [종합]        │           │
│  │ 소수 개념     │ │ 1단원 총정리  │           │
│  │ 5문제 | 10분  │ │ 20문제 | 30분 │           │
│  └──────────────┘ └──────────────┘           │
│                                                │
│  📊 내 성적 보기 | 🏆 랭킹 보기               │
└────────────────────────────────────────────────┘
```

### 3.2 각 부분 설명

**① 상단 메뉴바**
- 왼쪽: 플랫폼 이름
- 오른쪽: 내 이름, 설정, 로그아웃

**② 내 정보 카드**
- **레벨**: 현재 내 레벨 (문제를 풀면 올라가요!)
- **경험치 바**: 다음 레벨까지 필요한 XP
- **현재 연속**: 며칠 연속 접속했는지
- **최고 연속**: 최고 기록

**③ 테스트 카드**
- **[연산]** - 분홍색: 계산 문제
- **[개념]** - 파란색: 개념 이해 문제
- **[빈칸]** - 노란색: 빈칸 채우기 문제
- **[종합]** - 보라색: 단원 종합 문제

각 카드에는:
- 테스트 이름
- 문제 개수
- 예상 소요 시간
이 표시돼요!

**④ 하단 메뉴**
- **내 성적 보기**: 내가 푼 문제들의 결과
- **랭킹 보기**: 우리 반 친구들과 비교

### 3.3 테스트 카드 색상 의미

| 색상 | 종류 | 특징 |
|------|------|------|
| 🟥 분홍 | 연산 | 계산 위주, 빠르게 풀기 |
| 🟦 파랑 | 개념 | 이해가 중요, 천천히 생각하기 |
| 🟨 노랑 | 빈칸 | 개념 암기, 반복 학습 |
| 🟪 보라 | 종합 | 모든 것 종합, 실력 테스트 |

## 4. 테스트 풀기 📝

### 4.1 테스트 시작하기

**STEP 1: 테스트 선택**
1. 메인 화면에서 풀고 싶은 테스트 카드를 클릭하세요
2. 테스트 정보가 나와요:
   ```
   📊 테스트 정보
   - 제목: 분수의 덧셈과 뺄셈
   - 학년: 중학교 1학년
   - 문제 수: 10개
   - 제한 시간: 15분
   - 배점: 100점 만점
   ```

3. 준비가 되었으면 "시작하기" 버튼을 클릭!

**STEP 2: 테스트 화면**
```
┌────────────────────────────────────────────────┐
│  ⏱️ 남은 시간: 14:23          문제 1/10          │
├────────────────────────────────────────────────┤
│                                                │
│  [문제] Lv.3                        [개념]     │
│                                                │
│  1번 문제                                      │
│                                                │
│  2/3 + 1/4 의 값은?                            │
│                                                │
│  ┌──────────────────────────────────────────┐ │
│  │  ⓐ  5/7                                  │ │
│  ├──────────────────────────────────────────┤ │
│  │  ⓑ  11/12                                │ │
│  ├──────────────────────────────────────────┤ │
│  │  ⓒ  8/12                                 │ │
│  ├──────────────────────────────────────────┤ │
│  │  ⓓ  3/7                                  │ │
│  └──────────────────────────────────────────┘ │
│                                                │
│              [이전] [다음] [제출]              │
└────────────────────────────────────────────────┘
```

### 4.2 문제 풀기

**선택지 문제**
1. 정답이라고 생각하는 보기를 클릭하세요
2. 선택하면 보기가 파란색으로 변해요
3. 다른 보기를 클릭하면 선택이 바뀌어요
4. "다음" 버튼을 눌러 다음 문제로!

**단답형 문제**
1. 답을 입력하는 칸이 나와요
2. 정답을 입력하세요 (예: `11/12` 또는 `0.916`)
3. 띄어쓰기 주의! (분수는 `/` 사용)
4. "다음" 버튼을 눌러 다음 문제로!

**TIP 💡**
- 모르는 문제는 일단 넘어가세요 (나중에 돌아올 수 있어요)
- 시간을 계속 확인하세요
- 급하게 풀지 말고 차근차근!

### 4.3 문제 사이 이동하기

**다음 문제로 가기**
- "다음" 버튼 클릭
- 답을 선택하지 않아도 넘어갈 수 있어요

**이전 문제로 돌아가기**
- "이전" 버튼 클릭
- 답을 바꿀 수 있어요

**문제 번호로 바로 가기**
- 위쪽 "문제 1/10" 부분 클릭
- 원하는 문제 번호 선택

### 4.4 테스트 제출하기

**언제 제출하나요?**
- 모든 문제를 다 풀었을 때
- 시간이 거의 다 되었을 때
- 더 이상 풀 수 없을 것 같을 때

**제출 방법**
1. "제출" 버튼 클릭
2. 확인 창이 뜨면:
   ```
   정말 제출하시겠습니까?

   답변한 문제: 8/10
   답변하지 않은 문제: 2개

   제출하면 수정할 수 없습니다.

   [취소]  [제출하기]
   ```

3. "제출하기" 클릭!

**주의 ⚠️**
- 제출하면 다시 되돌릴 수 없어요
- 답하지 않은 문제는 0점 처리돼요
- 시간 내에 꼭 제출하세요!

## 5. 빈칸 채우기 문제 ✏️

### 5.1 빈칸 채우기란?

**이게 뭐예요?**
개념을 완벽하게 외우기 위한 특별한 문제 형식이에요!
중요한 단어가 빈칸으로 나오면 채워넣는 거예요.

**왜 하나요?**
- 개념을 정확하게 암기할 수 있어요
- 반복해서 풀면 자동으로 외워져요
- 시험에 나오는 중요한 부분을 익힐 수 있어요

### 5.2 빈칸 채우기 방법

**1회차: 완독하기**
```
┌────────────────────────────────────────────────┐
│  1회차: 개념 완독                               │
│                                                │
│  세 변의 길이가 같은 삼각형은 정삼각형이다.     │
│                                                │
│  [다음 문제]                                   │
└────────────────────────────────────────────────┘
```
- 빈칸이 하나도 없어요
- 개념을 천천히 읽고 이해하세요
- 중요한 단어를 눈여겨보세요

**2회차: 중요 단어 채우기**
```
┌────────────────────────────────────────────────┐
│  2회차: 중요 개념 (2~3개 빈칸)                 │
│                                                │
│  ___ 변의 ___가 같은 삼각형은 ___이다.         │
│   ↑      ↑                       ↑           │
│   세    길이                  정삼각형         │
│                                                │
│  [Tab으로 이동] [제출]                         │
└────────────────────────────────────────────────┘
```
- 중요한 단어 2~3개가 빈칸이에요
- 각 빈칸에 정답을 입력하세요
- Tab 키를 누르면 다음 빈칸으로 이동해요

**3회차: 거의 모든 단어 채우기**
```
┌────────────────────────────────────────────────┐
│  3회차: 완전 숙달 (4~6개 빈칸)                 │
│                                                │
│  ___ ___  ___   ___ ___은 ___이다.             │
│                                                │
│  이제 외울 수 있겠죠?                          │
│                                                │
│  [Tab으로 이동] [제출]                         │
└────────────────────────────────────────────────┘
```
- 거의 모든 단어가 빈칸이에요
- 완벽하게 외웠는지 확인하는 단계예요
- 다 맞히면 정말 잘한 거예요!

### 5.3 빈칸 채우기 팁

**입력 방법**
1. 첫 번째 빈칸을 클릭하거나 자동으로 커서가 가있어요
2. 정답을 입력하세요
3. **Tab 키**를 눌러 다음 빈칸으로 이동
4. **Shift + Tab**을 누르면 이전 빈칸으로 이동

**띄어쓰기 주의!**
```
✅ 정답: 정삼각형
❌ 오답: 정 삼각형 (띄어쓰기 들어감)
❌ 오답: 정삼각형이다 (불필요한 단어 추가)
```

**대소문자 구분**
```
✅ 정답: 정삼각형 또는 정三角形
✅ 정답: x, X (보통 구분 안 함)
```

**TIP 💡**
- 확실하지 않으면 1회차를 다시 보세요
- 모르는 빈칸은 일단 비워두고 다른 것부터 채우세요
- 제출하면 부분 점수를 받을 수 있어요!

### 5.4 부분 점수

빈칸 채우기는 부분 점수를 받을 수 있어요!

**예시: 3개 빈칸, 10점 문제**
```
3개 모두 정답: 10점 (100%)
2개 정답, 1개 오답: 6점 (66%)
1개 정답, 2개 오답: 3점 (33%)
모두 오답: 0점 (0%)
```

**TIP 💡**
- 모르는 빈칸이 있어도 포기하지 마세요
- 아는 것만 맞혀도 점수를 받아요!
- 틀린 빈칸은 해설에서 정답을 확인하세요

## 7. 레벨과 경험치 🎮

### 7.1 레벨 시스템

**레벨이 뭐예요?**

- 💎 특별한 뱃지 획득

### 7.2 경험치(XP) 얻기

**어떻게 얻나요?**

| 활동 | 획득 XP |
|------|---------|
| 문제 1개 정답 | 5 XP |
| 문제 1개 정답 (난이도 높음) | 7~10 XP |
| 콤보 3개 | +5 XP (보너스) |
| 콤보 5개 | +10 XP (보너스) |
| 콤보 10개 | +20 XP (보너스) |
| 테스트 완료 | +30 XP |
| 100점 만점 | +50 XP (보너스) |
| 매일 접속 | +10 XP |

**경험치 계산 예시**
```
테스트 완료: +30 XP
10문제 정답: +50 XP (10 × 5)
콤보 5개 달성: +10 XP
─────────────────
총 획득: 90 XP
```

### 7.3 콤보 시스템

**콤보가 뭐예요?**
문제를 연속으로 맞히면 쌓이는 보너스예요!

**콤보 쌓기**
```
1번 문제 정답: 콤보 1개
2번 문제 정답: 콤보 2개
3번 문제 정답: 콤보 3개 → +5 XP 보너스!
4번 문제 틀림: 콤보 리셋 (0으로 돌아감)
5번 문제 정답: 콤보 1개 (다시 시작)
```

**콤보 보너스**
- 3 콤보: +5 XP
- 5 콤보: +10 XP
- 10 콤보: +20 XP
- 15 콤보: +30 XP

**TIP 💡**
- 콤보를 유지하려면 실수하지 않도록 집중!
- 쉬운 문제부터 풀어서 콤보를 만들어요
- 콤보가 끊겨도 괜찮아요, 다시 시작하면 돼요!

### 7.4 연속 출석

**연속 출석이 뭐예요?**
매일 접속해서 문제를 풀면 쌓이는 기록이에요!

**연속 출석 보상**
```
연속 1일: +10 XP
연속 3일: +20 XP
연속 7일: +50 XP (1주일 달성!)
연속 30일: +200 XP (1달 달성!)
```

**연속 기록 유지하기**
- 매일 최소 1개 이상의 테스트를 완료하세요
- 주말에도 잊지 말고 접속하세요!
- 하루라도 안 하면 기록이 리셋돼요 (0으로)

**TIP 💡**
- 알림을 설정해두면 잊지 않아요
- 짧은 테스트라도 매일 하는 게 좋아요
- 친구와 함께 하면 더 재미있어요!

## 8. 자주 묻는 질문 (FAQ) ❓

### 8.1 로그인 관련

**Q1: 비밀번호를 잊어버렸어요!**
A: 선생님께 문의하세요. 선생님이 비밀번호를 초기화해주실 수 있어요.

**Q2: 다른 기기에서도 로그인할 수 있나요?**
A: 네! 같은 이메일과 비밀번호로 어디서든 로그인할 수 있어요.

**Q3: 동시에 여러 기기에서 로그인하면 어떻게 되나요?**
A: 나중에 로그인한 기기에서만 사용할 수 있어요. 이전 기기는 자동으로 로그아웃돼요.

### 8.2 테스트 관련

**Q4: 테스트 도중에 나가면 어떻게 되나요?**
A: 다시 돌아와서 이어서 풀 수 있어요. 하지만 시간은 계속 흘러요!

**Q5: 시간 안에 못 풀면 어떻게 되나요?**
A: 시간이 끝나면 자동으로 제출돼요. 답하지 않은 문제는 0점 처리돼요.

**Q6: 같은 테스트를 여러 번 풀 수 있나요?**
A: 네! 몇 번이든 다시 풀 수 있어요. 가장 높은 점수가 기록에 남아요.

**Q7: 문제가 이상한 것 같아요.**
A: 선생님께 알려주세요. 문제에 오류가 있을 수 있어요.

### 8.3 점수 관련

**Q8: 점수는 어떻게 계산되나요?**
A: 맞힌 문제 × 문제당 배점 = 내 점수예요.
   예: 10문제 중 8개 정답, 문제당 10점 = 80점

**Q9: 콤보 보너스는 점수에 포함되나요?**
A: 아니요! 콤보 보너스는 XP만 추가돼요. 테스트 점수에는 영향 없어요.

**Q10: 부분 점수는 언제 받나요?**
A: 빈칸 채우기 문제에서만 부분 점수를 받을 수 있어요.

### 8.4 레벨/XP 관련

**Q11: XP는 어떻게 쓰나요?**
A: XP는 레벨 업에만 사용돼요. 자동으로 쌓이고 레벨이 올라가요.

**Q12: 레벨이 내려갈 수도 있나요?**
A: 아니요! 한번 오른 레벨은 절대 내려가지 않아요.

**Q13: 우리 반에서 제일 높은 레벨은 몇 레벨인가요?**
A: "🏆 랭킹 보기"에서 우리 반 친구들의 레벨을 볼 수 있어요!

### 8.5 기술 문제

**Q14: 화면이 안 보여요 / 깨져 보여요.**
A:
1. 페이지를 새로고침(F5)하세요
2. 브라우저 캐시를 지워보세요
3. 다른 브라우저를 사용해보세요 (크롬 권장)

**Q15: 답을 선택했는데 제출이 안 돼요.**
A:
1. 인터넷 연결을 확인하세요
2. 페이지를 새로고침하고 다시 시도하세요
3. 계속 안 되면 선생님께 알려주세요

**Q16: 모바일에서도 되나요?**
A: 네! 하지만 화면이 큰 태블릿이나 컴퓨터를 권장해요.

## 9. 문제 해결하기 🔧

### 9.1 일반적인 문제

#### 문제 1: "페이지를 찾을 수 없습니다" 오류

**원인**
- 잘못된 주소 입력
- 서버 점검 중
- 인터넷 연결 끊김

**해결 방법**
1. 주소를 정확히 입력했는지 확인
2. 인터넷 연결 상태 확인
3. 5분 후 다시 시도
4. 계속 안 되면 선생님께 문의

#### 문제 2: 답을 선택했는데 다음으로 안 넘어가요

**원인**
- 버튼을 제대로 클릭하지 않음
- 화면이 멈춤
- 인터넷 느림

**해결 방법**
1. "다음" 버튼을 다시 한번 클릭
2. 페이지 새로고침 (F5)
3. 답이 저장되었는지 확인하고 계속

#### 문제 3: 시간이 흘러가지 않아요 / 너무 빨라요

**원인**
- 컴퓨터 시계가 잘못됨
- 브라우저 문제

**해결 방법**
1. 컴퓨터 시계 설정 확인
2. 브라우저 재시작
3. 계속 문제되면 선생님께 알림

### 9.2 긴급 상황 대처

#### 상황 1: 테스트 중에 컴퓨터가 꺼졌어요!

**대처 방법**
1. 당황하지 마세요!
2. 컴퓨터를 다시 켜고 로그인하세요
3. 메인 화면에서 "이어서 풀기" 또는 해당 테스트를 다시 클릭
4. 남은 시간 안에 계속 풀 수 있어요

**TIP 💡**
- 답은 자동 저장되기 때문에 대부분 보존돼요
- 하지만 시간은 계속 흘러요!
- 빨리 돌아와서 이어서 푸세요

#### 상황 2: 인터넷이 끊겼어요!

**대처 방법**
1. Wi-Fi 또는 데이터 연결 확인
2. 다시 연결되면 페이지 새로고침
3. 로그인 화면이 나오면 다시 로그인
4. 테스트를 이어서 풀 수 있어요

**주의 ⚠️**
- 인터넷이 끊긴 동안에는 아무것도 할 수 없어요
- 빨리 연결을 복구하세요!

#### 상황 3: 제출했는데 결과가 안 나와요!

**대처 방법**
1. 5초 정도 기다려보세요 (처리 시간)
2. 페이지를 새로고침해보세요
3. "내 성적" 페이지에서 결과를 확인해보세요
4. 안 보이면 선생님께 알려주세요

**언제 선생님께 도움을 요청하나요?**
- 로그인이 안 될 때
