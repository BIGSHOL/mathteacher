## 2. 영역별 계통도

### 2.1 4개 영역 통합 (2022 개정)

| 2015 개정 | 2022 개정 | 통합 의의 |
|-----------|-----------|-----------|
| 수와 연산 | **수와 연산** | 자연수 → 실수 → 복소수 확장 일관성 |
| 규칙성 / 문자와 식 / 함수 | **변화와 관계** | 패턴 → 대수식 → 함수적 모델링 연속성 |
| 도형 / 측정 | **도형과 측정** | 기하(질적) + 측정(양적) 통합 |
| 자료와 가능성 / 확률과 통계 | **자료와 가능성** | 데이터 리터러시 위계 확립 |

### 2.2 전체 계통 흐름도

```
초등학교 (1-6학년)
    │
    │ 수 감각, 기초 연산, 직관적 도형
    │
    ▼
중학교 (1-3학년)
    │
    │ 일반화, 추상화, 논증 시작
    │ 정수/유리수 → 실수, 방정식/함수
    │
    ▼
고등학교 (공통 + 선택)
    │
    │ 체계 완성, 심화, 응용
    │ 복소수, 행렬, 미적분, 확률/통계
    │
    ▼
대학/실생활 적용
```

## 3. 수와 연산 계통

### 3.1 수 체계의 확장

```
자연수(N) → 정수(Z) → 유리수(Q) → 실수(R) → 복소수(C)
   ↑           ↑           ↑           ↑           ↑
  초1-2       중1         초5-중2      중3         고1
```

### 3.2 학교급별 핵심 내용

| 단계 | 학년 | 핵심 개념 | 연산 범위 |
|------|------|-----------|-----------|
| **초등** | 1-2 | 자연수, 0 | 두 자리 수 덧뺄셈, 곱셈 도입 |
| | 3-4 | 분수, 소수 | 자연수 사칙연산 완성 |
| | 5-6 | 혼합계산 | 분수/소수 사칙연산 |
| **중등** | 중1 | 정수, 유리수 | 음수 계산, 혼합연산 |
| | 중2 | 순환소수 | 수의 조밀성 이해 |
| | 중3 | 제곱근, 실수 | 무리수 도입, 실수 완성 |
| **고등** | 고1 | 복소수 | 허수 연산, 체계 완성 |
| | 고1 | **행렬** (신설) | 데이터 구조화 연산 |

### 3.3 연산 법칙의 일관성

| 법칙 | 자연수 | 정수 | 유리수 | 실수 | 복소수 | 행렬 |
|------|--------|------|--------|------|--------|------|
| 교환법칙 | ✓ | ✓ | ✓ | ✓ | ✓ | **✗** |
| 결합법칙 | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| 분배법칙 | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |

> **계통적 메시지**: 수 체계가 확장되어도 연산 법칙은 유지됨 (행렬 제외)

## 4. 변화와 관계 계통

### 4.1 대수적 사고의 발달

```
규칙 찾기 → 등호 이해 → 문자 사용 → 방정식/부등식 → 함수 → 미적분
   ↑            ↑           ↑              ↑          ↑       ↑
 초1-2        초3-4        중1            중1-3       중2-고1   고2-3
```

### 4.2 학교급별 핵심 내용

| 단계 | 학년 | 핵심 개념 | 계통적 의의 |
|------|------|-----------|-------------|
| **초등** | 1-2 | 패턴 찾기 | 규칙성 인식 |
| | 3-4 | **등호(=) 동치 관계** ⭐ | 방정식 기초 (오개념 예방) |
| | 5-6 | 비와 비율, 속력/밀도/농도 | 변수 간 관계 |
| **중등** | 중1 | 문자와 식, 좌표평면 | 일반화, 시각화 |
| | 중2 | 일차함수 | 변화율 개념 |
| | 중3 | 이차함수, **최댓값/최솟값** ⭐ | 함수의 거동 분석 |
| **고등** | 고1 | 방정식, 부등식, 도형의 방정식 | 대수와 기하 결합 |
| | 고2-3 | 지수/로그함수, 삼각함수, 미적분 | 해석학 도약 |

### 4.3 핵심 변화: 등호의 이해

| 이해 수준 | 설명 | 학년 |
|-----------|------|------|
| 연산 기호 (오개념) | "3+4=" 는 "계산하라" | 초등 저학년 흔함 |
| **동치 관계** ⭐ | "3+4 = 5+2" 양변이 같음 | 초3-4 명시적 학습 |
| 방정식 기초 | $x + 5 = 12$ | 중1 |

> **계통적 장치**: 등호 오개념을 초등에서 예방 → 중등 방정식 학습 촉진

## 5. 도형과 측정 계통

### 5.1 기하학적 사고의 발달

```
직관적 인식 → 분석적 수준 → 논증 → 해석기하 → 벡터
     ↑             ↑          ↑         ↑         ↑
   초1-2         초3-4       중1-3      고1      고2(기하)
```

### 5.2 학교급별 핵심 내용

| 단계 | 학년 | 핵심 개념 | 계통적 의의 |
|------|------|-----------|-------------|
| **초등** | 1-2 | 모양 분류 | 직관적 인식 |
| | 3-4 | 변, 꼭짓점, 각도 | 분석적 수준 |
| | 5-6 | 넓이, 부피 공식 | 측정의 정량화 |
| **중등** | 중1-2 | 작도, 합동, 피타고라스 | **증명** 도입 ⭐ |
| | 중3 | 삼각비 | 비율과 도형 결합 |
| **고등** | 고1 | 도형의 방정식 | 해석기하 (좌표 도입) |
| | 고2 (기하) | **공간벡터** ⭐ (재도입) | 공간 지각 + 대수 |

### 5.3 핵심 변화: 증명의 부활

| 2015 개정 | 2022 개정 |
|-----------|-----------|
| "정당화" | **"증명"** |

> **계통적 의의**: 논리적 추론 강조 → 고교 수학의 엄밀성 연결

### 5.4 핵심 변화: 공간벡터 재도입

| 2015 개정 | 2022 개정 |
|-----------|-----------|
| 삭제됨 | \<기하\> 과목에 재도입 |

> **배경**: 이공계 대학 교육과의 연계 단절 문제 해소

## 6. 자료와 가능성 계통

### 6.1 데이터 리터러시의 발달

```
분류하기 → 표/그래프 → 대푯값/산포도 → 확률 → 통계적 추정
    ↑          ↑            ↑           ↑          ↑
  초1-2      초3-6         중1-3       중2-고1     고2
```

### 6.2 학교급별 핵심 내용

| 단계 | 학년 | 핵심 개념 | 계통적 의의 |
|------|------|-----------|-------------|
| **초등** | 1-2 | 분류하기 | 기준에 따른 정리 |
| | 3-4 | 표, 막대/꺾은선그래프 | 시각화 기초 |
| | 5-6 | 평균, 그림그래프 | 대푯값 개념 |
| **중등** | **중1** | **대푯값 (중앙값, 최빈값)** ⭐ | 통계적 소양 조기 함양 |
| | 중2 | 확률 기본 | 불확실성 정량화 |
| | **중3** | **상자 그림(Box Plot)** ⭐ | 현대 통계 시각화 |
| **고등** | 고1-2 | 순열, 조합, 확률분포 | 경우의 수 + 확률 |
| | 고2-3 | **모비율, 표본비율** ⭐ (재도입) | 통계적 추정 완성 |

### 6.3 핵심 변화: 대푯값 하향 이동

| 2015 개정 | 2022 개정 |
|-----------|-----------|
| 중3 | **중1** |

> **이유**: 자유학기제 등 탐구 활동 시기에 실제 데이터 다루기

### 6.4 핵심 변화: 상자 그림 도입 (중3)

| 특징 | 설명 |
|------|------|
| 시각화 | 데이터 분포와 산포도 표현 |
| 비교 | 두 집단 특성 비교 용이 |
| 현대적 | 통계학 시각화 추세 반영 |

## 8. 교수학습 시사점

### 8.1 거시적 안목의 교수 설계

> 개별 단원이 12년 수학 학습에서 어떤 **위치(Coordinate)**를 차지하는지 파악

| 관점 | 질문 |
|------|------|
| 선수 학습 | 이 단원을 위해 무엇을 알아야 하나? |
| 후속 학습 | 이 단원이 무엇의 토대가 되나? |
| 연결성 | 다른 영역/교과와 어떻게 연결되나? |

### 8.2 연결 역량의 3차원

| 차원 | 설명 | 예시 |
|------|------|------|
| **영역 간** | 수학 내부 영역 연결 | 함수로 기하 문제 해결 |
| **타 교과** | 과학, 사회 등과 연결 | 속력(과학), 통계(사회) |
| **실생활** | 삶의 문제 모델링 | 이자 계산, 기후 데이터 |

### 8.3 정보처리 역량과 공학 도구

| 원칙 | 내용 |
|------|------|
| 도구 활용 | 복잡한 계산은 공학 도구에 맡김 |
| 학생 집중 | 문제 설정과 결과 해석에 집중 |
| 효과 | 계산에 매몰되지 않고 큰 흐름(계통) 파악 |

### 8.4 평가 체계 정합성

| 권장 | 지양 |
|------|------|
| 과정 중심 평가 | 단순 공식 암기 확인 |
| 서·논술형 평가 | 답만 요구하는 문제 |
| 연결 역량 평가 | 파편화된 지식 측정 |
