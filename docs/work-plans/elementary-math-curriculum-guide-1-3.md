**곱셈구구 지도 순서**:
```
2단, 5단 → 가장 규칙적 (뛰어 세기 연결)
3단, 6단 → 2배 관계 (3×n의 2배 = 6×n)
4단, 8단 → 2배 관계
7단, 9단, 1단 → 나머지 단
```

**0과 1의 곱 성질**:
- a × 0 = 0 (어떤 수든 0을 곱하면 0)
- a × 1 = a (어떤 수든 1을 곱하면 자기 자신)

#### 3.2.2 나눗셈

| ID | 개념명 | 정의 | 예시 |

$$(\text{나누는 수}) \times (\text{몫}) + (\text{나머지}) = (\text{나누어지는 수})$$

| E3-NUM-08 | 분수 개념 | 전체를 n등분한 것 중 m개 | 3/4 = 4등분 중 3조각 |
| E3-NUM-09 | 단위분수 | 분자가 1인 분수 | 1/2, 1/3, 1/4 |
| E3-NUM-10 | 진분수 | 분자 < 분모 | 2/5, 3/7 |
| E3-NUM-11 | 가분수 | 분자 ≥ 분모 | 5/3, 7/4 |
| E3-NUM-12 | 대분수 | 자연수 + 진분수 | 1²/₃ |
| E3-NUM-13 | 동분모 덧셈 | 분자끼리 더함 | 2/5 + 1/5 = 3/5 |
| E3-NUM-14 | 동분모 뺄셈 | 분자끼리 뺌 | 4/7 - 2/7 = 2/7 |

**분수 크기 비교 (단위분수)**:
$$\frac{1}{3} > \frac{1}{5} \quad (\text{분모가 작을수록 큼!})$$

**⚠️ 자주 틀리는 부분**:
| 오류 | 설명 |
|------|------|
| 분모 더하기 | 2/5 + 1/5 = 3/10 (✘) |
| 크기 비교 착각 | 1/5 > 1/3 (분모가 크니까) (✘) |

**가분수 ↔ 대분수 변환**:
```
가분수 → 대분수: 5/3 = 5÷3 = 1...2 → 1²/₃
대분수 → 가분수: 1²/₃ = (1×3+2)/3 = 5/3
```

### 3.3 도형과 측정 (GEO)

#### 3.3.1 선의 종류

| 선의 종류 | 정의 | 특징 |
|-----------|------|------|
| 선분 | 두 점을 곧게 이은 선 | 시작점, 끝점 있음 (길이 측정 가능) |
| 반직선 | 한 점에서 한쪽으로 끝없이 | 시작점만 있음 (방향 중요) |
| 직선 | 양쪽으로 끝없이 뻗은 선 | 시작/끝 없음 (길이 측정 불가) |

**⚠️ 주의**: 반직선 AB ≠ 반직선 BA (방향이 다름!)

#### 3.3.2 각과 직각

| 개념 | 정의 | 특징 |
|------|------|------|
| 각 | 한 점에서 그은 두 반직선 | 꼭짓점 + 두 변 |
| 직각 | 종이 두 번 접었을 때 생기는 각 | 90° |
| 직각삼각형 | 직각이 있는 삼각형 | 직각 1개 |
| 직사각형 | 네 각이 모두 직각 | 직각 4개 |

#### 3.3.3 원

| 용어 | 정의 |
|------|------|
| 원의 중심 | 원의 가장 안쪽 점 |
| 반지름 | 중심에서 원 위의 점까지 선분 |
| 지름 | 중심을 지나는 원 위 두 점 잇는 선분 |

**성질**:
- 한 원에서 반지름은 모두 같음
- 지름 = 반지름 × 2 (d = 2r)

**⚠️ 주의**: 지름은 반드시 **중심을 통과**해야 함!

#### 3.3.4 시간 연산

| 연산 | 핵심 | 예시 |
|------|------|------|
| 시간 덧셈 | 60분 → 1시간 올림 | 2시 50분 + 20분 = 3시 10분 |
| 시간 뺄셈 | 1시간 → 60분 내림 | 3시 10분 - 30분 = 2시 40분 |

**⚠️ 60진법 오류**:
- 2시 50분 + 20분 = 2시 70분 (✘) → 3시 10분 (✓)
- 1시간 빌려와 100분 계산 (✘) → 60분으로 계산 (✓)
